SYNOPSIS                   *cp_mgmt_run_ips_update*

     • Runs IPS database update. If “package-path” is not provided server will try to get the latest package from the User Center.
     • All operations are performed over Web Services API.

PARAMETERS                   *cp_mgmt_run_ips_update-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   package_path                   Offline update package path.
   string
   version                        Version of checkpoint. If not given one, the latest version taken.
   string
   wait_for_task   Choices:
   boolean         • no           Wait for the task to end. Such as publish task.
                   • yes ←
