SYNOPSIS                   *fortios_system_password_policy*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and password_policy category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_password_policy-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

PARAMETERS                   *fortios_system_password_policy-parameters*

         Parameter            Choices/Defaults                                                                          Comments
   host                                            FortiOS or FortiGate IP address.
   string
   https                    Choices:
   boolean                  • no                   Indicates if the requests towards FortiGate must use HTTPS protocol.
                            • yes ←
   password               Default:                 FortiOS or FortiGate password.
   string                 ""
   ssl_verify               Choices:
   boolean                  • no                   Ensures FortiGate certificate must be verified by a proper CA.
                            • yes ←
   system_password_policy Default:                 Configure password policy for locally defined administrator passwords and IPsec VPN pre-shared keys.
   dictionary             null
    apply_to                Choices:
    string                  • admin-password       Apply password policy to administrator passwords or IPsec pre-shared keys or both. Separate entries with a space.
                            • ipsec-preshared-key
    change_4_characters     Choices:
    string                  • enable               Enable/disable changing at least 4 characters for a new password (This attribute overrides reuse-password if both are enabled).
                            • disable
    expire_day                                     Number of days after which passwords expire (1 - 999 days).
    integer
    expire_status           Choices:
    string                  • enable               Enable/disable password expiration.
                            • disable
    min_lower_case_letter                          Minimum number of lowercase characters in password (0 - 128).
    integer
    min_non_alphanumeric                           Minimum number of non-alphanumeric characters in password (0 - 128).
    integer
    min_number                                     Minimum number of numeric characters in password (0 - 128).
    integer
    min_upper_case_letter                          Minimum number of uppercase characters in password (0 - 128).
    integer
    minimum_length                                 Minimum password length (8 - 128).
    integer
    reuse_password          Choices:
    string                  • enable               Enable/disable reusing of password (if both reuse-password and change-4-characters are enabled, change-4-characters overrides).
                            • disable
    status                  Choices:
    string                  • enable               Enable/disable setting a password policy for locally defined administrator passwords and IPsec VPN pre-shared keys.
                            • disable
   username                                        FortiOS or FortiGate username.
   string
   vdom                   Default:                 Virtual domain, among those defined previously. A vdom is a virtual instance of the FortiGate that can be configured and used as a different unit.
   string                 "root"

NOTES                   *fortios_system_password_policy-notes*

   Note

     • Requires fortiosapi library developed by Fortinet
     • Run as a local_action in your playbook

EXAMPLES                   *fortios_system_password_policy-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure password policy for locally defined administrator passwords and IPsec VPN pre-shared keys.
     fortios_system_password_policy:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       system_password_policy:
         apply_to: "admin-password"
         change_4_characters: "enable"
         expire_day: "5"
         expire_status: "enable"
         min_lower_case_letter: "7"
         min_non_alphanumeric: "8"
         min_number: "9"
         min_upper_case_letter: "10"
         minimum_length: "11"
         reuse_password: "enable"
         status: "enable"
