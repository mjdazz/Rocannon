SYNOPSIS                   *portinstall*

     • Manage packages for FreeBSD using ‘portinstall’.

PARAMETERS                   *portinstall-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   name                          name of package to install/remove
   - / required
   state          Choices:
   -              • present ←    state of the package
                  • absent
   use_packages   Choices:
   boolean        • no           use packages instead of ports whenever available
                  • yes ←

EXAMPLES                   *portinstall-examples*

 # Install package foo
 - portinstall:
     name: foo
     state: present

 # Install package security/cyrus-sasl2-saslauthd
 - portinstall:
     name: security/cyrus-sasl2-saslauthd
     state: present

 # Remove packages foo and bar
 - portinstall:
     name: foo,bar
     state: absent
