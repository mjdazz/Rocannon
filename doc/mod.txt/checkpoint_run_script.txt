SYNOPSIS                   *checkpoint_run_script*

     • Run scripts on Check Point devices. All operations are performed over Web Services API.

PARAMETERS                   *checkpoint_run_script-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   script                             Script body contents.
   string / required
   script_name                        Name of the script.
   string / required
   targets                            Targets the script should be run against. Can reference either name or UID.
   list / required

EXAMPLES                   *checkpoint_run_script-examples*

 - name: Run script
   checkpoint_run_script:
     script_name: "List root"
     script: ls -l /
     targets:
       - mycheckpointgw

RETURN VALUES                   *checkpoint_run_script-return values*

   Common return values are documented here, the following are the fields unique to this module:

            Key          Returned                                                                             Description
   checkpoint_run_script always.  The checkpoint run script output.
   list
