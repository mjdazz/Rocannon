SYNOPSIS                   *fortios_switch_controller_lldp_settings*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify switch_controller feature and lldp_settings category. Examples include all
       parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_switch_controller_lldp_settings-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

NOTES                   *fortios_switch_controller_lldp_settings-notes*

   Note

     • Requires fortiosapi library developed by Fortinet

EXAMPLES                   *fortios_switch_controller_lldp_settings-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure FortiSwitch LLDP settings.
     fortios_switch_controller_lldp_settings:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       switch_controller_lldp_settings:
         fast_start_interval: "3"
         management_interface: "internal"
         status: "enable"
         tx_hold: "6"
         tx_interval: "7"
