SYNOPSIS                   *npm*

     • Manage node.js packages with Node Package Manager (npm)

REQUIREMENTS                   *npm-requirements*

   The below requirements are needed on the host that executes this module.

     • npm installed in bin path (recommended /usr/local/bin)

PARAMETERS                   *npm-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   ci               Choices:
   boolean          • no ←         Install packages based on package-lock file, same as running npm ci
   added in 2.8     • yes
   executable                      The executable location for npm.
   path                            This is useful if you are using a version manager, such as nvm
   global           Choices:
   boolean          • no ←         Install the node.js library globally
                    • yes
   ignore_scripts   Choices:
   boolean          • no ←         Use the --ignore-scripts flag when installing.
                    • yes
   name                            The name of a node.js library to install
   string
   path                            The base path where to install the node.js libraries
   path
   production       Choices:
   boolean          • no ←         Install dependencies in production mode, excluding devDependencies
                    • yes
   registry                        The registry to install modules from.
   string
                    Choices:
   state            • present ←    The state of the node.js library
   string           • absent
                    • latest
   unsafe_perm      Choices:
   boolean          • no ←         Use the --unsafe-perm flag when installing.
   added in 2.8     • yes
   version                         The version to be installed
   string

EXAMPLES                   *npm-examples*

 - name: Install "coffee-script" node.js package.
   npm:
     name: coffee-script
     path: /app/location

 - name: Install "coffee-script" node.js package on version 1.6.1.
   npm:
     name: coffee-script
     version: '1.6.1'
     path: /app/location

 - name: Install "coffee-script" node.js package globally.
   npm:
     name: coffee-script
     global: yes

 - name: Remove the globally package "coffee-script".
   npm:
     name: coffee-script
     global: yes
     state: absent

 - name: Install "coffee-script" node.js package from custom registry.
   npm:
     name: coffee-script
     registry: 'http://registry.mysite.com'

 - name: Install packages based on package.json.
   npm:
     path: /app/location

 - name: Update packages based on package.json to their latest version.
   npm:
     path: /app/location
     state: latest

 - name: Install packages based on package.json using the npm installed with nvm v0.10.1.
   npm:
     path: /app/location
     executable: /opt/nvm/v0.10.1/bin/npm
     state: present
