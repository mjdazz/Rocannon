SYNOPSIS                   *fortios_wireless_controller_hotspot20_hs_profile*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify wireless_controller_hotspot20 feature and hs_profile category. Examples include
       all parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.4

REQUIREMENTS                   *fortios_wireless_controller_hotspot20_hs_profile-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8
