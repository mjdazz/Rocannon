SYNOPSIS                   *kibana_plugin*

     • This module can be used to manage Kibana plugins.

PARAMETERS                   *kibana_plugin-parameters*

    Parameter          Choices/Defaults                                                                               Comments
   force          Choices:
   boolean        • no ←                        Delete and re-install the plugin. Can be useful for plugins update.
                  • yes
   name                                         Name of the plugin to install.
   - / required
   plugin_bin   Default:                        Location of the Kibana binary.
   -            "/opt/kibana/bin/kibana"
   plugin_dir   Default:                        Your configured plugin directory specified in Kibana.
   -            "/opt/kibana/installedPlugins/"
   state          Choices:
   -              • present ←                   Desired state of a plugin.
                  • absent
   timeout      Default:                        Timeout setting: 30s, 1m, 1h etc.
   -            "1m"
   url                                          Set exact URL to download the plugin from.
   -                                            For local file, prefix its absolute path with file://
   version                                      Version of the plugin to be installed.
   -                                            If plugin exists with previous version, plugin will NOT be updated unless force is set to yes.

EXAMPLES                   *kibana_plugin-examples*

 - name: Install Elasticsearch head plugin
   kibana_plugin:
     state: present
     name: elasticsearch/marvel

 - name: Install specific version of a plugin
   kibana_plugin:
     state: present
     name: elasticsearch/marvel
     version: '2.3.3'

 - name: Uninstall Elasticsearch head plugin
   kibana_plugin:
     state: absent
     name: elasticsearch/marvel

RETURN VALUES                   *kibana_plugin-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   cmd     success  the launched command during plugin management (install / remove)
   string
   name    success  the plugin name to install or remove
   string
   state   success  the state for the managed plugin
   string
   stderr  success  the command stderr
   string
   stdout  success  the command stdout
   string
   timeout success  the timeout for plugin download
   string
   url     success  the url from where the plugin is installed from
   string
