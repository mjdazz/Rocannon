SYNOPSIS                   *ibm_sa_host_ports*

     • This module adds ports to or removes them from the hosts on IBM Spectrum Accelerate Family storage systems.

REQUIREMENTS                   *ibm_sa_host_ports-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • pyxcli

PARAMETERS                   *ibm_sa_host_ports-parameters*

         Parameter        Choices/Defaults                                                                          Comments
   endpoints                               The hostname or management IP of Spectrum Accelerate storage system.
   string / required
   fcaddress                               Fiber channel address.
   -
   host                                    Host name.
   - / required
   iscsi_name                              iSCSI initiator name.
   -
   num_of_visible_targets                  Number of visible targets.
   -
   password                                Password for username on the spectrum accelerate storage system.
   string / required
   state                    Choices:
   - / required             • present ←    Host ports state.
                            • absent
   username                                Management user on the spectrum accelerate storage system.
   string / required

NOTES                   *ibm_sa_host_ports-notes*

   Note

     • This module requires pyxcli python library. Use ‘pip install pyxcli’ in order to get pyxcli.

EXAMPLES                   *ibm_sa_host_ports-examples*

 - name: Add ports for host.
   ibm_sa_host_ports:
     host: test_host
     iscsi_name: iqn.1994-05.com"""
     username: admin
     password: secret
     endpoints: hostdev-system
     state: present

 - name: Remove ports for host.
   ibm_sa_host_ports:
     host: test_host
     iscsi_name: iqn.1994-05.com"""
     username: admin
     password: secret
     endpoints: hostdev-system
     state: absent
