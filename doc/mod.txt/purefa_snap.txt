SYNOPSIS                   *purefa_snap*

     • Create or delete volumes and volume snapshots on Pure Storage FlashArray.

REQUIREMENTS                   *purefa_snap-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_snap-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashArray API token for admin privileged user.
   string / required
   eradicate           Choices:
   boolean             • no ←         Define whether to eradicate the snapshot on delete or leave in trash.
                       • yes
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   name                               The name of the source volume.
   string / required
   overwrite           Choices:
   boolean             • no ←         Define whether to overwrite existing volume when creating from snapshot.
                       • yes
                       Choices:
   state               • absent       Define whether the volume snapshot should exist or not.
   string              • copy
                       • present ←
   suffix                             Suffix of snapshot name.
   string
   target                             Name of target volume if creating from snapshot.
   string

NOTES                   *purefa_snap-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_snap-examples*

 - name: Create snapshot foo.ansible
   purefa_snap:
     name: foo
     suffix: ansible
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: present

 - name: Create R/W clone foo_clone from snapshot foo.snap
   purefa_snap:
     name: foo
     suffix: snap
     target: foo_clone
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: copy

 - name: Overwrite existing volume foo_clone with snapshot foo.snap
   purefa_snap:
     name: foo
     suffix: snap
     target: foo_clone
     overwrite: true
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: copy

 - name: Delete and eradicate snapshot named foo.snap
   purefa_snap:
     name: foo
     suffix: snap
     eradicate: true
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: absent
