SYNOPSIS                   *purefb_network*

     • This module manages network interfaces on Pure Storage FlashBlade.
     • When creating a network interface a subnet must already exist with a network prefix that covers the IP address of the interface being created.

REQUIREMENTS                   *purefb_network-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purity_fb >= 1.1

PARAMETERS                   *purefb_network-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   address                            IP address of interface.
   string
   api_token                          FlashBlade API token for admin privileged user.
   string
   fb_url                             FlashBlade management IP address or Hostname.
   string
   itype               Choices:       Type of interface.
   string              • vip ←
   name                               Interface Name.
   string / required
   services            Choices:       Define which services are configured for the interfaces.
   string              • data ←
   state               Choices:
   string              • present ←    Create, delete or modifies a network interface.
                       • absent

NOTES                   *purefb_network-notes*

   Note

     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_network-examples*

 - name: Create new network interface named foo
   purefb_network:
     name: foo
     address: 10.21.200.23
     state: present
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Change IP address of network interface named foo
   purefb_network:
     name: foo
     state: present
     address: 10.21.200.123
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Delete network interface named foo
   purefb_network:
     name: foo
     state: absent
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641
