SYNOPSIS                   *fortios_firewall_ssh_local_key*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall_ssh feature and local_key category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_ssh_local_key-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_ssh_local_key-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: SSH proxy local keys.
     fortios_firewall_ssh_local_key:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_ssh_local_key:
         name: "default_name_3"
         password: "<your_own_value>"
         private_key: "<your_own_value>"
         public_key: "<your_own_value>"
         source: "built-in"
