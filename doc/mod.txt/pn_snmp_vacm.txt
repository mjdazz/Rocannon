SYNOPSIS                   *pn_snmp_vacm*

     • This module can be used to create View Access Control Models (VACM), modify VACM and delete VACM.

PARAMETERS                   *pn_snmp_vacm-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_auth             Choices:
   boolean             • no           authentication required.
                       • yes
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_oid_restrict                    restrict OID.
   string
   pn_priv             Choices:
   boolean             • no           privileges.
                       • yes
   pn_user_name                       SNMP administrator name.
   string
   pn_user_type        Choices:
   string              • rouser       SNMP user type.
                       • rwuser
                       Choices:
   state               • present      State the action to perform. Use present to create snmp-vacm and absent to delete snmp-vacm and update to modify snmp-vacm.
   string / required   • absent
                       • update

EXAMPLES                   *pn_snmp_vacm-examples*

 - name: create snmp vacm
   pn_snmp_vacm:
     pn_cliswitch: "sw01"
     state: "present"
     pn_user_name: "foo"
     pn_user_type: "rouser"

 - name: update snmp vacm
   pn_snmp_vacm:
     pn_cliswitch: "sw01"
     state: "update"
     pn_user_name: "foo"
     pn_user_type: "rwuser"

 - name: delete snmp vacm
   pn_snmp_vacm:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_user_name: "foo"

RETURN VALUES                   *pn_snmp_vacm-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the snmp-vacm command.
   list
   stdout  always   set of responses from the snmp-vacm command.
   list
