SYNOPSIS                   *vmware_dvswitch_nioc*

     • This module can be used to manage distributed switch Network IO Control configurations.

REQUIREMENTS                   *vmware_dvswitch_nioc-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

EXAMPLES                   *vmware_dvswitch_nioc-examples*

 - name: Enable NIOC
   vmware_dvswitch_nioc:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
     switch: dvSwitch
     version: version3
     resources:
         - name: vmotion
           limit: -1
           reservation: 128
           shares_level: normal
         - name: vsan
           limit: -1
           shares_level: custom
           shares: 99
           reservation: 256
     state: present
   delegate_to: localhost

 - name: Disable NIOC
   vmware_dvswitch_nioc:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
     switch: dvSwitch
     state: absent
   delegate_to: localhost
