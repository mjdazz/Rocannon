SYNOPSIS                   *exos_lldp_global*

     • This module configures and manages the Link Layer Discovery Protocol(LLDP) attributes on Extreme Networks EXOS platforms.

PARAMETERS                   *exos_lldp_global-parameters*

         Parameter       Choices/Defaults                                                                          Comments
   config                                 A dictionary of LLDP options
   dictionary
    interval             Default:         Frequency at which LLDP advertisements are sent (in seconds). By default - 30 seconds.
    integer              30
    tlv_select                            This attribute can be used to specify the TLVs that need to be sent in the LLDP packets. By default, only system name and system description is sent
    dictionary
     management_address    Choices:
     boolean               • no           Used to specify the management address in TLV messages
                           • yes
     port_description      Choices:
     boolean               • no           Used to specify the port description TLV
                           • yes
     system_capabilities   Choices:
     boolean               • no           Used to specify the system capabilities TLV
                           • yes
     system_description    Choices:
     boolean               • no           Used to specify the system description TLV
                           • yes ←
     system_name           Choices:
     boolean               • no           Used to specify the system name TLV
                           • yes ←
                           Choices:
   state                   • merged ←     The state of the configuration after module completion.
   string                  • replaced
                           • deleted

NOTES                   *exos_lldp_global-notes*

   Note

     • Tested against Extreme Networks EXOS version 30.2.1.8 on x460g2.
     • This module works with connection httpapi. See EXOS Platform Options
