SYNOPSIS                   *pn_snmp_community*

     • This module can be used to create SNMP communities for SNMPv1 or delete SNMP communities for SNMPv1 or modify SNMP communities for SNMPv1.

PARAMETERS                   *pn_snmp_community-parameters*

        Parameter      Choices/Defaults                                                                           Comments
   pn_cliswitch                         Target switch to run the CLI on.
   -
   pn_community_string                  community name.
   string
   pn_community_type     Choices:
   string                • read-only    community type.
                         • read-write
                         Choices:
   state                 • present      State the action to perform. Use present to create snmp-community and absent to delete snmp-community update to update snmp-community.
   string / required     • absent
                         • update

EXAMPLES                   *pn_snmp_community-examples*

 - name: Create snmp community
   pn_snmp_community:
     pn_cliswitch: "sw01"
     state: "present"
     pn_community_string: "foo"
     pn_community_type: "read-write"

 - name: Delete snmp community
   pn_snmp_community:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_community_string: "foo"

 - name: Modify snmp community
   pn_snmp_community:
     pn_cliswitch: "sw01"
     state: "update"
     pn_community_string: "foo"
     pn_community_type: "read-only"

RETURN VALUES                   *pn_snmp_community-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the snmp-community command.
   list
   stdout  always   set of responses from the snmp-community command.
   list
