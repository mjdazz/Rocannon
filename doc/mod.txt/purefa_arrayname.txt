SYNOPSIS                   *purefa_arrayname*

     • Configure name of array for Pure Storage FlashArrays.
     • Ideal for Day 0 initial configuration.

REQUIREMENTS                   *purefa_arrayname-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_arrayname-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashArray API token for admin privileged user.
   string / required
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   name                               Name of the array. Must conform to correct naming schema.
   string / required
   state               Choices:       Set the array name
   string              • present ←

NOTES                   *purefa_arrayname-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_arrayname-examples*

 - name: Set new array name
   purefa_arrayname:
     name: new-array-name
     state: present
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
