SYNOPSIS                   *pn_igmp_snooping*

     • This module can be used to modify Internet Group Management Protocol (IGMP) snooping.

PARAMETERS                   *pn_igmp_snooping-parameters*

            Parameter          Choices/Defaults                                                                       Comments
   pn_cliswitch                                 Target switch to run the CLI on.
   string
   pn_enable                     Choices:
   boolean                       • no           enable or disable IGMP snooping.
                                 • yes
   pn_enable_vlans                              enable per VLAN IGMP snooping.
   string
   pn_igmpv2_vlans                              VLANs on which to use IGMPv2 protocol.
   string
   pn_igmpv3_vlans                              VLANs on which to use IGMPv3 protocol.
   string
   pn_no_snoop_linklocal_vlans                  Remove snooping of link-local groups(224.0.0.0/24) on these vlans.
   string
   pn_query_interval                            IGMP query interval in seconds.
   string
   pn_query_max_response_time                   maximum response time, in seconds, advertised in IGMP queries.
   string
   pn_scope                      Choices:
   -                             • local        IGMP snooping scope - fabric or local.
                                 • fabric
   pn_snoop_linklocal_vlans                     Allow snooping of link-local groups(224.0.0.0/24) on these vlans.
   string
   pn_vxlan                      Choices:
   boolean                       • no           enable or disable IGMP snooping on vxlans.
                                 • yes
   state                         Choices:       State the action to perform. Use update to modify the igmp-snooping.
   string / required             • update

EXAMPLES                   *pn_igmp_snooping-examples*

 - name: 'Modify IGMP Snooping'
   pn_igmp_snooping:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_vxlan: True
     pn_enable_vlans: '1-399,401-4092'
     pn_no_snoop_linklocal_vlans: 'none'
     pn_igmpv3_vlans: '1-399,401-4092'

 - name: 'Modify IGMP Snooping'
   pn_igmp_snooping:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_vxlan: False
     pn_enable_vlans: '1-399'
     pn_no_snoop_linklocal_vlans: 'none'
     pn_igmpv3_vlans: '1-399'

RETURN VALUES                   *pn_igmp_snooping-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the igmp-snooping command.
   list
   stdout  always   set of responses from the igmp-snooping command.
   list
