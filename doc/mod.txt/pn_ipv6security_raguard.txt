SYNOPSIS                   *pn_ipv6security_raguard*

     • This module can be used to add ipv6 RA Guard Policy, Update ipv6 RA guard Policy and Remove ipv6 RA Guard Policy.

PARAMETERS                   *pn_ipv6security_raguard-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   pn_access_list                      RA Guard Access List of Source IPs.
   string
   pn_cliswitch                        Target switch to run the CLI on.
   string
   pn_device            Choices:
   string               • host         RA Guard Device. host or router.
                        • router
   pn_name                             RA Guard Policy Name.
   string / required
   pn_prefix_list                      RA Guard Prefix List.
   string
                        Choices:
   pn_router_priority   • low          RA Guard Router Priority.
   string               • medium
                        • high
                        Choices:
   state                • present ←    ipv6security-raguard configuration command.
   string               • update
                        • absent

EXAMPLES                   *pn_ipv6security_raguard-examples*

 - name: ipv6 security ragurad create
   pn_ipv6security_raguard:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_device: "host"

 - name: ipv6 security ragurad create
   pn_ipv6security_raguard:
     pn_cliswitch: "sw01"
     pn_name: "foo1"
     pn_device: "host"
     pn_access_list: "sample"
     pn_prefix_list: "sample"
     pn_router_priority: "low"

 - name: ipv6 security ragurad modify
   pn_ipv6security_raguard:
     pn_cliswitch: "sw01"
     pn_name: "foo1"
     pn_device: "router"
     pn_router_priority: "medium"
     state: "update"

 - name: ipv6 security ragurad delete
   pn_ipv6security_raguard:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "absent"

RETURN VALUES                   *pn_ipv6security_raguard-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the ipv6security-raguard command.
   list
   stdout  always   set of responses from the ipv6security-raguard command.
   list
