SYNOPSIS                   *make*

     • Run targets in a Makefile.

REQUIREMENTS                   *make-requirements*

   The below requirements are needed on the host that executes this module.

     • make

PARAMETERS                   *make-parameters*

      Parameter    Choices/Defaults                                                                             Comments
   chdir                            Change to this directory before running make.
   path / required
   file
   path                             Use a custom Makefile.
   added in 2.5
   params                           Any extra parameters to pass to make.
   dictionary
   target                           The target to run.
   string                           Typically this would be something like install,test or all."

EXAMPLES                   *make-examples*

 - name: Build the default target
   make:
     chdir: /home/ubuntu/cool-project

 - name: Run 'install' target as root
   make:
     chdir: /home/ubuntu/cool-project
     target: install
   become: yes

 - name: Build 'all' target with extra arguments
   make:
     chdir: /home/ubuntu/cool-project
     target: all
     params:
       NUM_THREADS: 4
       BACKEND: lapack

 - name: Build 'all' target with a custom Makefile
   make:
     chdir: /home/ubuntu/cool-project
     target: all
     file: /some-project/Makefile
