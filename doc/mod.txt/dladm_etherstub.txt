SYNOPSIS                   *dladm_etherstub*

     • Create or delete etherstubs on Solaris/illumos systems.

PARAMETERS                   *dladm_etherstub-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   name                          Etherstub name.
   - / required
   state          Choices:
   -              • present ←    Create or delete Solaris/illumos etherstub.
                  • absent
   temporary      Choices:
   boolean        • no ←         Specifies that the etherstub is temporary. Temporary etherstubs do not persist across reboots.
                  • yes

EXAMPLES                   *dladm_etherstub-examples*

 # Create 'stub0' etherstub
 - dladm_etherstub:
     name: stub0
     state: present

 # Remove 'stub0 etherstub
 - dladm_etherstub:
     name: stub0
     state: absent

RETURN VALUES                   *dladm_etherstub-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key    Returned                                                                                   Description
   name               etherstub name
   string    always   Sample:
                      switch0
   state              state of the target
   string    always   Sample:
                      present
   temporary          etherstub's persistence
   boolean   always   Sample:
                      True
