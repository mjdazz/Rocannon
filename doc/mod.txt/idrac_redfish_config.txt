SYNOPSIS                   *idrac_redfish_config*

     • For use with Dell iDRAC operations that require Redfish OEM extensions
     • Builds Redfish URIs locally and sends them to remote iDRAC controllers to set or update a configuration attribute.

PARAMETERS                   *idrac_redfish_config-parameters*

          Parameter        Choices/Defaults                                                                         Comments
   baseuri                                  Base URI of iDRAC
   string / required
   category                                 Category to execute on iDRAC
   string / required
   command                                  List of commands to execute on iDRAC
   list / required
   manager_attribute_name  Default:         name of iDRAC attribute to update
   string                  "null"
   manager_attribute_value Default:         value of iDRAC attribute to update
   string                  "null"
   password                                 Password for authentication with iDRAC
   string / required
   timeout                 Default:         Timeout in seconds for URL requests to iDRAC controller
   integer                 10
   username                                 User for authentication with iDRAC
   string / required

EXAMPLES                   *idrac_redfish_config-examples*

 - name: Enable NTP in iDRAC
   idrac_redfish_config:
     category: Manager
     command: SetManagerAttributes
     manager_attribute_name: NTPConfigGroup.1.NTPEnable
     manager_attribute_value: Enabled
     baseuri: "{{ baseuri }}"
     username: "{{ username}}"
     password: "{{ password }}"
 - name: Set NTP server 1 to {{ ntpserver1 }} in iDRAC
   idrac_redfish_config:
     category: Manager
     command: SetManagerAttributes
     manager_attribute_name: NTPConfigGroup.1.NTP1
     manager_attribute_value: "{{ ntpserver1 }}"
     baseuri: "{{ baseuri }}"
     username: "{{ username}}"
     password: "{{ password }}"
 - name: Set Timezone to {{ timezone }} in iDRAC
   idrac_redfish_config:
     category: Manager
     command: SetManagerAttributes
     manager_attribute_name: Time.1.Timezone
     manager_attribute_value: "{{ timezone }}"
     baseuri: "{{ baseuri }}"
     username: "{{ username}}"
     password: "{{ password }}"
