SYNOPSIS                   *memset_server_info*

     • Retrieve server information.
     • This module was called memset_server_facts before Ansible 2.9. The usage did not change.

   Aliases: memset_server_facts

PARAMETERS                   *memset_server_info-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   api_key                       The API key obtained from the Memset control panel.
   - / required
   name                          The server product name (i.e. testyaa1).
   - / required

NOTES                   *memset_server_info-notes*

   Note

     • An API key generated via the Memset customer control panel is needed with the following minimum scope - server.info.

EXAMPLES                   *memset_server_info-examples*

 - name: get details for testyaa1
   memset_server_info:
     name: testyaa1
     api_key: 5eb86c9896ab03919abcf03857163741
   delegate_to: localhost
