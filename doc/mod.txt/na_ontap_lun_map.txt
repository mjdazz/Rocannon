SYNOPSIS                   *na_ontap_lun_map*

     • Map and unmap LUNs on NetApp ONTAP.

REQUIREMENTS                   *na_ontap_lun_map-requirements*

   The below requirements are needed on the host that executes this module.

     • A physical or virtual clustered Data ONTAP system. The modules support Data ONTAP 9.1 and onward
     • Ansible 2.6
     • Python2 netapp-lib (2017.10.30) or later. Install using ‘pip install netapp-lib’
     • Python3 netapp-lib (2018.11.13) or later. Install using ‘pip install netapp-lib’
     • To enable http on the cluster you must run the following commands ‘set -privilege advanced;’ ‘system services web modify -http-enabled true;’

EXAMPLES                   *na_ontap_lun_map-examples*

 - name: Create LUN mapping
   na_ontap_lun_map:
     state: present
     initiator_group_name: ansibleIgroup3234
     path: /vol/iscsi_path/iscsi_lun
     vserver: ci_dev
     hostname: "{{ netapp_hostname }}"
     username: "{{ netapp_username }}"
     password: "{{ netapp_password }}"

 - name: Unmap LUN
   na_ontap_lun_map:
     state: absent
     initiator_group_name: ansibleIgroup3234
     path: /vol/iscsi_path/iscsi_lun
     vserver: ci_dev
     hostname: "{{ netapp_hostname }}"
     username: "{{ netapp_username }}"
     password: "{{ netapp_password }}"

RETURN VALUES                   *na_ontap_lun_map-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key     Returned                                                                                  Description
   lun_naa_id          The Network Address Authority (NAA) identifier for the LUN.
   string     success  Sample:
                       600a0980383045372f5d4c5a70315474
   lun_node            NetApp controller that is hosting the LUN.
   string     success  Sample:
                       node01
   lun_ostype          Specifies the OS of the host accessing the LUN.
   string     success  Sample:
                       vmware
   lun_serial          A unique, 12-byte, ASCII string used to identify the LUN.
   string     success  Sample:
                       80E7/]LZp1Tt
   lun_size            Size of the LUN in bytes.
   integer    success  Sample:
                       2199023255552
   lun_state           Online or offline status of the LUN.
   string     success  Sample:
                       online
