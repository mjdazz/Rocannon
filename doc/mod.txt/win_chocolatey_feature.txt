SYNOPSIS                   *win_chocolatey_feature*

     • Used to enable or disable features in Chocolatey.

PARAMETERS                   *win_chocolatey_feature-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   name                               The name of the feature to manage.
   string / required                  Run choco.exe feature list to get a list of features that can be managed.
   state               Choices:       When disabled then the feature will be disabled.
   string              • disabled     When enabled then the feature will be enabled.
                       • enabled ←

EXAMPLES                   *win_chocolatey_feature-examples*

 - name: Disable file checksum matching
   win_chocolatey_feature:
     name: checksumFiles
     state: disabled

 - name: Stop Chocolatey on the first package failure
   win_chocolatey_feature:
     name: stopOnFirstPackageFailure
     state: enabled
