SYNOPSIS                   *vmware_vcenter_statistics*

     • This module can be used to configure the vCenter server statistics.
     • The remaining settings can be configured with the module vmware_vcenter_settings.

REQUIREMENTS                   *vmware_vcenter_statistics-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_vcenter_statistics-notes*

   Note

     • Tested with vCenter Server Appliance (vCSA) 6.5 and 6.7

EXAMPLES                   *vmware_vcenter_statistics-examples*

 - name: Configure vCenter statistics
   vmware_vcenter_statistics:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
     interval_past_day:
       enabled: true
       interval_minutes: 5
       save_for_days: 1
       level: 1
     interval_past_week:
       enabled: true
       level: 1
     interval_past_month:
       enabled: true
       level: 1
     interval_past_year:
       enabled: true
       save_for_years: 1
       level: 1
     validate_certs: no
   delegate_to: localhost
