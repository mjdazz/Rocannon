SYNOPSIS                   *aws_netapp_cvs_pool*

     • Create, Update, Delete Pool on AWS Cloud Volumes Service.

PARAMETERS                   *aws_netapp_cvs_pool-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_key                            The access key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   api_url                            The url to the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   from_name                          rename the existing pool name ( The human readable name of the Pool )
   string                             from_name is the existing name, and name the new name
                                      can be used with update operation
   name                               pool name ( The human readable name of the Pool )
   string / required                  name can be used for create, update and delete operations
   region                             The region to which the Pool is associated.
   string / required
   secret_key                         The secret_key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
                       Choices:
   serviceLevel        • basic        The service level of the Pool
   string              • standard     can be used with pool create, update operations
                       • extreme
   sizeInBytes                        Size of the Pool in bytes
   integer                            can be used with pool create, update operations
                                      minimum value is 4000000000000 bytes
   state               Choices:
   string / required   • present      Whether the specified pool should exist or not.
                       • absent
   validate_certs      Choices:
   boolean             • no           Should https certificates be validated?
                       • yes ←
   vendorID                           A vendor ID for the Pool. E.g. an ID allocated by a vendor service for the Pool.
   string                             can be used with pool create, update operations
                                      must be unique

NOTES                   *aws_netapp_cvs_pool-notes*

   Note

     • The modules prefixed with aws_cvs_netapp are built to Manage AWS Cloud Volume Service .

EXAMPLES                   *aws_netapp_cvs_pool-examples*

 - name: Create a new Pool
   aws_netapp_cvs_pool:
     state: present
     name: TestPoolBB12
     serviceLevel: extreme
     sizeInBytes: 4000000000000
     vendorID: ansiblePoolTestVendorBB12
     region: us-east-1
     api_url: cds-aws-bundles.netapp.com
     api_key: MyAPiKey
     secret_key: MySecretKey

 - name: Delete a Pool
   aws_netapp_cvs_pool:
     state: absent
     name: TestPoolBB7
     region: us-east-1
     api_url: cds-aws-bundles.netapp.com
     api_key: MyAPiKey
     secret_key: MySecretKey

 - name: Update a Pool
   aws_netapp_cvs_pool:
     state: present
     from_name: TestPoolBB12
     name: Mynewpool7
     vendorID: ansibleVendorMynewpool15
     serviceLevel: extreme
     sizeInBytes: 4000000000000
     region: us-east-1
     api_url: cds-aws-bundles.netapp.com
     api_key: MyAPiKey
     secret_key: MySecretKey
