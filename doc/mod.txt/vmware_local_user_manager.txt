SYNOPSIS                   *vmware_local_user_manager*

     • Manage local users on an ESXi host

REQUIREMENTS                   *vmware_local_user_manager-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi installed

NOTES                   *vmware_local_user_manager-notes*

   Note

     • Tested on ESXi 6.0
     • Be sure that the ESXi user used for login, has the appropriate rights to create / delete / edit users

EXAMPLES                   *vmware_local_user_manager-examples*

 - name: Add local user to ESXi
   vmware_local_user_manager:
     hostname: esxi_hostname
     username: root
     password: vmware
     local_user_name: foo
   delegate_to: localhost
