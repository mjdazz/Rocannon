SYNOPSIS                   *ce_netstream_global*

     • Manages global parameters of NetStream on HUAWEI CloudEngine switches.

PARAMETERS                   *ce_netstream_global-parameters*

        Parameter       Choices/Defaults                                                                           Comments
   index_switch           Choices:
   -                      • 16 ←         Specifies the netstream index-switch.
                          • 32
   interface                             Netstream global interface.
   - / required
   sampler_direction      Choices:
   -                      • inbound      Specifies the netstream sampler direction.
                          • outbound
   sampler_interval                      Specifies the netstream sampler interval, length is 1 - 65535.
   -
   state                  Choices:
   -                      • present ←    Specify desired state of the resource.
                          • absent
   statistics_direction   Choices:
   -                      • inbound      Specifies the netstream statistic direction.
                          • outbound
   statistics_record                     Specifies the flexible netstream statistic record, length is 1 - 32.
   -
   type                   Choices:
   -                      • ip ←         Specifies the type of netstream global.
                          • vxlan

NOTES                   *ce_netstream_global-notes*

   Note

     • Recommended connection is network_cli.
     • This module also works with local connections for legacy playbooks.

EXAMPLES                   *ce_netstream_global-examples*

 - name: netstream global module test
   hosts: cloudengine
   connection: local
   gather_facts: no
   vars:
     cli:
       host: "{{ inventory_hostname }}"
       port: "{{ ansible_ssh_port }}"
       username: "{{ username }}"
       password: "{{ password }}"
       transport: cli

   tasks:

   - name: Configure a netstream sampler at interface 10ge1/0/2, direction is outbound,interval is 30.
     ce_netstream_global:
       interface: 10ge1/0/2
       type: ip
       sampler_interval: 30
       sampler_direction: outbound
       state: present
       provider: "{{ cli }}"
   - name: Configure a netstream flexible statistic at interface 10ge1/0/2, record is test1, type is ip.
     ce_netstream_global:
       type: ip
       interface: 10ge1/0/2
       statistics_record: test1
       provider: "{{ cli }}"
   - name: Set the vxlan index-switch to 32.
     ce_netstream_global:
       type: vxlan
       interface: all
       index_switch: 32
       provider: "{{ cli }}"
