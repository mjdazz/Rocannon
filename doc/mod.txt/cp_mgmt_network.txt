SYNOPSIS                   *cp_mgmt_network*

     • Manages network objects on Check Point devices including creating, updating and removing objects.
     • All operations are performed over Web Services API.

EXAMPLES                   *cp_mgmt_network-examples*

 - name: add-network
   cp_mgmt_network:
     name: New Network 1
     state: present
     subnet: 192.0.2.0
     subnet_mask: 255.255.255.0

 - name: set-network
   cp_mgmt_network:
     color: green
     mask_length: 16
     name: New Network 1
     new_name: New Network 2
     state: present
     subnet: 192.0.0.0

 - name: delete-network
   cp_mgmt_network:
     name: New Network 2
     state: absent
