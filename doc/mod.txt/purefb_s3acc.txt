SYNOPSIS                   *purefb_s3acc*

     • Create or delete object store accounts on a Pure Storage FlashBlade.

REQUIREMENTS                   *purefb_s3acc-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purity_fb >= 1.1

PARAMETERS                   *purefb_s3acc-parameters*

   Parameter Choices/Defaults                                                                                Comments
   api_token                  FlashBlade API token for admin privileged user.
   string
   fb_url                     FlashBlade management IP address or Hostname.
   string
   name                       The name of object store account
   string
   state       Choices:
   string      • absent       Create or delete object store account
               • present ←

NOTES                   *purefb_s3acc-notes*

   Note

     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_s3acc-examples*

 - name: Create object store account foo
   purefb_s3acc:
     name: foo
     fb_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592

 - name: Delete object store account foo
   purefb_s3acc:
     name: foo
     state: absent
     fb_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
