SYNOPSIS                   *checkpoint_access_layer_facts*

     • Get access layer facts on Check Point devices. All operations are performed over Web Services API.

PARAMETERS                   *checkpoint_access_layer_facts-parameters*

   Parameter Choices/Defaults                                                                                Comments
   name                       Name of the access layer object.
   string
   uid                        UID of access layer object.
   string
