SYNOPSIS                   *rhsm_repository*

     • Manage (Enable/Disable) RHSM repositories to the Red Hat Subscription Management entitlement platform using the subscription-manager command.

REQUIREMENTS                   *rhsm_repository-requirements*

   The below requirements are needed on the host that executes this module.

     • subscription-manager

PARAMETERS                   *rhsm_repository-parameters*

   Parameter Choices/Defaults                                                                                Comments
   name                       The ID of repositories to enable.
   - /                        To operate on several repositories this can accept a comma separated list or a YAML list.
   required
   purge       Choices:
   boolean     • no ←         Disable all currently enabled repositories that are not not specified in name. Only set this to True if passing in a list of repositories to the name field. Using this
   added in    • yes          with loop will most likely not have the desired result.
   2.8
               Choices:
   state       • present ←
   - /         • enabled      If state is equal to present or disabled, indicates the desired repository state.
   required    • absent
               • disabled

NOTES                   *rhsm_repository-notes*

   Note

     • In order to manage RHSM repositories the system must be already registered to RHSM manually or using the Ansible redhat_subscription module.

EXAMPLES                   *rhsm_repository-examples*

 - name: Enable a RHSM repository
   rhsm_repository:
     name: rhel-7-server-rpms

 - name: Disable all RHSM repositories
   rhsm_repository:
     name: '"'
     state: disabled

 - name: Enable all repositories starting with rhel-6-server
   rhsm_repository:
     name: rhel-6-server"
     state: enabled

 - name: Disable all repositories except rhel-7-server-rpms
   rhsm_repository:
     name: rhel-7-server-rpms
     purge: True

RETURN VALUES                   *rhsm_repository-return values*

   Common return values are documented here, the following are the fields unique to this module:

       Key      Returned                                                                                 Description
   repositories success  The list of RHSM repositories with their states.
   list                  When this module is used to change the repository states, this list contains the updated states after the changes.
