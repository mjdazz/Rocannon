SYNOPSIS                   *ipinfoio_facts*

     • Gather IP geolocation facts of a host’s IP address using ipinfo.io API

PARAMETERS                   *ipinfoio_facts-parameters*

   Parameter         Choices/Defaults                                                                                Comments
   http_agent Default:                        Set http user agent
   -          "ansible-ipinfoio-module/0.0.1"
   timeout    Default:                        HTTP connection timeout in seconds
   -          10

NOTES                   *ipinfoio_facts-notes*

   Note

     • Check http://ipinfo.io/ for more information

EXAMPLES                   *ipinfoio_facts-examples*

 # Retrieve geolocation data of a host's IP address
 - name: get IP geolocation data
   ipinfoio_facts:

RETURNED FACTS                   *ipinfoio_facts-returned facts*

   Facts returned by this module are added/updated in the hostvars host facts and can be referenced by name just like any other host fact. They do not need to be registered in order to use them.

     Fact   Returned                                                                                   Description
   city              City name
   string            Sample:
                     Mountain View
   country           ISO 3166-1 alpha-2 country code
   string            Sample:
                     US
   hostname          Domain name
   string            Sample:
                     google-public-dns-a.google.com
   ip                Public IP address of a host
   string            Sample:
                     8.8.8.8
   loc               Latitude and Longitude of the location
   string            Sample:
                     37.3860,-122.0838
   org               organization's name
   string            Sample:
                     AS3356 Level 3 Communications, Inc.
   postal            Postal code
   string            Sample:
                     94035
   region            State or province name
   string            Sample:
                     California
