SYNOPSIS                   *sysctl*

     • This module manipulates sysctl entries and optionally performs a /sbin/sysctl -p after changing them.

PARAMETERS                   *sysctl-parameters*

    Parameter    Choices/Defaults                                                                               Comments
   ignoreerrors   Choices:
   boolean        • no ←           Use this option to ignore errors about unknown keys.
                  • yes
   name                            The dot-separated path (aka key) specifying the sysctl variable.
   - / required                    aliases: key
   reload         Choices:
   boolean        • no             If yes, performs a /sbin/sysctl -p if the sysctl_file is updated. If no, does not reload sysctl even if the sysctl_file is updated.
                  • yes ←
   state          Choices:
   -              • present ←      Whether the entry should be present or absent in the sysctl file.
                  • absent
   sysctl_file  Default:           Specifies the absolute path to sysctl.conf, if not /etc/sysctl.conf.
   -            "/etc/sysctl.conf"
   sysctl_set     Choices:
   boolean        • no ←           Verify token value with the sysctl command and set with -w if necessary
                  • yes
   value                           Desired value of the sysctl key.
   -                               aliases: val

EXAMPLES                   *sysctl-examples*

 # Set vm.swappiness to 5 in /etc/sysctl.conf
 - sysctl:
     name: vm.swappiness
     value: '5'
     state: present

 # Remove kernel.panic entry from /etc/sysctl.conf
 - sysctl:
     name: kernel.panic
     state: absent
     sysctl_file: /etc/sysctl.conf

 # Set kernel.panic to 3 in /tmp/test_sysctl.conf
 - sysctl:
     name: kernel.panic
     value: '3'
     sysctl_file: /tmp/test_sysctl.conf
     reload: no

 # Set ip forwarding on in /proc and verify token value with the sysctl command
 - sysctl:
     name: net.ipv4.ip_forward
     value: '1'
     sysctl_set: yes

 # Set ip forwarding on in /proc and in the sysctl file and reload if necessary
 - sysctl:
     name: net.ipv4.ip_forward
     value: '1'
     sysctl_set: yes
     state: present
     reload: yes
