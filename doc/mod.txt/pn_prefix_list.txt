SYNOPSIS                   *pn_prefix_list*

     • This module can be used to create or delete prefix list.

PARAMETERS                   *pn_prefix_list-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            Prefix List Name.
   string / required
   pn_scope            Choices:
   string              • local        scope of prefix-list.
                       • fabric
   state               Choices:
   string              • present ←    State the action to perform. Use present to create prefix-list and absent to delete prefix-list.
                       • absent

EXAMPLES                   *pn_prefix_list-examples*

 - name: Create prefix list
   pn_prefix_list:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_scope: "local"
     state: "present"

 - name: Delete prefix list
   pn_prefix_list:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "absent"

RETURN VALUES                   *pn_prefix_list-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the prefix-list command.
   list
   stdout  always   set of responses from the prefix-list command.
   list
