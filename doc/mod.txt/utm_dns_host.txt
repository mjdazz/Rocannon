SYNOPSIS                   *utm_dns_host*

     • Create, update or destroy a dns entry in SOPHOS UTM.
     • This module needs to have the REST Ability of the UTM to be activated.

PARAMETERS                   *utm_dns_host-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   address        Default:         The IPV4 Address of the entry. Can be left empty for automatic resolving.
   -              "0.0.0.0"
   address6       Default:         The IPV6 Address of the entry. Can be left empty for automatic resolving.
   -              "::"
   comment                         An optional comment to add to the dns host object
   -
   headers                         A dictionary of additional headers to be sent to POST and PUT requests.
   dictionary                      Is needed for some modules
   hostname                        The hostname for the dns host object
   -
   interface                       The reference name of the interface to use. If not provided the default interface will be used
   -
   name                            The name of the object. Will be used to identify the entry
   - / required
   resolved         Choices:
   boolean          • no ←         whether the hostname's ipv4 address is already resolved or not
                    • yes
   resolved6        Choices:
   boolean          • no ←         whether the hostname's ipv6 address is already resolved or not
                    • yes
   state            Choices:       The desired state of the object.
   string           • absent       present will create or update an object
                    • present ←    absent will delete an object if it was present
   timeout        Default:         the timeout for the utm to resolve the ip address for the hostname again
   -              0

   string /                        The REST Endpoint of the Sophos UTM.

   utm_port       Default:         The port of the REST interface.
   integer        4444

   validate_certs   Choices:

EXAMPLES                   *utm_dns_host-examples*

 - name: Create UTM dns host entry
   utm_dns_host:
     utm_host: sophos.host.name
     utm_token: abcdefghijklmno1234
     name: TestDNSEntry
     hostname: testentry.some.tld
     state: present

 - name: Remove UTM dns host entry
   utm_dns_host:
     utm_host: sophos.host.name
     utm_token: abcdefghijklmno1234
     name: TestDNSEntry
     state: absent
