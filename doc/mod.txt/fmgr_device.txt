SYNOPSIS                   *fmgr_device*

     • Add or remove a device or list of devices from FortiManager Device Manager using JSON RPC API.

PARAMETERS                   *fmgr_device-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   adom               Default:         The ADOM the configuration should belong to.
   - / required       "root"
   blind_add            Choices:       When adding a device, module will check if it exists, and skip if it does.
   -                    • enable       If enabled, this option will stop the module from checking if it already exists, and blindly add the device.
                        • disable ←
   device_ip                           The IP of the device being added to FortiManager. Supports both IPv4 and IPv6.
   -
   device_password                     The password of the device being added to FortiManager.
   -
   device_serial                       The serial number of the device being added to FortiManager.
   -
   device_unique_name                  The desired "friendly" name of the device being added to FortiManager.
   -
   device_username                     The username of the device being added to FortiManager.
   -
   mode                 Choices:
   -                    • add ←        The desired mode of the specified object.
                        • delete

NOTES                   *fmgr_device-notes*

   Note

     • Full Documentation at https://ftnt-ansible-docs.readthedocs.io/en/latest/.

EXAMPLES                   *fmgr_device-examples*

 - name: DISCOVER AND ADD DEVICE FGT1
   fmgr_device:
     adom: "root"
     device_username: "admin"
     device_password: "admin"
     device_ip: "10.10.24.201"
     device_unique_name: "FGT1"
     device_serial: "FGVM000000117994"
     mode: "add"
     blind_add: "enable"

 - name: DISCOVER AND ADD DEVICE FGT2
   fmgr_device:
     adom: "root"
     device_username: "admin"
     device_password: "admin"
     device_ip: "10.10.24.202"
     device_unique_name: "FGT2"
     device_serial: "FGVM000000117992"
     mode: "delete"
