SYNOPSIS                   *fortios_router_static6*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify router feature and static6 category. Examples include all parameters and values
       need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_router_static6-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

NOTES                   *fortios_router_static6-notes*

   Note

EXAMPLES                   *fortios_router_static6-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure IPv6 static routing tables.
     fortios_router_static6:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       router_static6:
         bfd: "enable"
         blackhole: "enable"
         comment: "Optional comments."
         device: "<your_own_value> (source system.interface.name)"
         devindex: "7"
         distance: "8"
         dst: "<your_own_value>"
         gateway: "<your_own_value>"
         priority: "11"
         seq_num: "12"
         status: "enable"
         virtual_wan_link: "enable"
