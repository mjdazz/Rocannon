SYNOPSIS                   *snap*

     • Manages snaps packages.

PARAMETERS                   *snap-parameters*

   Parameter Choices/Defaults                                                                                Comments
   channel   Default:         Define which release of a snap is installed and tracked for updates. This option can only be specified if there is a single snap in the task.
   string    "stable"
   classic     Choices:       Confinement policy. The classic confinement allows a snap to have the same level of access to the system as "classic" packages, like those managed by APT. This option
   boolean     • no ←         corresponds to the --classic argument. This option can only be specified if there is a single snap in the task.
               • yes
   name
   - /                        Name of the snap to install or remove. Can be a list of snaps.
   required
   state       Choices:
   -           • absent       Desired state of the package.
               • present ←

EXAMPLES                   *snap-examples*

 # Install "foo" and "bar" snap
 - name: Install foo
   snap:
     name:
       - foo
       - bar

 # Remove "foo" snap
 - name: Remove foo
   snap:
     name: foo
     state: absent

 # Install a snap with classic confinement
 - name: Install "foo" with option --classic
   snap:
     name: foo
     classic: yes

 # Install a snap with from a specific channel
 - name: Install "foo" with option --channel=latest/edge
   snap:
     name: foo
     channel: latest/edge

RETURN VALUES                   *snap-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key                    Returned                                                                                Description
   channel         When snaps are installed           The channel the snaps were installed from
   string
   classic         When snaps are installed           Whether or not the snaps were installed with the classic confinement
   boolean
   cmd             When changed is true               The command that was executed on the host
   string
   snaps_installed When any snaps have been installed The list of actually installed snaps
   list
   snaps_removed   When any snaps have been removed   The list of actually removed snaps
   list
