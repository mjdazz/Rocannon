SYNOPSIS                   *fortios_firewall_service_custom*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall_service feature and custom category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_service_custom-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_service_custom-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure custom services.
     fortios_firewall_service_custom:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_service_custom:
         app_category:
          -
             id:  "4"
         app_service_type: "disable"
         application:
          -
             id:  "7"
         category: "<your_own_value> (source firewall.service.category.name)"
         check_reset_range: "disable"
         color: "10"
         comment: "Comment."
         fqdn: "<your_own_value>"
         helper: "auto"
         icmpcode: "14"
         icmptype: "15"
         iprange: "<your_own_value>"
         name: "default_name_17"
         protocol: "TCP/UDP/SCTP"
         protocol_number: "19"
         proxy: "enable"
         sctp_portrange: "<your_own_value>"
         session_ttl: "22"
         tcp_halfclose_timer: "23"
         tcp_halfopen_timer: "24"
         tcp_portrange: "<your_own_value>"
         tcp_timewait_timer: "26"
         udp_idle_timer: "27"
         udp_portrange: "<your_own_value>"
         visibility: "enable"
