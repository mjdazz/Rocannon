SYNOPSIS                   *nxos_bfd_interfaces*

     • Manages attributes of Bidirectional Forwarding Detection (BFD) on the interface.

PARAMETERS                   *nxos_bfd_interfaces-parameters*

           Parameter          Choices/Defaults                                                                        Comments
   config                                      The provided configuration
   list / elements=dictionary
              bfd               Choices:
              string            • enable       Enable/Disable Bidirectional Forwarding Detection (BFD) on the interface.
                                • disable
              echo              Choices:
              string            • enable       Enable/Disable BFD Echo functionality on the interface.
                                • disable
              name                             The name of the interface.
              string
                                Choices:
   state                        • merged ←
   string                       • replaced     The state of the configuration after module completion
                                • overridden
                                • deleted

EXAMPLES                   *nxos_bfd_interfaces-examples*

 # Using deleted

 - name: Configure interfaces
   nxos_bfd_interfaces:
     state: deleted


 # Using merged

 - name: Configure interfaces
   nxos_bfd_interfaces:
     config:
       - name: Ethernet1/1
         bfd: enable
         echo: enable
       - name: Ethernet1/2
         bfd: disable
         echo: disable
     state: merged


 # Using overridden

 - name: Configure interfaces
   nxos_bfd_interfaces:
     config:
       - name: Ethernet1/1
         bfd: enable
         echo: enable
       - name: Ethernet1/2
         bfd: disable
         echo: disable
     state: overridden


 # Using replaced

 - name: Configure interfaces
   nxos_bfd_interfaces:
     config:
       - name: Ethernet1/1
         bfd: enable
         echo: enable
       - name: Ethernet1/2
         bfd: disable
         echo: disable
     state: replaced
