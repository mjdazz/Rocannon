SYNOPSIS                   *grove*

     • The grove module sends a message for a service to a Grove.io channel.

PARAMETERS                   *grove-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   channel_token                   Token of the channel to post to.
   - / required
   icon_url                        Icon for the service
   -
   message                         Message content
   - / required
   service        Default:         Name of the service (displayed as the "user" in the message)
   -              "ansible"
   url                             Service URL for the web client
   -
   validate_certs   Choices:
   boolean          • no           If no, SSL certificates will not be validated. This should only be used on personally controlled sites using self-signed certificates.
                    • yes ←

EXAMPLES                   *grove-examples*

 - grove: >
     channel_token=6Ph62VBBJOccmtTPZbubiPzdrhipZXtg
     service=my-app
     message=deployed {{ target }}
