SYNOPSIS                   *panos_loadcfg-synopsis*

     • Load configuration on PAN-OS device

REQUIREMENTS                   *panos_loadcfg-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_loadcfg-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   commit              Choices:
   boolean             • no           commit if changed
                       • yes ←
   file                               configuration file to load
   -
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_loadcfg-examples*

 # Import and load config file from URL
   - name: import configuration
     panos_import:
       ip_address: "192.168.1.1"
       password: "admin"
       url: "{{ConfigURL}}"
       category: "configuration"
     register: result
   - name: load configuration
     panos_loadcfg:
       ip_address: "192.168.1.1"
       password: "admin"
       file: "{{result.filename}}"
