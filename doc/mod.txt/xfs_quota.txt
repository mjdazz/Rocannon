SYNOPSIS                   *xfs_quota*

     • Configure quotas on XFS filesystems.
     • Before using this module /etc/projects and /etc/projid need to be configured.

REQUIREMENTS                   *xfs_quota-requirements*

   The below requirements are needed on the host that executes this module.

     • xfsprogs

PARAMETERS                   *xfs_quota-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   bhard                              Hard blocks quota limit.
   string                             This argument supports human readable sizes.
   bsoft                              Soft blocks quota limit.
   string                             This argument supports human readable sizes.
   ihard                              Hard inodes quota limit.
   integer
   isoft                              Soft inodes quota limit.
   integer
   mountpoint                         The mount point on which to apply the quotas.
   string / required
   name                               The name of the user, group or project to apply the quota to, if other than default.
   string
   rtbhard                            Hard realtime blocks quota limit.
   string                             This argument supports human readable sizes.
   rtbsoft                            Soft realtime blocks quota limit.
   string                             This argument supports human readable sizes.
   state               Choices:       Whether to apply the limits or remove them.
   string              • present ←    When removing limit, they are set to 0, and not quite removed.
                       • absent
                       Choices:
   type                • user         The XFS quota type.
   string / required   • group
                       • project

EXAMPLES                   *xfs_quota-examples*

 - name: Set default project soft and hard limit on /opt of 1g
   xfs_quota:
     type: project
     mountpoint: /opt
     bsoft: 1g
     bhard: 1g
     state: present

 - name: Remove the default limits on /opt
   xfs_quota:
     type: project
     mountpoint: /opt
     state: absent

 - name: Set default soft user inode limits on /home of 1024 inodes and hard of 2048
   xfs_quota:
     type: user
     mountpoint: /home
     isoft: 1024
     ihard: 2048
