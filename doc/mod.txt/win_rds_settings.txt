SYNOPSIS                   *win_rds_settings*

     • Configure general settings of a Remote Desktop Gateway server.

REQUIREMENTS                   *win_rds_settings-requirements*

   The below requirements are needed on the host that executes this module.

     • Windows Server 2008R2 (6.1) or higher.
     • The Windows Feature “RDS-Gateway” must be enabled.

PARAMETERS                   *win_rds_settings-parameters*

                 Parameter               Choices/Defaults                                                                   Comments
   certificate_hash                                        Certificate hash (thumbprint) for the Remote Desktop Gateway server. The certificate hash is the unique identifier for the certificate.
   string
   enable_only_messaging_capable_clients   Choices:
   boolean                                 • no            If enabled, only clients that support logon messages and administrator messages can connect.
                                           • yes
   max_connections                                         The maximum number of connections allowed.
   integer                                                 If set to 0, no new connections are allowed.
                                                           If set to -1, the number of connections is unlimited.
                                           Choices:        Specifies whether to use SSL Bridging.
   ssl_bridging                            • https_http    none: no SSL bridging.
   string                                  • https_https   https_http: HTTPS-HTTP bridging.
                                           • none          https_https: HTTPS-HTTPS bridging.

EXAMPLES                   *win_rds_settings-examples*

 - name: Configure the Remote Desktop Gateway
   win_rds_settings:
     certificate_hash: B0D0FA8408FC67B230338FCA584D03792DA73F4C
     max_connections: 50
   notify:
     - Restart TSGateway service
