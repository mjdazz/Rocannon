SYNOPSIS                   *helm*

     • Install, upgrade, delete and list packages with the Helm package manager.

REQUIREMENTS                   *helm-requirements*

   The below requirements are needed on the host that executes this module.

     • pyhelm
     • grpcio

PARAMETERS                   *helm-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   chart         Default:         A map describing the chart to install. See examples for available options.
   -             {}
   disable_hooks   Choices:
   boolean         • no ←         Whether to disable hooks during the uninstall process.
                   • yes
   host          Default:         Tiller's server host.
   -             "localhost"
   name                           Release name to manage.
   -
   namespace     Default:         Kubernetes namespace where the chart should be installed.
   -             "default"
   port          Default:         Tiller's server port.
   -             44134
                   Choices:
   state           • absent       Whether to install present, remove absent, or purge purged a package.
   -               • purged
                   • present ←
   values        Default:         A map of value options for the chart.
   -             {}

EXAMPLES                   *helm-examples*

 - name: Install helm chart
   helm:
     host: localhost
     chart:
       name: memcached
       version: 0.4.0
       source:
         type: repo
         location: https://kubernetes-charts.storage.googleapis.com
     state: present
     name: my-memcached
     namespace: default

 - name: Uninstall helm chart
   helm:
     host: localhost
     state: absent
     name: my-memcached

 - name: Install helm chart from a git repo
   helm:
     host: localhost
     chart:
       source:
         type: git
         location: https://github.com/user/helm-chart.git
     state: present
     name: my-example
     namespace: default

 - name: Install helm chart from a git repo specifying path
   helm:
     host: localhost
     chart:
       source:
         type: git
         location: https://github.com/helm/charts.git
         path: stable/memcached
     state: present
     name: my-memcached
     namespace: default
