SYNOPSIS                   *fortios_firewall_internet_service*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall feature and internet_service category. Examples include all parameters
       and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_internet_service-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_internet_service-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Show Internet Service application.
     fortios_firewall_internet_service:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_internet_service:
         database: "isdb"
         direction: "src"
         entry:
          -
             id:  "6"
             ip_number: "7"
             ip_range_number: "8"
             port: "9"
             protocol: "10"
         icon_id: "11"
         id:  "12"
         name: "default_name_13"
         offset: "14"
         reputation: "15"
         sld_id: "16"
