SYNOPSIS                   *hcloud_image_info*

     • Gather infos about your Hetzner Cloud images.
     • This module was called hcloud_location_facts before Ansible 2.9, returning ansible_facts and hcloud_location_facts. Note that the hcloud_image_info module no longer returns ansible_facts and
       the value was renamed to hcloud_image_info!

   Aliases: hcloud_image_facts

REQUIREMENTS                   *hcloud_image_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_image_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the image you want to get.
   integer
   label_selector                                   The label selector for the images you want to get.
   string
   name                                             The name of the image you want to get.
   string
                       Choices:
   type                • system ←                   The label selector for the images you want to get.
   string              • snapshot
                       • backup

RETURN VALUES                   *hcloud_image_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

          Key        Returned                                                                               Description
   hcloud_image_info always   The image infos as list
   complex
       description            Detail description of the image
       string        always   Sample:
                              Ubuntu 18.04 Standard 64 bit
       id                     Numeric identifier of the image
       integer       always   Sample:
                              1937415
       labels        always   User-defined labels (key-value pairs)
       dictionary
       name                   Name of the image
       string        always   Sample:
                              ubuntu-18.04
       os_flavor              OS flavor of the image
       string        always   Sample:
                              ubuntu
       os_version             OS version of the image
       string        always   Sample:
                              18.04
       status                 Status of the image
       string        always   Sample:
                              available
       type                   Type of the image
       string        always   Sample:
                              system
