SYNOPSIS                   *ip_netns*

     • Create or delete network namespaces using the ip command.

REQUIREMENTS                   *ip_netns-requirements*

   The below requirements are needed on the host that executes this module.

     • ip

PARAMETERS                   *ip_netns-parameters*

   Parameter Choices/Defaults                                                                                Comments
   name                       Name of the namespace
   -
   state       Choices:
   -           • present ←    Whether the namespace should exist
               • absent

EXAMPLES                   *ip_netns-examples*

 # Create a namespace named mario
 - name: Create a namespace named mario
   namespace:
     name: mario
     state: present
 - name: Delete a namespace named luigi
   namespace:
     name: luigi
     state: absent
