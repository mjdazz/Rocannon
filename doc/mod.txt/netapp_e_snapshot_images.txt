SYNOPSIS                   *netapp_e_snapshot_images*

     • Create and delete snapshots images on snapshot groups for NetApp E-series storage arrays.
     • Only the oldest snapshot image can be deleted so consistency is preserved.
     • Related: Snapshot volumes are created from snapshot images.

PARAMETERS                   *netapp_e_snapshot_images-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   api_password                    The password to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_url                         The url to the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_username                    The username to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   snapshot_group                  The name of the snapshot group in which you want to create a snapshot image.
   - / required
   state            Choices:
   - / required     • create       Whether a new snapshot image should be created or oldest be deleted.
                    • remove
   validate_certs Default:         Should https certificates be validated?
   -              "yes"

EXAMPLES                   *netapp_e_snapshot_images-examples*

 - name: Create Snapshot
   netapp_e_snapshot_images:
     ssid: "{{ ssid }}"
     api_url: "{{ netapp_api_url }}"
     api_username: "{{ netapp_api_username }}"
     api_password: "{{ netapp_api_password }}"
     validate_certs: "{{ validate_certs }}"
     snapshot_group: "3300000060080E5000299C24000005B656D9F394"
     state: 'create'

RETURN VALUES                   *netapp_e_snapshot_images-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key        Returned                                                                                   Description
   image_id                  ID of snapshot image
   string   state == created Sample:
                             3400000060080E5000299B640063074057BC5C5E
   msg                       State of operation
   string   always           Sample:
                             Created snapshot image
