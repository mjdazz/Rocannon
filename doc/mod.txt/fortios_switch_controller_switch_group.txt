SYNOPSIS                   *fortios_switch_controller_switch_group*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify switch_controller feature and switch_group category. Examples include all
       parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_switch_controller_switch_group-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

PARAMETERS                   *fortios_switch_controller_switch_group-parameters*

             Parameter            Choices/Defaults                                                                      Comments
   host                                            FortiOS or FortiGate IP address.
   string
   https                            Choices:
   boolean                          • no           Indicates if the requests towards FortiGate must use HTTPS protocol.
                                    • yes ←
   password                       Default:         FortiOS or FortiGate password.
   string                         ""
   ssl_verify                       Choices:
   boolean                          • no           Ensures FortiGate certificate must be verified by a proper CA.
                                    • yes ←
   state                            Choices:
   string / required                • present      Indicates whether to create or remove the object.
                                    • absent
   switch_controller_switch_group Default:         Configure FortiSwitch switch groups.
   dictionary                     null
        description                                Optional switch group description.
        string
        members                                    FortiSwitch members belonging to this switch group.
        list
             name                                  Managed device ID. Source switch-controller.managed-switch.switch-id.
             string / required
        name                                       Switch group name.
        string / required
   username                                        FortiOS or FortiGate username.
   string
   vdom                           Default:         Virtual domain, among those defined previously. A vdom is a virtual instance of the FortiGate that can be configured and used as a different unit.
   string                         "root"

NOTES                   *fortios_switch_controller_switch_group-notes*

   Note

     • Requires fortiosapi library developed by Fortinet
     • Run as a local_action in your playbook

EXAMPLES                   *fortios_switch_controller_switch_group-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure FortiSwitch switch groups.
     fortios_switch_controller_switch_group:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       switch_controller_switch_group:
         description: "<your_own_value>"
         members:
          -
             name: "default_name_5 (source switch-controller.managed-switch.switch-id)"
         name: "default_name_6"
