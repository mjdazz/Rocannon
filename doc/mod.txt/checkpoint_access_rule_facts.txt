SYNOPSIS                   *checkpoint_access_rule_facts*

     • Get access rules objects facts on Check Point devices. All operations are performed over Web Services API.

PARAMETERS                   *checkpoint_access_rule_facts-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   layer                              Layer the access rule is attached to.
   string / required
   name                               Name of the access rule. If not provided, UID is required.
   string
   uid                                UID of the access rule. If not provided, name is required.
   string

EXAMPLES                   *checkpoint_access_rule_facts-examples*

 - name: Get access rule facts
   checkpoint_access_rule_facts:
     layer: Network
     name: "Drop attacker"
