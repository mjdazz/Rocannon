SYNOPSIS                   *fortios_system_sdn_connector*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and sdn_connector category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_sdn_connector-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8
