SYNOPSIS                   *ucs_dns_server*

     • Configure DNS servers on Cisco UCS Manager.
     • Examples can be used with the UCS Platform Emulator.

REQUIREMENTS                   *ucs_dns_server-requirements*

   The below requirements are needed on the host that executes this module.

     • ucsmsdk

PARAMETERS                   *ucs_dns_server-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   delegate_to       Default:         Where the module will be run
   string            "localhost"
                                      A user-defined description of the DNS server.
   description                        Enter up to 256 characters.
   string                             You can use any characters or spaces except the following:
                                      ` (accent mark), (backslash), ^ (carat), " (double quote), = (equal sign), > (greater than), < (less than), or ' (single quote).
                                      aliases: descr
                                      DNS server IP address.
   dns_server                         Enter a valid IPV4 Address.
   string                             UCS Manager supports up to 4 DNS Servers
                                      aliases: name
   hostname                           IP address or hostname of Cisco UCS Manager.
   string / required
   password                           Password for Cisco UCS Manager authentication.
   string / required
   port                               Port number to be used during connection (by default uses 443 for https and 80 for http connection).
   integer
   proxy                              If use_proxy is no, specfies proxy to be used for connection. e.g. 'http://proxy.xy.z:8080'
   string
   state               Choices:       If absent, will remove a DNS server.
   string              • absent       If present, will add or update a DNS server.
                       • present ←
   use_proxy           Choices:
   boolean             • no           If no, will not use the proxy as defined by system environment variable.
                       • yes ←
   use_ssl             Choices:
   boolean             • no           If no, an HTTP connection will be used instead of the default HTTPS connection.
                       • yes ←
   username          Default:         Username for Cisco UCS Manager authentication.
   string            "admin"

EXAMPLES                   *ucs_dns_server-examples*

 - name: Configure DNS server
   ucs_dns_server:
     hostname: 172.16.143.150
     username: admin
     password: password
     dns_server: 10.10.10.10
     description: DNS Server IP address
     state: present
     delegate_to: localhost

 - name: Remove DNS server
   ucs_dns_server:
     hostname: 172.16.143.150
     username: admin
     password: password
     dns_server: 10.10.10.10
     state: absent
     delegate_to: localhost
