SYNOPSIS                   *ipify_facts*

     • If behind NAT and need to know the public IP of your internet gateway.

PARAMETERS                   *ipify_facts-parameters*

     Parameter        Choices/Defaults                                                                              Comments
   api_url        Default:                 URL of the ipify.org API service.
   string         "https://api.ipify.org/" ?format=json will be appended per default.
   timeout        Default:                 HTTP connection timeout in seconds.
   integer        10
   validate_certs   Choices:
   boolean          • no                   When set to NO, SSL certificates will not be validated.
   added in 2.4     • yes ←

NOTES                   *ipify_facts-notes*

   Note

     • Visit https://www.ipify.org to get more information.

EXAMPLES                   *ipify_facts-examples*

 # Gather IP facts from ipify.org
 - name: Get my public IP
   ipify_facts:

 # Gather IP facts from your own ipify service endpoint with a custom timeout
 - name: Get my public IP
   ipify_facts:
     api_url: http://api.example.com/ipify
     timeout: 20

RETURN VALUES                   *ipify_facts-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key       Returned                                                                                Description
   ipify_public_ip          Public IP of the internet gateway.
   string          success  Sample:
                            1.2.3.4
