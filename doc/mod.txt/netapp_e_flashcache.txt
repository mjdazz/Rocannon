SYNOPSIS                   *netapp_e_flashcache*

     • Create or remove SSD caches on a NetApp E-Series storage array.

PARAMETERS                   *netapp_e_flashcache-parameters*

     Parameter     Choices/Defaults                                                                              Comments
   api_password                      The password to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_url                           The url to the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_username                      The username to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   cache_size_min                    The minimum size (in size_units) of the ssd cache. The cache will be expanded if this exceeds the current size of the cache.
   -
   disk_count                        The minimum number of disks to use for building the cache. The cache will be expanded if this number exceeds the number of disks already in place
   -
                    Choices:
   io_type          • filesystem ←   The type of workload to optimize the cache for.
   -                • database
                    • media
   name                              The name of the SSD cache to manage
   - / required
                    Choices:
                    • bytes
                    • b
                    • kb
   size_unit        • mb
   -                • gb ←           The unit to be applied to size arguments
                    • tb
                    • pb
                    • eb
                    • zb
                    • yb
   ssid                              The ID of the array to manage (as configured on the web services proxy).
   - / required
   state            Choices:
   - / required     • present ←      Whether the specified SSD cache should exist or not.
                    • absent
   validate_certs   Choices:
   boolean          • no             Should https certificates be validated?
                    • yes ←

EXAMPLES                   *netapp_e_flashcache-examples*

 - name: Flash Cache
   netapp_e_flashcache:
     ssid: "{{ ssid }}"
     api_url: "{{ netapp_api_url }}"
     api_username: "{{ netapp_api_username }}"
     api_password: "{{ netapp_api_password }}"
     validate_certs: "{{ netapp_api_validate_certs }}"
     name: SSDCacheBuiltByAnsible
