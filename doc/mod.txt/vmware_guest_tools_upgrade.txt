SYNOPSIS                   *vmware_guest_tools_upgrade*

     • This module upgrades the VMware Tools on Windows and Linux guests.

REQUIREMENTS                   *vmware_guest_tools_upgrade-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_guest_tools_upgrade-notes*

   Note

     • In order to upgrade VMTools, please power on virtual machine before hand - either ‘manually’ or using module vmware_guest_powerstate.

EXAMPLES                   *vmware_guest_tools_upgrade-examples*

 - name: Upgrade VMware Tools using uuid
   vmware_guest_tools_upgrade:
     hostname: "{{ vcenter_hostname }}"
     username: "{{ vcenter_username }}"
     password: "{{ vcenter_password }}"
     datacenter: "{{ datacenter_name }}"
     uuid: 421e4592-c069-924d-ce20-7e7533fab926
   delegate_to: localhost

 - name: Upgrade VMware Tools using MoID
   vmware_guest_tools_upgrade:
     hostname: "{{ vcenter_hostname }}"
     username: "{{ vcenter_username }}"
     password: "{{ vcenter_password }}"
     datacenter: "{{ datacenter_name }}"
     moid: vm-42
   delegate_to: localhost
