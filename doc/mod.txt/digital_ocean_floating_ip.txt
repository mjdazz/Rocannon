SYNOPSIS                   *digital_ocean_floating_ip*

     • Create/delete/assign a floating IP.

REQUIREMENTS                   *digital_ocean_floating_ip-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6

PARAMETERS                   *digital_ocean_floating_ip-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   droplet_id                    The Droplet that the Floating IP has been assigned to.
   -
   ip                            Public IP address of the Floating IP. Used to remove an IP
   -
   oauth_token                   DigitalOcean OAuth token.
   - / required
   region                        The region that the Floating IP is reserved to.
   -
   state          Choices:
   -              • present ←    Indicate desired state of the target.
                  • absent

NOTES                   *digital_ocean_floating_ip-notes*

   Note

     • Version 2 of DigitalOcean API is used.

EXAMPLES                   *digital_ocean_floating_ip-examples*

 - name: "Create a Floating IP in region lon1"
   digital_ocean_floating_ip:
     state: present
     region: lon1

 - name: "Create a Floating IP assigned to Droplet ID 123456"
   digital_ocean_floating_ip:
     state: present
     droplet_id: 123456

 - name: "Delete a Floating IP with ip 1.2.3.4"
   digital_ocean_floating_ip:
     state: absent
     ip: "1.2.3.4"
