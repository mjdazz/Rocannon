SYNOPSIS                   *win_rds_rap*

     • Creates, removes and configures a Remote Desktop resource authorization policy (RD RAP).
     • A RD RAP allows you to specify the network resources (computers) that users can connect to remotely through a Remote Desktop Gateway server.

REQUIREMENTS                   *win_rds_rap-requirements*

   The below requirements are needed on the host that executes this module.

     • Windows Server 2008R2 (6.1) or higher.
     • The Windows Feature “RDS-Gateway” must be enabled.

                         • allow_any                 allow_any: Allow users to connect to any network resource.
   description                                       Optional description of the resource authorization policy.
   string

   - / required
                         Choices:                    The state of resource authorization policy.
   state                 • absent                    If absent will ensure the policy is removed.
   string                • disabled                  If present will ensure the policy is configured and exists.

EXAMPLES                   *win_rds_rap-examples*

 - name: Create a new RDS RAP
   win_rds_rap:
     name: My RAP
     description: Allow all users to connect to any resource through ports 3389 and 3390
     user_groups:
       - BUILTIN\users
     computer_group_type: allow_any
     allowed_ports:
       - 3389
       - 3390
     state: enabled
