SYNOPSIS                   *win_ping*

     • Checks management connectivity of a windows host.
     • This is NOT ICMP ping, this is just a trivial test module.
     • For non-Windows targets, use the ping module instead.
     • For Network targets, use the net_ping module instead.

PARAMETERS                   *win_ping-parameters*

   Parameter Choices/Defaults                                                                                Comments
   data      Default:         Alternate data to return instead of 'pong'.
   string    "pong"           If this parameter is set to crash, the module will cause an exception.

SEE ALSO                   *win_ping-see also*

   See also

   ping – Try to connect to host, verify a usable python and return pong on success
   The official documentation on the ping module.

EXAMPLES                   *win_ping-examples*

 # Test connectivity to a windows host
 # ansible winserver -m win_ping

 - name: Induce an exception to see what happens
   win_ping:
