SYNOPSIS                   *rundeck_acl_policy*

     • Create, update and remove Rundeck ACL policies through HTTP API.

PARAMETERS                   *rundeck_acl_policy-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   api_version  Default:         Sets the API version used by module.
   -            14               API version must be at least 14.
   name                          Sets the project name.
   - / required
   policy                        Sets the ACL policy content.
   -                             ACL policy content is a YAML object as described in http://rundeck.org/docs/man5/aclpolicy.html.
                                 It can be a YAML string or a pure Ansible inventory YAML object.
   project                       Sets the project which receive the ACL policy.
   -                             If unset, it's a system ACL policy.
   state          Choices:
   -              • present ←    Create or remove Rundeck project.
                  • absent
   token                         Sets the token to authenticate against Rundeck API.
   - / required
   url                           Sets the rundeck instance URL.
   - / required

EXAMPLES                   *rundeck_acl_policy-examples*

 - name: Create or update a rundeck ACL policy in project Ansible
   rundeck_acl_policy:
     name: "Project_01"
     api_version: 18
     url: "https://rundeck.example.org"
     token: "mytoken"
     state: present
     project: "Ansible"
     policy:
       description: "my policy"
       context:
         application: rundeck
       for:
         project:
           - allow: read
       by:
         group: "build"

 - name: Remove a rundeck system policy
   rundeck_acl_policy:
     name: "Project_02"
     url: "https://rundeck.example.org"
     token: "mytoken"
     state: absent

RETURN VALUES                   *rundeck_acl_policy-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key        Returned                                                                               Description
   after            success  Dictionary containing ACL policy informations after modification.
   dictionary
   before           success  Dictionary containing ACL policy informations before modification.
   dictionary
   rundeck_response failed   Rundeck response when a failure occurs.
   string
