SYNOPSIS                   *purefa_smtp*

     • Set or erase configuration for the SMTP settings.
     • If username/password are set this will always force a change as there is no way to see if the password is different from the current SMTP configuration.
     • Pure Storage Ansible Team (@sdodsley) <pure-ansible-team@purestorage.com>

REQUIREMENTS                   *purefa_smtp-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_smtp-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashArray API token for admin privileged user.
   string / required
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   password                           The SMTP password.
   string
   relay_host                         IPv4 or IPv6 address or FQDN. A port number may be appended.
   string
   sender_domain                      Domain name.
   string
   state               Choices:
   string              • absent       Set or delete SMTP configuration
                       • present ←
   user                               The SMTP username.
   string

NOTES                   *purefa_smtp-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_smtp-examples*

 - name: Delete existing SMTP settings
   purefa_smtp:
     state: absent
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592

 - name: Set SMTP settings
   purefa_smtp:
     sender_domain: purestorage.com
     password: account_password
     user: smtp_account
     relay_host: 10.2.56.78:2345
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
