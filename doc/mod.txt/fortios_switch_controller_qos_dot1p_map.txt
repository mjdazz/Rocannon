SYNOPSIS                   *fortios_switch_controller_qos_dot1p_map*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify switch_controller_qos feature and dot1p_map category. Examples include all
       parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_switch_controller_qos_dot1p_map-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_switch_controller_qos_dot1p_map-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure FortiSwitch QoS 802.1p.
     fortios_switch_controller_qos_dot1p_map:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       switch_controller_qos_dot1p_map:
         description: "<your_own_value>"
         name: "default_name_4"
         priority_0: "queue-0"
         priority_1: "queue-0"
         priority_2: "queue-0"
         priority_3: "queue-0"
         priority_4: "queue-0"
         priority_5: "queue-0"
         priority_6: "queue-0"
         priority_7: "queue-0"
