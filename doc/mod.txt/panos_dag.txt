SYNOPSIS                   *panos_dag-synopsis*

     • Create a dynamic address group object in the firewall used for policy rules

REQUIREMENTS                   *panos_dag-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_dag-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   commit              Choices:
   boolean             • no           commit if changed
                       • yes ←
   dag_filter                         dynamic filter user by the dynamic address group
   - / required
   dag_name                           name of the dynamic address group
   - / required
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_dag-examples*

 - name: dag
   panos_dag:
     ip_address: "192.168.1.1"
     password: "admin"
     dag_name: "dag-1"
     dag_filter: "'aws-tag.aws:cloudformation:logical-id.ServerInstance' and 'instanceState.running'"
