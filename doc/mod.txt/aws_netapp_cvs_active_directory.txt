SYNOPSIS                   *aws_netapp_cvs_active_directory*

     • Create, Update, Delete ActiveDirectory on AWS Cloud Volumes Service.

PARAMETERS                   *aws_netapp_cvs_active_directory-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_key                            The access key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   api_url                            The url to the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   DNS                                DNS server address for the Active Directory domain
   string                             Required when state=present
                                      Required when state=present, to modify ActiveDirectory properties.
   domain                             Name of the Active Directory domain
   string / required
   netBIOS                            NetBIOS name of the server.
   string
   password                           Password of the Active Directory domain administrator
   string
   region                             The region to which the Active Directory credentials are associated.
   string / required
   secret_key                         The secret_key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   state               Choices:
   string / required   • present      Whether the specified ActiveDirectory should exist or not.
                       • absent
   username                           Username of the Active Directory domain administrator
   string
   validate_certs      Choices:
   boolean             • no           Should https certificates be validated?
                       • yes ←

NOTES                   *aws_netapp_cvs_active_directory-notes*

   Note

     • The modules prefixed with aws_cvs_netapp are built to Manage AWS Cloud Volume Service .

EXAMPLES                   *aws_netapp_cvs_active_directory-examples*

 - name: Create Active Directory
   aws_netapp_cvs_active_directory.py:
     state: present
     region: us-east-1
     DNS: 101.102.103.123
     domain: mydomain.com
     password: netapp1!
     netBIOS: testing
     username: user1
     api_url : My_CVS_Hostname
     api_key: My_API_Key
     secret_key : My_Secret_Key

 - name: Update Active Directory
   aws_netapp_cvs_active_directory.py:
     state: present
     region: us-east-1
     DNS: 101.102.103.123
     domain: mydomain.com
     password: netapp2!
     netBIOS: testingBIOS
     username: user2
     api_url : My_CVS_Hostname
     api_key: My_API_Key
     secret_key : My_Secret_Key

 - name: Delete Active Directory
   aws_netapp_cvs_active_directory.py:
     state: absent
     region: us-east-1
     domain: mydomain.com
     api_url : My_CVS_Hostname
     api_key: My_API_Key
     secret_key : My_Secret_Key
