SYNOPSIS                   *avi_autoscalelaunchconfig*

     • This module is used to configure AutoScaleLaunchConfig object
     • more examples at https://github.com/avinetworks/devops

   The below requirements are needed on the host that executes this module.

     • avisdk

EXAMPLES                   *avi_autoscalelaunchconfig-examples*

 - name: Create an Autoscale Launch configuration.
   avi_autoscalelaunchconfig:
     controller: '{{ controller }}'
     username: '{{ username }}'
     password: '{{ password }}'
     image_id: default
     name: default-autoscalelaunchconfig
     tenant_ref: admin
