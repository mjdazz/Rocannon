SYNOPSIS                   *scaleway_user_data*

     • This module manages user_data on compute instances on Scaleway.
     • It can be used to configure cloud-init for instance

PARAMETERS                   *scaleway_user_data-parameters*

      Parameter          Choices/Defaults                                                                             Comments
   api_timeout      Default:                   HTTP timeout to Scaleway API in seconds.
   integer          30                         aliases: timeout
   api_token                                   Scaleway OAuth token.
   string                                      aliases: oauth_token
   api_url          Default:                   Scaleway API URL.
   string           "https://api.scaleway.com" aliases: base_url
   query_parameters Default:                   List of parameters passed to the query string.
   dictionary       {}
                      Choices:
   region             • ams1
   - / required       • EMEA-NL-EVS            Scaleway compute zone
                      • par1
                      • EMEA-FR-PAR1
   server_id                                   Scaleway Compute instance ID of the server
   - / required
   user_data                                   User defined data. Typically used with `cloud-init`.
   -                                           Pass your cloud-init script here as a string
   validate_certs     Choices:
   boolean            • no                     Validate SSL certs of the Scaleway API.
                      • yes ←

NOTES                   *scaleway_user_data-notes*

   Note

     • Also see the API documentation on https://developer.scaleway.com/
     • If api_token is not set within the module, the following environment variables can be used in decreasing order of precedence SCW_TOKEN, SCW_API_KEY, SCW_OAUTH_TOKEN or SCW_API_TOKEN.
     • If one wants to use a different api_url one can also set the SCW_API_URL environment variable.

EXAMPLES                   *scaleway_user_data-examples*

 - name: Update the cloud-init
   scaleway_user_data:
     server_id: '5a33b4ab-57dd-4eb6-8b0a-d95eb63492ce'
     region: ams1
     user_data:
       cloud-init: 'final_message: "Hello World!"'
