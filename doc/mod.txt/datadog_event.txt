SYNOPSIS                   *datadog_event*

     • Allows to post events to Datadog (www.datadoghq.com) service.
     • Uses http://docs.datadoghq.com/api/#events API.

PARAMETERS                   *datadog_event-parameters*

      Parameter        Choices/Defaults                                                                             Comments
   aggregation_key                          An arbitrary string to use for aggregation.
   -
                     Choices:
   alert_type        • error
   -                 • warning              Type of alert.
                     • info ←
                     • success
   api_key                                  Your DataDog API key.
   - / required
   app_key                                  Your DataDog app key.
   - / required
   date_happened   Default:                 POSIX timestamp of the event.
   -               "now"                    Default value is now.
   host            Default:
   -               "{{ ansible_hostname }}" Host name to associate with the event.
   added in 2.4
   priority          Choices:
   -                 • normal ←             The priority of the event.
                     • low
   tags                                     Comma separated list of tags to apply to the event.
   -
   text                                     The body of the event.
   - / required
   title                                    The event title.
   - / required
   validate_certs    Choices:
   boolean           • no                   If no, SSL certificates will not be validated. This should only be used on personally controlled sites using self-signed certificates.
                     • yes ←

EXAMPLES                   *datadog_event-examples*

 # Post an event with low priority
 - datadog_event:
     title: Testing from ansible
     text: Test
     priority: low
     api_key: 9775a026f1ca7d1c6c5af9d94d9595a4
     app_key: j4JyCYfefWHhgFgiZUqRm63AXHNZQyPGBfJtAzmN
 # Post an event with several tags
 - datadog_event:
     title: Testing from ansible
     text: Test
     api_key: 9775a026f1ca7d1c6c5af9d94d9595a4
     app_key: j4JyCYfefWHhgFgiZUqRm63AXHNZQyPGBfJtAzmN
     tags: 'aa,bb,#host:{{ inventory_hostname }}'
