SYNOPSIS                   *bearychat*

     • The bearychat module sends notifications to https://bearychat.com via the Incoming Robot integration.

PARAMETERS                   *bearychat-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   attachments                   Define a list of attachments. For more information, see https://github.com/bearyinnovative/bearychat-tutorial/blob/master/robots/incoming.md#attachments
   -
   channel                       Channel to send the message to. If absent, the message goes to the default channel selected by the url.
   -
   markdown       Choices:
   boolean        • no           If yes, text will be parsed as markdown.
                  • yes ←
   text                          Message to send.
   -
   url                           BearyChat WebHook URL. This authenticates you to the bearychat service. It looks like https://hook.bearychat.com/=ae2CF/incoming/e61bd5c57b164e04b11ac02e66f47f60.
   - / required

EXAMPLES                   *bearychat-examples*

 - name: Send notification message via BearyChat
   local_action:
     module: bearychat
     url: |
       https://hook.bearychat.com/=ae2CF/incoming/e61bd5c57b164e04b11ac02e66f47f60
     text: "{{ inventory_hostname }} completed"

 - name: Send notification message via BearyChat all options
   local_action:
     module: bearychat
     url: |
       https://hook.bearychat.com/=ae2CF/incoming/e61bd5c57b164e04b11ac02e66f47f60
     text: "{{ inventory_hostname }} completed"
     markdown: no
     channel: "#ansible"
     attachments:
       - title: "Ansible on {{ inventory_hostname }}"
         text: "May the Force be with you."
         color: "#ffffff"
         images:
           - http://example.com/index.png
