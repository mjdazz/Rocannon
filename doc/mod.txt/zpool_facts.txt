SYNOPSIS                   *zpool_facts*

     • Gather facts from ZFS pool properties.

PARAMETERS                   *zpool_facts-parameters*

   Parameter  Choices/Defaults                                                                                Comments
   name                        ZFS pool name.
   -                           aliases: pool, zpool
   parsable     Choices:
   boolean      • no ←         Specifies if property values should be displayed in machine friendly format.
                • yes
   properties Default:         Specifies which dataset properties should be queried in comma-separated format. For more information about dataset properties, check zpool(1M) man page.
   -          "all"            aliases: props

EXAMPLES                   *zpool_facts-examples*

 # Gather facts about ZFS pool rpool
 - zpool_facts: pool=rpool

 # Gather space usage about all imported ZFS pools
 - zpool_facts: properties='free,size'

 - debug: msg='ZFS pool {{ item.name }} has {{ item.free }} free space out of {{ item.size }}.'
   with_items: '{{ ansible_zfs_pools }}'
