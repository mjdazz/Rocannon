SYNOPSIS                   *purefb_snap*

     • Create or delete volumes and filesystem snapshots on Pure Storage FlashBlades.

REQUIREMENTS                   *purefb_snap-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purity_fb >= 1.1

PARAMETERS                   *purefb_snap-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashBlade API token for admin privileged user.
   string
   eradicate           Choices:
   boolean             • no ←         Define whether to eradicate the snapshot on delete or leave in trash.
                       • yes
   fb_url                             FlashBlade management IP address or Hostname.
   string
   name                               The name of the source filesystem.
   string / required
   state               Choices:
   string              • absent       Define whether the filesystem snapshot should exist or not.
                       • present ←
   suffix                             Suffix of snapshot name.
   string

NOTES                   *purefb_snap-notes*

   Note

     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_snap-examples*

 - name: Create snapshot foo.ansible
   purefb_snap:
     name: foo
     suffix: ansible
     fb_url: 10.10.10.2
     fb_api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: present

 - name: Delete snapshot named foo.snap
   purefb_snap:
     name: foo
     suffix: snap
     fb_url: 10.10.10.2
     fb_api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: absent

 - name: Recover deleted snapshot foo.ansible
   purefb_snap:
     name: foo
     suffix: ansible
     fb_url: 10.10.10.2
     fb_api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: present

 - name: Eradicate snapshot named foo.snap
   purefb_snap:
     name: foo
     suffix: snap
     eradicate: true
     fb_url: 10.10.10.2
     fb_api_token: e31060a7-21fc-e277-6240-25983c6c4592
     state: absent
