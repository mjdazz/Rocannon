SYNOPSIS                   *purefb_smtp*

     • Configure SMTP for a Pure Storage FlashBlade.
     • Whilst there can be no relay host, a sender domain must be configured.

REQUIREMENTS                   *purefb_smtp-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purity_fb >= 1.1

PARAMETERS                   *purefb_smtp-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashBlade API token for admin privileged user.
   string
   domain                             Domain name for alert messages
   string / required
   fb_url                             FlashBlade management IP address or Hostname.
   string
   host                               Relay server name
   string

NOTES                   *purefb_smtp-notes*

   Note

     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_smtp-examples*

 - name: Configure SMTP settings
   purefb_smtp:
     host: hostname
     domain: xyz.com
     fb_url: 10.10.10.2
     api_token: T-9f276a18-50ab-446e-8a0c-666a3529a1b6
