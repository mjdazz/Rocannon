SYNOPSIS                   *podman_image_info*

     • Gather info about images using podman

PARAMETERS                   *podman_image_info-parameters*

   Parameter  Choices/Defaults                                                                                Comments
   executable Default:         Path to podman executable if it is not in the $PATH on the machine running podman
   string     "podman"
   name                        List of tags or UID to gather info about. If no name is given return info about all images.
   -

NOTES                   *podman_image_info-notes*

   Note

     • Podman may required elevated privileges in order to run properly.

EXAMPLES                   *podman_image_info-examples*

 - name: Gather info for all images
   podman_image_info:

 - name: Gather info on a specific image
   podman_image_info:
     name: nginx

 - name: Gather info on several images
   podman_image_info:
     name:
       - redis
       - quay.io/bitnami/wildfly
