SYNOPSIS                   *win_iis_webapplication*

     • Creates, removes, and configures IIS web applications.

PARAMETERS                   *win_iis_webapplication-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   application_pool                   The application pool in which the new site executes.
   string
   name                               Name of the web application.
   string / required
   physical_path                      The physical path on the remote host to use for the new application.
   string                             The specified folder must already exist.
   site                               Name of the site on which the application is created.
   string / required
   state               Choices:
   string              • absent       State of the web application.
                       • present ←

EXAMPLES                   *win_iis_webapplication-examples*

 - name: Add ACME webapplication on IIS.
   win_iis_webapplication:
     name: api
     site: acme
     state: present
     physical_path: C:\apps\acme\api
