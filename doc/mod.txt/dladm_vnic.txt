SYNOPSIS                   *dladm_vnic*

     • Create or delete VNICs on Solaris/illumos systems.

PARAMETERS                   *dladm_vnic-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   link                          VNIC underlying link name.
   - / required
   mac          Default:         Sets the VNIC's MAC address. Must be valid unicast MAC address.
   -            "no"             aliases: macaddr
   name                          VNIC name.
   - / required
   state          Choices:
   -              • present ←    Create or delete Solaris/illumos VNIC.
                  • absent
   temporary      Choices:
   boolean        • no ←         Specifies that the VNIC is temporary. Temporary VNICs do not persist across reboots.
                  • yes
   vlan         Default:         Enable VLAN tagging for this VNIC. The VLAN tag will have id vlan.
   -            "no"             aliases: vlan_id

EXAMPLES                   *dladm_vnic-examples*

 # Create 'vnic0' VNIC over 'bnx0' link
 - dladm_vnic:
     name: vnic0
     link: bnx0
     state: present

 # Create VNIC with specified MAC and VLAN tag over 'aggr0'
 - dladm_vnic:
     name: vnic1
     link: aggr0
     mac: '00:00:5E:00:53:23'
     vlan: 4

 # Remove 'vnic0' VNIC
 - dladm_vnic:
     name: vnic0
     link: bnx0
     state: absent

RETURN VALUES                   *dladm_vnic-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key         Returned                                                                                   Description
   link                          VNIC underlying link name
   string    always              Sample:
                                 igb0
   mac                           MAC address to use for VNIC
   string    if mac is specified Sample:
                                 00:00:5E:00:53:42
   name                          VNIC name
   string    always              Sample:
                                 vnic0
   state                         state of the target
   string    always              Sample:
                                 present
   temporary                     VNIC's persistence
   boolean   always              Sample:
                                 True
   vlan                          VLAN to use for VNIC
   integer   success             Sample:
                                 42
