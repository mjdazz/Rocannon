SYNOPSIS                   *ipadm_addr*

     • Create/delete static/dynamic IP addresses on network interfaces on Solaris/illumos systems.
     • Up/down static/dynamic IP addresses on network interfaces on Solaris/illumos systems.
     • Manage IPv6 link-local addresses on network interfaces on Solaris/illumos systems.

PARAMETERS                   *ipadm_addr-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   address                       Specifiies an IP address to configure in CIDR notation.
   -                             aliases: addr
   addrobj                       Specifies an unique IP address on the system.
   - / required
                  Choices:
   addrtype       • static ←     Specifiies a type of IP address to configure.
   -              • dhcp
                  • addrconf
                  Choices:
                  • absent
                  • present ←
   state          • up           Create/delete/enable/disable an IP address on the network interface.
   -              • down
                  • enabled
                  • disabled
                  • refreshed
   temporary      Choices:
   boolean        • no ←         Specifies that the configured IP address is temporary. Temporary IP addresses do not persist across reboots.
                  • yes
   wait         Default:         Specifies the time in seconds we wait for obtaining address via DHCP.
   -            60

EXAMPLES                   *ipadm_addr-examples*

 - name: Configure IP address 10.0.0.1 on e1000g0
   ipadm_addr: addr=10.0.0.1/32 addrobj=e1000g0/v4 state=present

 - name: Delete addrobj
   ipadm_addr: addrobj=e1000g0/v4 state=absent

 - name: Configure link-local IPv6 address
   ipadm_addr: addtype=addrconf addrobj=vnic0/v6

 - name: Configure address via DHCP and wait 180 seconds for address obtaining
   ipadm_addr: addrobj=vnic0/dhcp addrtype=dhcp wait=180
