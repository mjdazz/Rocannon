SYNOPSIS                   *pn_vrouter_ospf*

     • This module can be used to add OSPF protocol to vRouter and remove OSPF protocol from a vRouter

PARAMETERS                   *pn_vrouter_ospf-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_netmask                         OSPF network IP address netmask.
   string
   pn_network                         OSPF network IP address.
   string / required
   pn_ospf_area                       stub area number for the configuration.
   string
   pn_vrouter_name                    name of service config.
   string / required
   state               Choices:
   string              • present ←    vrouter-ospf configuration command.
                       • absent

EXAMPLES                   *pn_vrouter_ospf-examples*

 - name: Add OSPF to vRouter
   pn_vrouter_ospf:
     state: 'present'
     pn_vrouter_name: 'sw01-vrouter'
     pn_network: '105.104.104.1'
     pn_netmask: '24'
     pn_ospf_area: '0'
 - name: "Remove OSPF to vRouter"
   pn_vrouter_ospf:
     state: 'absent'
     pn_vrouter_name: 'sw01-vrouter'
     pn_network: '105.104.104.1'

RETURN VALUES                   *pn_vrouter_ospf-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the vrouter-ospf command.
   list
   stdout  always   set of responses from the vrouter-ospf command.
   list
