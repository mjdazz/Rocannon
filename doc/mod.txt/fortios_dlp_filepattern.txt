SYNOPSIS                   *fortios_dlp_filepattern*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify dlp feature and filepattern category. Examples include all parameters and values
       need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_dlp_filepattern-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_dlp_filepattern-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure file patterns used by DLP blocking.
     fortios_dlp_filepattern:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       dlp_filepattern:
         comment: "Optional comments."
         entries:
          -
             file_type: "7z"
             filter_type: "pattern"
             pattern: "<your_own_value>"
         id:  "8"
         name: "default_name_9"
