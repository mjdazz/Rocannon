SYNOPSIS                   *cp_mgmt_discard*

     • All changes done by user are discarded and removed from database.
     • All operations are performed over Web Services API.

PARAMETERS                   *cp_mgmt_discard-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   uid                            Session unique identifier. Specify it to discard a different session than the one you currently use.
   string
   version                        Version of checkpoint. If not given one, the latest version taken.
   string
   wait_for_task   Choices:
   boolean         • no           Wait for the task to end. Such as publish task.
                   • yes ←
