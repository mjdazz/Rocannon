SYNOPSIS                   *pn_vrouter_packet_relay*

     • This module can be used to add packet relay configuration for DHCP on vrouter and remove packet relay configuration for DHCP on vrouter.

PARAMETERS                   *pn_vrouter_packet_relay-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_forward_ip                      forwarding IP address.
   string / required
   pn_forward_proto    Choices:       protocol type to forward packets.
   string              • dhcp ←
   pn_nic                             NIC.
   string / required
   pn_vrouter_name                    name of service config.
   string / required
   state               Choices:
   string              • present ←    vrouter-packet-relay configuration command.
                       • absent

EXAMPLES                   *pn_vrouter_packet_relay-examples*

 - name: vRouter packet relay add
   pn_vrouter_packet_relay:
     pn_cliswitch: "sw01"
     pn_forward_ip: "192.168.10.1"
     pn_nic: "eth0.4092"
     pn_vrouter_name: "sw01-vrouter"

 - name: vRouter packet relay remove
   pn_vrouter_packet_relay:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_forward_ip: "192.168.10.1"
     pn_nic: "eth0.4092"
     pn_vrouter_name: "sw01-vrouter"

RETURN VALUES                   *pn_vrouter_packet_relay-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the vrouter-packet-relay command.
   list
   stdout  always   set of responses from the vrouter-packet-relay command.
   list
