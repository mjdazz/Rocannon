SYNOPSIS                   *onyx_ptp_interface*

     • This module provides declarative management of PTP interfaces configuration on Mellanox ONYX network devices.

PARAMETERS                   *onyx_ptp_interface-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   announce_interval                  configure PTP announce setting for interval, Range -3-1
   -
   announce_timeout                   configure PTP announce setting for timeout, Range 2-10
   -
   delay_request                      configure PTP delay request interval, Range 0-5
   -
   name                               ethernet or vlan interface name that we want to configure PTP on it
   - / required
   state               Choices:
   -                   • enabled ←    Enable/Disable PTP on Interface
                       • disabled
   sync_interval                      configure PTP sync interval, Range -7--1
   -

NOTES                   *onyx_ptp_interface-notes*

   Note

     • Tested on ONYX 3.6.8130
     • PTP Protocol must be enabled on switch.
     • Interface must not be a switch port interface.

EXAMPLES                   *onyx_ptp_interface-examples*

 - name: configure PTP interface
   onyx_ptp_interface:
     state: enabled
     name: Eth1/1
     delay_request: 0
     announce_interval: -2
     announce_timeout: 3

RETURN VALUES                   *onyx_ptp_interface-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key    Returned                                                                                   Description
   commands          The list of configuration mode commands to send to the device.
   list     always   Sample:
                     ['interface ethernet 1/16 ptp enable', 'interface ethernet 1/16 ptp delay-req interval 0', 'interface ethernet 1/16 ptp announce interval -1']
