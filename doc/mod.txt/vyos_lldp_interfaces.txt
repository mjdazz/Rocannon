SYNOPSIS                   *vyos_lldp_interfaces*

     • This module manages attributes of lldp interfaces on VyOS network devices.

PARAMETERS                   *vyos_lldp_interfaces-parameters*

           Parameter         Choices/Defaults                                                                        Comments
   config                                     A list of lldp interfaces configurations.
   list
    enable                     Choices:
    boolean                    • no           to disable lldp on the interface.
                               • yes ←
    location                                  LLDP-MED location data.
    dictionary
     civic_based                              Civic-based location data.
     dictionary
       ca_info                                LLDP-MED address info
       list
          ca_type                             LLDP-MED Civic Address type.
          integer / required
          ca_value                            LLDP-MED Civic Address value.
          string / required
       country_code                           Country Code
       string / required
     coordinate_based                         Coordinate-based location.
     dictionary
       altitude                               Altitude in meters.
       integer
                               Choices:
       datum                   • WGS84        Coordinate datum type.
       string                  • NAD83
                               • MLLW
       latitude                               Latitude.
       string / required
       longitude                              Longitude.
       string / required
     elin                                     Emergency Call Service ELIN number (between 10-25 numbers).
     string
    name                                      Name of the lldp interface.
    string / required
                               Choices:
   state                       • merged ←
   string                      • replaced     The state of the configuration after module completion.
                               • overridden
                               • deleted

NOTES                   *vyos_lldp_interfaces-notes*

   Note

     • Tested against VyOS 1.1.8 (helium).
     • This module works with connection network_cli. See the VyOS OS Platform Options.

RETURN VALUES                   *vyos_lldp_interfaces-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key      Returned                                                                                   Description
   after                 The configuration as structured data after module completion.
   list     when changed Sample:
                         The configuration returned will always be in the same format of the parameters above.
   before                The configuration as structured data prior to module invocation.
   list     always       Sample:
                         The configuration returned will always be in the same format of the parameters above.
   commands              The set of commands pushed to the remote device.
   list     always       Sample:
                         ["set service lldp interface eth2 'disable'", 'delete service lldp interface eth1 location']
