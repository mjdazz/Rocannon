SYNOPSIS                   *hcloud_datacenter_info*

     • Gather info about your Hetzner Cloud datacenters.
     • This module was called hcloud_datacenter_facts before Ansible 2.9, returning ansible_facts and hcloud_datacenter_facts. Note that the hcloud_datacenter_info module no longer returns
       ansible_facts and the value was renamed to hcloud_datacenter_info!

   Aliases: hcloud_datacenter_facts

REQUIREMENTS                   *hcloud_datacenter_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_datacenter_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the datacenter you want to get.
   integer
   name                                             The name of the datacenter you want to get.
   string

SEE ALSO                   *hcloud_datacenter_info-see also*

   See also

   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_datacenter_info-examples*

 - name: Gather hcloud datacenter info
   hcloud_datacenter_info:
   register: output
 - name: Print the gathered info
   debug:
     var: output
