SYNOPSIS                   *pn_cpu_class*

     • This module can be used to create, modify and delete CPU class information.

PARAMETERS                   *pn_cpu_class-parameters*

       Parameter       Choices/Defaults                                                                             Comments
   pn_cliswitch                            Target switch to run the CLI on.
   string
                       Choices:
   pn_hog_protect      • disable           enable host-based hog protection.
   string              • enable
                       • enable-and-drop
   pn_name                                 name for the CPU class.
   string
   pn_rate_limit                           rate-limit for CPU class.
   string
   pn_scope            Choices:
   -                   • local             scope for CPU class.
                       • fabric
                       Choices:
   state               • present           State the action to perform. Use present to create cpu-class and absent to delete cpu-class update to modify the cpu-class.
   string / required   • absent
                       • update

EXAMPLES                   *pn_cpu_class-examples*

 - name: create cpu class
   pn_cpu_class:
     pn_cliswitch: 'sw01'
     state: 'present'
     pn_name: 'icmp'
     pn_rate_limit: '1000'
     pn_scope: 'local'

 - name: delete cpu class
   pn_cpu_class:
     pn_cliswitch: 'sw01'
     state: 'absent'
     pn_name: 'icmp'


 - name: modify cpu class
   pn_cpu_class:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_name: 'icmp'
     pn_rate_limit: '2000'

RETURN VALUES                   *pn_cpu_class-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the cpu-class command.
   list
   stdout  always   set of responses from the cpu-class command.
   list
