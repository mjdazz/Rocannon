SYNOPSIS                   *hcloud_location_info*

     • Gather infos about your Hetzner Cloud locations.
     • This module was called hcloud_location_facts before Ansible 2.9, returning ansible_facts and hcloud_location_facts. Note that the hcloud_location_info module no longer returns ansible_facts
       and the value was renamed to hcloud_location_info!

   Aliases: hcloud_location_facts

REQUIREMENTS                   *hcloud_location_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_location_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the location you want to get.
   integer
   name                                             The name of the location you want to get.
   string

SEE ALSO                   *hcloud_location_info-see also*

   See also

   Documentation for Hetzner Cloud API
   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_location_info-examples*

 - name: Gather hcloud location infos
   hcloud_location_info:
   register: output

 - name: Print the gathered infos
   debug:
     var: output

RETURN VALUES                   *hcloud_location_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

           Key          Returned                                                                             Description
   hcloud_location_info always   The location infos as list
   complex
         city                    City of the location
         string         always   Sample:
                                 Falkenstein
         country                 Country code of the location
         string         always   Sample:
                                 DE
         description             Detail description of the location
         string         always   Sample:
                                 Falkenstein DC Park 1
         id                      Numeric identifier of the location
         integer        always   Sample:
                                 1937415
         name                    Name of the location
         string         always   Sample:
                                 fsn1
