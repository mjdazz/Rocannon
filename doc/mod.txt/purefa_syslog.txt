SYNOPSIS                   *purefa_syslog*

     • Configure syslog configuration for Pure Storage FlashArrays.
     • Add or delete an individual syslog server to the existing list of serves.

REQUIREMENTS                   *purefa_syslog-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_syslog-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   address                            Syslog server address. This field supports IPv4, IPv6 or FQDN. An invalid IP addresses will cause the module to fail. No validation is performed for FQDNs.
   string / required
   api_token                          FlashArray API token for admin privileged user.
   string / required
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   port                               Port at which the server is listening. If no port is specified the system will use 514
   string
                       Choices:
   protocol            • tcp          Protocol which server uses
   string / required   • tls
                       • udp
   state               Choices:
   string              • absent       Create or delete syslog servers configuration
                       • present ←

NOTES                   *purefa_syslog-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_syslog-examples*

 - name: Delete existing syslog server entries
   purefa_syslog:
     address: syslog1.com
     protocol: tcp
     state: absent
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592

 - name: Set array syslog servers
   purefa_syslog:
     state: present
     address: syslog1.com
     protocol: udp
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
