SYNOPSIS                   *cp_mgmt_service_other*

     • Manages service-other objects on Check Point devices including creating, updating and removing objects.
     • All operations are performed over Web Services API.

EXAMPLES                   *cp_mgmt_service_other-examples*

 - name: add-service-other
   cp_mgmt_service_other:
     aggressive_aging:
       enable: true
       timeout: 360
       use_default_timeout: false
     ip_protocol: 51
     keep_connections_open_after_policy_installation: false
     match_for_any: true
     name: New_Service_1
     session_timeout: 0
     state: present
     sync_connections_on_cluster: true

 - name: set-service-other
   cp_mgmt_service_other:
     aggressive_aging:
       default_timeout: 3600
     color: green
     name: New_Service_1
     state: present

 - name: delete-service-other
   cp_mgmt_service_other:
     name: New_Service_2
     state: absent
