SYNOPSIS                   *ucs_vhba_template*

     • Configures vHBA templates on Cisco UCS Manager.
     • Examples can be used with the UCS Platform Emulator https://communities.cisco.com/ucspe.

REQUIREMENTS                   *ucs_vhba_template-requirements*

   The below requirements are needed on the host that executes this module.

     • ucsmsdk

EXAMPLES                   *ucs_vhba_template-examples*

 - name: Configure vHBA template
   ucs_vhba_template:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: vHBA-A
     fabric: A
     vsan: VSAN-A
     wwpn_pool: WWPN-Pool-A

 - name: Remote vHBA template
   ucs_vhba_template:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: vHBA-A
     state: absent
