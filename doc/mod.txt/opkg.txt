SYNOPSIS                   *opkg*

     • Manages OpenWrt packages

REQUIREMENTS                   *opkg-requirements*

   The below requirements are needed on the host that executes this module.

     • opkg
     • python

PARAMETERS                   *opkg-parameters*

    Parameter            Choices/Defaults                                                                               Comments
                  Choices:
                  • 

                  • depends

EXAMPLES                   *opkg-examples*

 - opkg:
     name: foo
     state: present

 - opkg:
     name: foo
     state: present
     update_cache: yes

 - opkg:
     name: foo
     state: absent

 - opkg:
     name: foo,bar
     state: absent

 - opkg:
     name: foo
     state: present
     force: overwrite
