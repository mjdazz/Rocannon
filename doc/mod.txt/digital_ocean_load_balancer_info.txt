SYNOPSIS                   *digital_ocean_load_balancer_info*

     • This module can be used to gather information about DigitalOcean provided load balancers.
     • This module was called digital_ocean_load_balancer_facts before Ansible 2.9. The usage did not change.

   Aliases: digital_ocean_load_balancer_facts

REQUIREMENTS                   *digital_ocean_load_balancer_info-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6

PARAMETERS                   *digital_ocean_load_balancer_info-parameters*

      Parameter     Choices/Defaults                                                                             Comments
   load_balancer_id                  Load balancer ID that can be used to identify and reference a load_balancer.
   -
                                     DigitalOcean OAuth token.
   oauth_token                       There are several other environment variables which can be used to provide this value.
   string                            i.e., - 'DO_API_TOKEN', 'DO_API_KEY', 'DO_OAUTH_TOKEN' and 'OAUTH_TOKEN'
                                     aliases: api_token
   timeout          Default:         The timeout in seconds used for polling DigitalOcean's API.
   integer          30
   validate_certs     Choices:       If set to no, the SSL certificates will not be validated.
   boolean            • no           This should only set to no used on personally controlled sites using self-signed certificates.
                      • yes ←

EXAMPLES                   *digital_ocean_load_balancer_info-examples*

 - name: Gather information about all load balancers
   digital_ocean_load_balancer_info:
     oauth_token: "{{ oauth_token }}"

 - name: Gather information about load balancer with given id
   digital_ocean_load_balancer_info:
     oauth_token: "{{ oauth_token }}"
     load_balancer_id: "4de7ac8b-495b-4884-9a69-1050c6793cd6"

 - name: Get name from load balancer id
   digital_ocean_load_balancer_info:
   register: resp_out
 - set_fact:
     load_balancer_name: "{{ item.name }}"
   loop: "{{ resp_out.data|json_query(name) }}"
   vars:
     name: "[?id=='4de7ac8b-495b-4884-9a69-1050c6793cd6']"
 - debug: var=load_balancer_name
