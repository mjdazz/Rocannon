SYNOPSIS                   *ios_lacp_interfaces*

     • This module provides declarative management of LACP on Cisco IOS network devices lacp_interfaces.

PARAMETERS                   *ios_lacp_interfaces-parameters*

           Parameter          Choices/Defaults                                                                        Comments
   config                                      A dictionary of LACP lacp_interfaces option
   list / elements=dictionary
        fast_switchover         Choices:
        boolean                 • no           LACP fast switchover supported on this port channel.
                                • yes
        max_bundle                             LACP maximum number of ports to bundle in this port channel.
        integer                                Refer to vendor documentation for valid port values.
        name                                   Name of the Interface for configuring LACP.
        string / required
        port_priority                          LACP priority on this interface.
        integer                                Refer to vendor documentation for valid port values.
                                Choices:
   state                        • merged ←
   string                       • replaced     The state of the configuration after module completion
                                • overridden
                                • deleted

NOTES                   *ios_lacp_interfaces-notes*

   Note

     • Tested against Cisco IOSv Version 15.2 on VIRL.
     • This module works with connection network_cli, See IOS Platform Options.

EXAMPLES                   *ios_lacp_interfaces-examples*

 # Using merged
 #
 # Before state:
 # -------------
 #
 # vios#show running-config | section ^interface
 # interface Port-channel10
 # interface Port-channel20
 # interface Port-channel30
 # interface GigabitEthernet0/1
 #  shutdown
 # interface GigabitEthernet0/2
 #  shutdown
 # interface GigabitEthernet0/3
 #  shutdown

 - name: Merge provided configuration with device configuration
   ios_lacp_interfaces:
     config:
       - name: GigabitEthernet0/1
         port_priority: 10
       - name: GigabitEthernet0/2
         port_priority: 20
       - name: GigabitEthernet0/3
         port_priority: 30
       - name: Port-channel10
         fast_switchover: True
         max_bundle: 5
     state: merged

 # After state:
 # ------------
 #
 # vios#show running-config | section ^interface

 #  lacp max-bundle 5
 # interface Port-channel20
 # interface Port-channel30
 # interface GigabitEthernet0/1
 #  shutdown
 #  lacp port-priority 10
 # interface GigabitEthernet0/2
 #  shutdown

 #  lacp port-priority 30
