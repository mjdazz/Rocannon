SYNOPSIS                   *win_rabbitmq_plugin*

     • Manage RabbitMQ plugins.

PARAMETERS                   *win_rabbitmq_plugin-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   names                              Comma-separated list of plugin names.
   string / required                  aliases: name
   new_only            Choices:       Only enable missing plugins.
   boolean             • no ←         Does not disable plugins that are not in the names list.
                       • yes
   prefix                             Specify a custom install prefix to a Rabbit.
   string
   state               Choices:
   string              • disabled     Specify if plugins are to be enabled or disabled.
                       • enabled ←

EXAMPLES                   *win_rabbitmq_plugin-examples*

 - name: Enables the rabbitmq_management plugin
   win_rabbitmq_plugin:
     names: rabbitmq_management
     state: enabled

RETURN VALUES                   *win_rabbitmq_plugin-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key    Returned                                                                                   Description
   disabled          List of plugins disabled during task run.
   list     always   Sample:
                     ['rabbitmq_management']
   enabled           List of plugins enabled during task run.
   list     always   Sample:
                     ['rabbitmq_management']
