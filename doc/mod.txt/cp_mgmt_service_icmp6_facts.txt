SYNOPSIS                   *cp_mgmt_service_icmp6_facts*

     • Get service-icmp6 objects facts on Check Point devices.
     • All operations are performed over Web Services API.
     • This module handles both operations, get a specific object and get several objects, For getting a specific object use the parameter ‘name’.

PARAMETERS                   *cp_mgmt_service_icmp6_facts-parameters*

      Parameter    Choices/Defaults                                                                             Comments
                     Choices:
   details_level     • uid          The level of detail for some of the fields in the response can vary from showing only the UID value of the object to a fully detailed representation of the
   string            • standard     object.
                     • full
   limit                            No more than that many results will be returned. This parameter is relevant only for getting few objects.
   integer
   name                             Object name. This parameter is relevant only for getting a specific object.
   string

   integer

         DESC        Choices:       Sorts results by the given field in descending order.

   show_membership   Choices:
   boolean           • no           Indicates whether to calculate and show "groups" field for every object in reply.

   version                          Version of checkpoint. If not given one, the latest version taken.
   string

EXAMPLES                   *cp_mgmt_service_icmp6_facts-examples*

 - name: show-service-icmp6
