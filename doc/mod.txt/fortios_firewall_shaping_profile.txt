SYNOPSIS                   *fortios_firewall_shaping_profile*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall feature and shaping_profile category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_shaping_profile-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_shaping_profile-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure shaping profiles.
     fortios_firewall_shaping_profile:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_shaping_profile:
         comment: "Comment."
         default_class_id: "4"
         profile_name: "<your_own_value>"
         shaping_entries:
          -
             class_id: "7"
             guaranteed_bandwidth_percentage: "8"
             id:  "9"
             maximum_bandwidth_percentage: "10"
             priority: "high"
