SYNOPSIS                   *vmware_folder_info*

     • The module can be used to gather a hierarchical view of the folders that exist within a datacenter

REQUIREMENTS                   *vmware_folder_info-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_folder_info-notes*

   Note

     • Tested on vSphere 6.5

EXAMPLES                   *vmware_folder_info-examples*

 - name: Provide information about vCenter folders
   vmware_folder_info:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
     datacenter: datacenter_name
   delegate_to: localhost
   register: vcenter_folder_info
