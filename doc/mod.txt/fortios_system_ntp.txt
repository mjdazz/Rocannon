SYNOPSIS                   *fortios_system_ntp*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and ntp category. Examples include all parameters and values need
       to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_ntp-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

PARAMETERS                   *fortios_system_ntp-parameters*

      Parameter     Choices/Defaults                                                                             Comments
   host                              FortiOS or FortiGate IP address.
   string
   https              Choices:
   boolean            • no           Indicates if the requests towards FortiGate must use HTTPS protocol.
                      • yes ←
   password         Default:         FortiOS or FortiGate password.
   string           ""
   ssl_verify         Choices:
   boolean            • no           Ensures FortiGate certificate must be verified by a proper CA.
                      • yes ←
   system_ntp       Default:         Configure system NTP information.
   dictionary       null
    interface                        FortiGate interface(s) with NTP server mode enabled. Devices on your network can contact these interfaces for NTP services.
    list
     interface_name                  Interface name. Source system.interface.name.
     string
    ntpserver                        Configure the FortiGate to connect to any available third-party NTP server.
    list
     authentication   Choices:
     string           • enable       Enable/disable MD5 authentication.
                      • disable
     id
     integer /                       NTP server ID.
     required
     key                             Key for MD5 authentication.
     string
     key_id                          Key ID for authentication.
     integer
     ntpv3            Choices:
     string           • enable       Enable to use NTPv3 instead of NTPv4.
                      • disable
     server                          IP address or hostname of the NTP Server.
     string
    ntpsync           Choices:
    string            • enable       Enable/disable setting the FortiGate system time by synchronizing with an NTP Server.
                      • disable
    server_mode       Choices:       Enable/disable FortiGate NTP Server Mode. Your FortiGate becomes an NTP server for other devices on your network. The FortiGate relays NTP requests to its
    string            • enable       configured NTP server.
                      • disable
    source_ip                        Source IP address for communication to the NTP server.
    string
    source_ip6                       Source IPv6 address for communication to the NTP server.
    string
    syncinterval                     NTP synchronization interval (1 - 1440 min).
    integer
    type              Choices:
    string            • fortiguard   Use the FortiGuard NTP server or any other available NTP Server.
                      • custom
   username                          FortiOS or FortiGate username.
   string
   vdom             Default:         Virtual domain, among those defined previously. A vdom is a virtual instance of the FortiGate that can be configured and used as a different unit.
   string           "root"

NOTES                   *fortios_system_ntp-notes*

   Note

     • Requires fortiosapi library developed by Fortinet

EXAMPLES                   *fortios_system_ntp-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure system NTP information.
     fortios_system_ntp:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       system_ntp:
         interface:
          -
             interface_name: "<your_own_value> (source system.interface.name)"
         ntpserver:
          -
             authentication: "enable"
             id:  "7"
             key: "<your_own_value>"
             key_id: "9"
             ntpv3: "enable"
             server: "192.168.100.40"
         ntpsync: "enable"
         server_mode: "enable"
         source_ip: "84.230.14.43"
         source_ip6: "<your_own_value>"
         syncinterval: "16"
         type: "fortiguard"
