SYNOPSIS                   *intersight_info*

     • Gathers information about servers in Cisco Intersight.
     • This module was called intersight_facts before Ansible 2.9. The usage did not change.

   Aliases: intersight_facts

PARAMETERS                   *intersight_info-parameters*

       Parameter            Choices/Defaults                                                                             Comments
   api_key_id                                        Public API Key ID associated with the private key.
   string / required
   api_private_key                                   Filename (absolute path) of a PEM formatted file that contains your private key to be used for Intersight API authentication.
   path / required
   api_uri           Default:                        URI used to access the Intersight API.
   string            "https://intersight.com/api/v1"
   server_names                                      Server names to retrieve information from.
   list / required                                   An empty list will return all servers.
   use_proxy           Choices:
   boolean             • no                          If no, it will not use a proxy, even if one is defined in an environment variable on the target hosts.
                       • yes ←
   validate_certs      Choices:
   boolean             • no                          Boolean control for verifying the api_uri TLS certificate
                       • yes ←

EXAMPLES                   *intersight_info-examples*

 - name: Get info for all servers
   intersight_info:
     api_private_key: ~/Downloads/SecretKey.txt
     api_key_id: 64612d300d0982/64612d300d0b00/64612d300d3650
     server_names:
 - debug:
     msg: "server name {{ item.Name }}, moid {{ item.Moid }}"
   loop: "{{ intersight_servers }}"
   when: intersight_servers is defined

 - name: Get info for servers by name
   intersight_info:
     api_private_key: ~/Downloads/SecretKey.txt
     api_key_id: 64612d300d0982/64612d300d0b00/64612d300d3650
     server_names:
       - SJC18-L14-UCS1-1
 - debug:
     msg: "server moid {{ intersight_servers[0].Moid }}"
   when: intersight_servers[0] is defined

RETURN VALUES                   *intersight_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

          Key         Returned                                                                              Description
   intersight_servers always   A list of Intersight Servers. See Cisco Intersight for details.
   complex
          Moid                 The unique identifier of this Managed Object instance.
          string      always   Sample:
                               5978bea36ad4b000018d63dc
          Name                 The name of the server.
          string      always   Sample:
                               SJC18-L14-UCS1-1
