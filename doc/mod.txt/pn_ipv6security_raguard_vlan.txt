SYNOPSIS                   *pn_ipv6security_raguard_vlan*

     • This module can be used to Add vlans to RA Guard Policy and Remove vlans to RA Guard Policy.

PARAMETERS                   *pn_ipv6security_raguard_vlan-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            RA Guard Policy Name.
   string / required
   pn_vlans                           Vlans attached to RA Guard Policy.
   string / required
   state               Choices:
   string              • present ←    ipv6security-raguard-vlan configuration command.
                       • absent

EXAMPLES                   *pn_ipv6security_raguard_vlan-examples*

 - name: ipv6 security raguard vlan add
   pn_ipv6security_raguard_vlan:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_vlans: "100-105"

 - name: ipv6 security raguard vlan add
   pn_ipv6security_raguard_vlan:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_vlans: "100"

 - name: ipv6 security raguard vlan remove
   pn_ipv6security_raguard_vlan:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_vlans: "100-105"
     state: 'absent'

RETURN VALUES                   *pn_ipv6security_raguard_vlan-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the ipv6security-raguard-vlan command.
   list
   stdout  always   set of responses from the ipv6security-raguard-vlan command.
   list
