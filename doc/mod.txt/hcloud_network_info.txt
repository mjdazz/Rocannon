SYNOPSIS                   *hcloud_network_info*

     • Gather info about your Hetzner Cloud networks.

REQUIREMENTS                   *hcloud_network_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_network_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the network you want to get.
   integer
   label_selector                                   The label selector for the network you want to get.
   string
   name                                             The name of the network you want to get.
   string

SEE ALSO                   *hcloud_network_info-see also*

   See also

   Documentation for Hetzner Cloud API
   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_network_info-examples*

 - name: Gather hcloud network info
   local_action:
     module: hcloud_network_info

 - name: Print the gathered info
   debug:
     var: hcloud_network_info

RETURN VALUES                   *hcloud_network_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

           Key         Returned                                                                              Description
   hcloud_network_info always   The network info as list
   complex
        id                      Numeric identifier of the network
        integer        always   Sample:
                                1937415
        ip_range                IP range of the network
        string         always   Sample:
                                10.0.0.0/16
        labels         always   Labels of the network
        dictionary
        name                    Name of the network
        string         always   Sample:
                                awsome-network
        routes         always   Routes belonging to the network
        complex
        servers        always   Servers attached to the network
        complex
        subnetworks    always   Subnetworks belonging to the network
        complex
