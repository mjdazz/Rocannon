SYNOPSIS                   *ucs_mac_pool*

     • Configures MAC address pools and MAC address blocks on Cisco UCS Manager.
     • Examples can be used with the UCS Platform Emulator https://communities.cisco.com/ucspe.

REQUIREMENTS                   *ucs_mac_pool-requirements*

   The below requirements are needed on the host that executes this module.

     • ucsmsdk

EXAMPLES                   *ucs_mac_pool-examples*

 - name: Configure MAC address pool
   ucs_mac_pool:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: mac-A
     first_addr: 00:25:B5:00:66:00
     last_addr: 00:25:B5:00:67:F3
     order: sequential

 - name: Remove MAC address pool
   ucs_mac_pool:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: mac-A
     state: absent
