SYNOPSIS                   *fortios_system_resource_limits*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and resource_limits category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_resource_limits-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

   boolean                   • no           Indicates if the requests towards FortiGate must use HTTPS protocol.

    integer

    integer

    integer
    firewall_policy                         Maximum number of firewall policies (IPv4, IPv6, policy46, policy64, DoS-policy4, DoS-policy6, multicast).
    integer
    ipsec_phase1                            Maximum number of VPN IPsec phase1 tunnels.
    integer
    ipsec_phase1_interface                  Maximum number of VPN IPsec phase1 interface tunnels.
    integer
    ipsec_phase2                            Maximum number of VPN IPsec phase2 tunnels.
    integer
    ipsec_phase2_interface                  Maximum number of VPN IPsec phase2 interface tunnels.
    integer
    log_disk_quota                          Log disk quota in MB.
    integer
    onetime_schedule                        Maximum number of firewall one-time schedules.
    integer
    proxy                                   Maximum number of concurrent proxy users.
    integer

    integer
    service_group                           Maximum number of firewall service groups.
    integer
    session                                 Maximum number of sessions.
    integer

    integer
    user                                    Maximum number of local users.
    integer
    user_group                              Maximum number of user groups.
    integer
   username                                 FortiOS or FortiGate username.
   string

   string                  "root"

NOTES                   *fortios_system_resource_limits-notes*

   Note

     • Requires fortiosapi library developed by Fortinet
     • Run as a local_action in your playbook

EXAMPLES                   *fortios_system_resource_limits-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure resource limits.
     fortios_system_resource_limits:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       system_resource_limits:
         custom_service: "3"
         dialup_tunnel: "4"
         firewall_address: "5"
         firewall_addrgrp: "6"
         firewall_policy: "7"
         ipsec_phase1: "8"
         ipsec_phase1_interface: "9"
         ipsec_phase2: "10"
         ipsec_phase2_interface: "11"
         log_disk_quota: "12"
         onetime_schedule: "13"
         proxy: "14"
         recurring_schedule: "15"
         service_group: "16"
         session: "17"
         sslvpn: "18"
         user: "19"
         user_group: "20"
