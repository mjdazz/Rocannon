SYNOPSIS                   *junos_lacp*

     • This module provides declarative management of global LACP on Juniper Junos network devices.

REQUIREMENTS                   *junos_lacp-requirements*

   The below requirements are needed on the host that executes this module.

     • ncclient (>=v0.6.4)

PARAMETERS                   *junos_lacp-parameters*

      Parameter     Choices/Defaults                                                                             Comments
   config                             A dictionary of LACP global options
   dictionary
    link_protection   Choices:        Enable LACP link-protection for the system. If the value is set to non-revertive it will not revert links when a better priority link comes up. By default the
    string            • revertive     link will be reverted.
                      • non-revertive
    system_priority                   LACP priority for the system.
    integer
                      Choices:
   state              • merged ←      The state of the configuration after module completion
   string             • replaced
                      • deleted

NOTES                   *junos_lacp-notes*

   Note

     • This module requires the netconf system service be enabled on the remote device being managed.
     • Tested against vSRX JUNOS version 18.1R1.
     • This module works with connection netconf. See the Junos OS Platform Options.

EXAMPLES                   *junos_lacp-examples*

 # Using deleted

 # Before state:
 # -------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 # system-priority 63;
 # link-protection {
 #    non-revertive;
 # }

 - name: Delete global LACP attributes
   junos_lacp:
     state: deleted

 # After state:
 # ------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 #


 # Using merged

 # Before state:
 # -------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 #

 - name: Merge global LACP attributes
   junos_lacp:
     config:
       system_priority: 63
       link_protection: revertive
     state: merged

 # After state:
 # ------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 # system-priority 63;
 # link-protection {
 #    non-revertive;
 # }


 # Using replaced

 # Before state:
 # -------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 # system-priority 63;
 # link-protection {
 #    non-revertive;
 # }

 - name: Replace global LACP attributes
   junos_lacp:
     config:
       system_priority: 30
       link_protection: non-revertive
     state: replaced

 # After state:
 # ------------
 # [email protected]# show chassis aggregated-devices ethernet lacp
 # system-priority 30;
 # link-protection;

                           The configuration returned will always be in the same format of the parameters above.

   dictionary always       Sample:
                           The configuration returned will always be in the same format of the parameters above.
