SYNOPSIS                   *win_pagefile*

     • Query current pagefile configuration.
     • Enable/Disable AutomaticManagedPagefile.
     • Create new or override pagefile configuration.

PARAMETERS                   *win_pagefile-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   automatic        Choices:
   boolean          • no           Configures AutomaticManagedPagefile for the entire system.
                    • yes
   drive                           The drive of the pagefile.
   string
   initial_size                    The initial size of the pagefile in megabytes.
   integer
   maximum_size                    The maximum size of the pagefile in megabytes.
   integer
   override         Choices:
   boolean          • no           Override the current pagefile on the drive.
                    • yes ←
   remove_all       Choices:
   boolean          • no ←         Remove all pagefiles in the system, not including automatic managed.
                    • yes
                    Choices:
   state            • absent       State of the pagefile.
   string           • present
                    • query ←
   system_managed   Choices:
   boolean          • no ←         Configures current pagefile to be managed by the system.
                    • yes
   test_path        Choices:
   boolean          • no           Use Test-Path on the drive to make sure the drive is accessible before creating the pagefile.
                    • yes ←

NOTES                   *win_pagefile-notes*

   Note

     • There is difference between automatic managed pagefiles that configured once for the entire system and system managed pagefile that configured per pagefile.
     • InitialSize 0 and MaximumSize 0 means the pagefile is managed by the system.
     • Value out of range exception may be caused by several different issues, two common problems - No such drive, Pagefile size is too small.
     • Setting a pagefile when AutomaticManagedPagefile is on will disable the AutomaticManagedPagefile.

EXAMPLES                   *win_pagefile-examples*

 - name: Query pagefiles configuration
   win_pagefile:

 - name: Query C pagefile
   win_pagefile:
     drive: C

 - name: Set C pagefile, don't override if exists
   win_pagefile:
     drive: C
     initial_size: 1024
     maximum_size: 1024
     override: no
     state: present

 - name: Set C pagefile, override if exists
   win_pagefile:
     drive: C
     initial_size: 1024
     maximum_size: 1024
     state: present

 - name: Remove C pagefile
   win_pagefile:
     drive: C
     state: absent

 - name: Remove all current pagefiles, enable AutomaticManagedPagefile and query at the end
   win_pagefile:
     remove_all: yes
     automatic: yes

 - name: Remove all pagefiles disable AutomaticManagedPagefile and set C pagefile
   win_pagefile:
     drive: C
     initial_size: 2048
     maximum_size: 2048
     remove_all: yes
     automatic: no
     state: present

 - name: Set D pagefile, override if exists
   win_pagefile:
     drive: d
     initial_size: 1024
     maximum_size: 1024
     state: present
