SYNOPSIS                   *pn_connection_stats_settings*

     • This module can be used to modify the settings for collecting statistical data about connections.

PARAMETERS                   *pn_connection_stats_settings-parameters*

                 Parameter               Choices/Defaults                                                                  Comments
   pn_client_server_stats_log_disk_space                  disk-space allocated for statistics (including rotated log files).
   string
   pn_client_server_stats_log_enable       Choices:
   boolean                                 • no           Enable or disable statistics.
                                           • yes
   pn_client_server_stats_log_interval                    interval to collect statistics.
   string
   pn_client_server_stats_max_memory                      maximum memory for client server statistics.
   string
   pn_cliswitch                                           Target switch to run the CLI on.
   string
   pn_connection_backup_enable             Choices:
   boolean                                 • no           Enable backup for connection statistics collection.
                                           • yes
   pn_connection_backup_interval                          backup interval for connection statistics collection.
   string
   pn_connection_max_memory                               maximum memory allowed for connection statistics.
   string
   pn_connection_stats_log_disk_space                     disk-space allocated for statistics (including rotated log files).
   string
   pn_connection_stats_log_enable          Choices:
   boolean                                 • no           enable or disable statistics.
                                           • yes
   pn_connection_stats_log_interval                       interval to collect statistics.
   string
   pn_connection_stats_max_memory                         maximum memory allowed for connection statistics.
   string
   pn_enable                               Choices:
   boolean                                 • no           Enable or disable collecting connections statistics.
                                           • yes
   pn_fabric_connection_backup_enable      Choices:
   boolean                                 • no           enable backup for fabric connection statistics collection.
                                           • yes
   pn_fabric_connection_backup_interval                   backup interval for fabric connection statistics collection.
   string
   pn_fabric_connection_max_memory                        maximum memory allowed for fabric connection statistics.
   string
   pn_service_stat_max_memory                             maximum memory allowed for service statistics.
   string
   state                                   Choices:       State the action to perform. Use update to modify the connection-stats-settings.
   string / required                       • update

EXAMPLES                   *pn_connection_stats_settings-examples*

 - name: "Modify connection stats settings"
   pn_connection_stats_settings:
     pn_cliswitch: "sw01"
     state: "update"
     pn_enable: False
     pn_fabric_connection_max_memory: "1000"

 - name: "Modify connection stats settings"
   pn_connection_stats_settings:
     pn_cliswitch: "sw01"
     state: "update"
     pn_enable: True

RETURN VALUES                   *pn_connection_stats_settings-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the connection-stats-settings command.
   list
   stdout  always   set of responses from the connection-stats-settings command.
   list
