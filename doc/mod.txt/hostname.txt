SYNOPSIS                   *hostname*

     • Set system’s hostname, supports most OSs/Distributions, including those using systemd.
     • Note, this module does NOT modify /etc/hosts. You need to modify it yourself using other modules like template or replace.
     • Windows, HP-UX and AIX are not currently supported.

REQUIREMENTS                   *hostname-requirements*

   The below requirements are needed on the host that executes this module.

     • hostname

    Parameter   Choices/Defaults                                                                               Comments

   - / required

   use            • redhat       Which strategy to use to update the hostname.
   -              • alpine       If not set we try to autodetect, but this can be problematic, specially with containers as they can present misleading information.
