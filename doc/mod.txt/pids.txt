SYNOPSIS                   *pids*

     • Retrieves a list of PIDs of given process name in Ansible controller/controlled machines.Returns an empty list if no process in that name exists.

REQUIREMENTS                   *pids-requirements*

   The below requirements are needed on the host that executes this module.

     • psutil(python module)

PARAMETERS                   *pids-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   name                               the name of the process you want to get PID for.
   string / required

EXAMPLES                   *pids-examples*

 # Pass the process name
 - name: Getting process IDs of the process
   pids:
       name: python
   register: pids_of_python

 - name: Printing the process IDs obtained
   debug:
     msg: "PIDS of python:{{pids_of_python.pids|join(',')}}"

RETURN VALUES                   *pids-return values*

   Common return values are documented here, the following are the fields unique to this module:

   Key                 Returned                                                                                     Description
   pids                                        Process IDs of the given process
   list list of none, one, or more process IDs Sample:
                                               [100, 200]
