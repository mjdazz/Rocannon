SYNOPSIS                   *pn_port_cos_bw*

     • This module can be used to update bw settings for CoS queues.

PARAMETERS                   *pn_port_cos_bw-parameters*

        Parameter      Choices/Defaults                                                                            Comments
   pn_cliswitch                          Target switch to run the CLI on.
   string
   pn_cos                                CoS priority.
   string
   pn_max_bw_limit                       Maximum b/w in percentage.
   string
   pn_min_bw_guarantee                   Minimum b/w in percentage.
   string
   pn_port                               physical port number.
   string
   pn_weight             Choices:
   string                • priority      Scheduling weight (1 to 127) after b/w guarantee met.
                         • no-priority
   state                 Choices:        State the action to perform. Use update to modify the port-cos-bw.
   string / required     • update

EXAMPLES                   *pn_port_cos_bw-examples*

 - name: port cos bw modify
   pn_port_cos_bw:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "1"
     pn_cos: "0"
     pn_min_bw_guarantee: "60"

 - name: port cos bw modify
   pn_port_cos_bw:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "all"
     pn_cos: "0"
     pn_weight: "priority"

RETURN VALUES                   *pn_port_cos_bw-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the port-cos-bw command.
   list
   stdout  always   set of responses from the port-cos-bw command.
   list
