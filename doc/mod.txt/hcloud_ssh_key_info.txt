SYNOPSIS                   *hcloud_ssh_key_info*

     • Gather facts about your Hetzner Cloud ssh_keys.
     • This module was called hcloud_ssh_key_facts before Ansible 2.9, returning ansible_facts and hcloud_ssh_key_facts. Note that the hcloud_ssh_key_info module no longer returns ansible_facts and
       the value was renamed to hcloud_ssh_key_info!

   Aliases: hcloud_ssh_key_facts

REQUIREMENTS                   *hcloud_ssh_key_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_ssh_key_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   fingerprint                                      The fingerprint of the ssh key you want to get.
   string
   id                                               The ID of the ssh key you want to get.
   integer
   label_selector                                   The label selector for the ssh key you want to get.
   string
   name                                             The name of the ssh key you want to get.
   string

SEE ALSO                   *hcloud_ssh_key_info-see also*

   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_ssh_key_info-examples*

 - name: Gather hcloud sshkey infos
   hcloud_ssh_key_info:
   register: output
 - name: Print the gathered infos
   debug:
     var: output.hcloud_ssh_key_info
