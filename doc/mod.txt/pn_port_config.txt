SYNOPSIS                   *pn_port_config*

     • This module can be used to modify a port configuration.

EXAMPLES                   *pn_port_config-examples*

 - name: port config modify
   pn_port_config:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "all"
     pn_dscp_map: "foo"

 - name: port config modify
   pn_port_config:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "all"
     pn_host_enable: true

RETURN VALUES                   *pn_port_config-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the port-config command.
   list
   stdout  always   set of responses from the port-config command.
   list
