SYNOPSIS                   *win_route*

     • Add or remove a static route.

PARAMETERS                   *win_route-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   destination                        Destination IP address in CIDR format (ip address/prefix length).
   string / required
   gateway                            The gateway used by the static route.
   string                             If gateway is not provided it will be set to 0.0.0.0.
   metric            Default:         Metric used by the static route.
   integer           1
   state               Choices:       If absent, it removes a network static route.
   string              • absent       If present, it adds a network static route.
                       • present ←

NOTES                   *win_route-notes*

   Note

     • Works only with Windows 2012 R2 and newer.

EXAMPLES                   *win_route-examples*

 ---
 - name: Add a network static route
   win_route:
     destination: 192.168.2.10/32
     gateway: 192.168.1.1
     metric: 1
     state: present

 - name: Remove a network static route
   win_route:
     destination: 192.168.2.10/32
     state: absent

RETURN VALUES                   *win_route-return values*

   Common return values are documented here, the following are the fields unique to this module:

    Key   Returned                                                                                    Description
   output          A message describing the task result.
   string always   Sample:
                   Route added
