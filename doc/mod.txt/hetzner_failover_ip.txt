SYNOPSIS                   *hetzner_failover_ip*

     • Manage Hetzner’s failover IPs.

PARAMETERS                   *hetzner_failover_ip-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   failover_ip                        The failover IP address.
   string / required
   hetzner_password                   The password for the Robot webservice user.
   string / required
   hetzner_user                       The username for the Robot webservice user.
   string / required
   state               Choices:       Defines whether the IP will be routed or not.
   string              • routed ←     If set to routed, value must be specified.
                       • unrouted
   timeout           Default:         Timeout to use when routing or unrouting the failover IP.
   integer           180              Note that the API call returns when the failover IP has been successfully routed to the new address, respectively successfully unrouted.
   value                              The new value for the failover IP address.
   string                             Required when setting state to routed.

   Retrieve information on failover IPs.

EXAMPLES                   *hetzner_failover_ip-examples*

 - name: Set value of failover IP 1.2.3.4 to 5.6.7.8
   hetzner_failover_ip:
     hetzner_user: foo
     hetzner_password: bar
     failover_ip: 1.2.3.4
     value: 5.6.7.8

 - name: Set value of failover IP 1.2.3.4 to unrouted
   hetzner_failover_ip:
     hetzner_user: foo
     hetzner_password: bar
     failover_ip: 1.2.3.4
     state: unrouted

RETURN VALUES                   *hetzner_failover_ip-return values*

   Common return values are documented here, the following are the fields unique to this module:

    Key   Returned                                                                                    Description
   state  success  Will be routed or unrouted.
   string
   value  success  The value of the failover IP.
   string          Will be none if the IP is unrouted.
