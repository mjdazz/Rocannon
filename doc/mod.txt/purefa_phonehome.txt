SYNOPSIS                   *purefa_phonehome*

     • Enable or Disable Phonehome for a Pure Storage FlashArray.

REQUIREMENTS                   *purefa_phonehome-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_phonehome-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashArray API token for admin privileged user.
   string / required
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   state               Choices:
   string              • present ←    Define state of phonehome
                       • absent

NOTES                   *purefa_phonehome-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_phonehome-examples*

 - name: Enable Phonehome
   purefa_phonehome:
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592

 - name: Disable Phonehome
   purefa_phonehome:
     state: disable
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
