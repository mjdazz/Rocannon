SYNOPSIS                   *nictagadm*

     • Create or delete nic tags on SmartOS systems.

PARAMETERS                   *nictagadm-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   etherstub           Choices:       Specifies that the nic tag will be attached to a created etherstub.
   boolean             • no ←         Parameter etherstub is mutually exclusive with both mtu, and mac.
                       • yes
   force               Choices:
   boolean             • no ←         When state is absent set this switch will use the -f parameter and delete the nic tag regardless of existing VMs.
                       • yes
   mac                                Specifies the mac address to attach the nic tag to when not creating an etherstub.
   string                             Parameters mac and etherstub are mutually exclusive.
   mtu                                Specifies the size of the mtu of the desired nic tag.
   integer                            Parameters mtu and etherstub are mutually exclusive.
   name                               Name of the nic tag.
   string / required
   state               Choices:
   string              • absent       Create or delete a SmartOS nic tag.
                       • present ←

EXAMPLES                   *nictagadm-examples*

 - name: Create 'storage0' on '00:1b:21:a3:f5:4d'
   nictagadm:
     name: storage0
     mac: 00:1b:21:a3:f5:4d
     mtu: 9000
     state: present

 - name: Remove 'storage0' nic tag
   nictagadm:
     name: storage0
     state: absent

RETURN VALUES                   *nictagadm-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key    Returned                                                                                   Description
   etherstub always   specifies if the nic tag will create and attach to an etherstub.
   boolean
   force     always   Shows if -f was used during the deletion of a nic tag
   boolean
   mac                MAC Address that the nic tag was attached to.
   string    always   Sample:
                      00:1b:21:a3:f5:4d
   mtu                specifies which MTU size was passed during the nictagadm add command. mtu and etherstub are mutually exclusive.
   integer   always   Sample:
                      1500
   name               nic tag name
   string    always   Sample:
                      storage0
   state              state of the target
   string    always   Sample:
                      present
