SYNOPSIS                   *win_iis_webbinding*

     • Creates, removes and configures a binding to an existing IIS Web site.

PARAMETERS                   *win_iis_webbinding-parameters*

         Parameter        Choices/Defaults                                                                          Comments
   certificate_hash                        Certificate hash (thumbprint) for the SSL binding. The certificate hash is the unique identifier for the certificate.
   string
   certificate_store_name Default:         Name of the certificate store where the certificate for the binding is located.
   string                 "my"
   host_header                             The host header to bind to / use for the new site.
   string                                  If you are creating/removing a catch-all binding, omit this parameter rather than defining it as '"'.
   ip                     Default:         The IP address to bind to / use for the new site.
   string                 """
   name                                    Names of web site.
   string / required                       aliases: website
   port                   Default:         The port to bind to / use for the new site.
   string                 80
   protocol               Default:         The protocol to be used for the Web binding (usually HTTP, HTTPS, or FTP).
   string                 "http"
   ssl_flags                               This parameter is only valid on Server 2012 and newer.
   string                                  Primarily used for enabling and disabling server name indication (SNI).
   added in 2.5                            Set to c(0) to disable SNI.
                                           Set to c(1) to enable SNI.
   state                    Choices:
   string                   • absent       State of the binding.
                            • present ←

EXAMPLES                   *win_iis_webbinding-examples*

 - name: Add a HTTP binding on port 9090
   win_iis_webbinding:
     name: Default Web Site
     port: 9090
     state: present

 - name: Remove the HTTP binding on port 9090
   win_iis_webbinding:
     name: Default Web Site
     port: 9090
     state: absent

 - name: Remove the default http binding
   win_iis_webbinding:
     name: Default Web Site
     port: 80
     ip: '"'
     state: absent

 - name: Add a HTTPS binding
   win_iis_webbinding:
     name: Default Web Site
     protocol: https
     port: 443
     ip: 127.0.0.1
     certificate_hash: B0D0FA8408FC67B230338FCA584D03792DA73F4C
     state: present

 - name: Add a HTTPS binding with host header and SNI enabled
   win_iis_webbinding:
     name: Default Web Site
     protocol: https
     port: 443
     host_header: test.com
     ssl_flags: 1
     certificate_hash: D1A3AF8988FD32D1A3AF8988FD323792DA73F4C
     state: present

RETURN VALUES                   *win_iis_webbinding-return values*

   Common return values are documented here, the following are the fields unique to this module:

        Key       Returned                                                                                Description
   binding_info            Information on the binding being manipulated
   dictionary     on       Sample:
   added in 2.5   success  "binding_info": { "bindingInformation": "127.0.0.1:443:", "certificateHash": "FF3910CE089397F1B5A77EB7BAFDD8F44CDE77DD", "certificateStoreName": "MY", "hostheader": "",
                           "ip": "127.0.0.1", "port": 443, "protocol": "https", "sslFlags": "not supported" }
   operation_type          The type of operation performed
   string         on       Can be removed, updated, matched, or added
   added in 2.5   success  Sample:
                           removed
   website_state           The state of the website being targetted
   string         always   Can be helpful in case you accidentally cause a binding collision which can result in the targetted site being stopped
   added in 2.5            Sample:
                           Started
