SYNOPSIS                   *vcenter_extension_info*

     • This module can be used to gather information about vCenter extension.

REQUIREMENTS                   *vcenter_extension_info-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vcenter_extension_info-notes*

   Note

     • Tested on vSphere 6.5

EXAMPLES                   *vcenter_extension_info-examples*

 - name: Gather info about vCenter Extensions
   vcenter_extension_info:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
   register: ext_info
   delegate_to: localhost
