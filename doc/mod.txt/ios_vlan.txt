SYNOPSIS                   *ios_vlan-synopsis*

     • This module provides declarative management of VLANs on Cisco IOS network devices.

EXAMPLES                   *ios_vlan-examples*

 - name: Create vlan
   ios_vlan:
     vlan_id: 100
     name: test-vlan
     state: present

 - name: Add interfaces to VLAN
   ios_vlan:
     vlan_id: 100
     interfaces:
       - GigabitEthernet0/0
       - GigabitEthernet0/1

 - name: Check if interfaces is assigned to VLAN
   ios_vlan:
     vlan_id: 100
     associated_interfaces:
       - GigabitEthernet0/0
       - GigabitEthernet0/1

 - name: Delete vlan
   ios_vlan:
     vlan_id: 100
     state: absent

 - name: Add vlan using aggregate
   ios_vlan:
     aggregate:
     - { vlan_id: 100, name: test-vlan, interfaces: [GigabitEthernet0/1, GigabitEthernet0/2], delay: 15, state: suspend }
     - { vlan_id: 101, name: test-vlan, interfaces: GigabitEthernet0/3 }

 - name: Move interfaces to a different VLAN
   ios_vlan:
     vlan_id: 102
     interfaces:
       - GigabitEthernet0/0
       - GigabitEthernet0/1
