SYNOPSIS                   *pn_fabric_local*

     • This module can be used to modify fabric local information.

PARAMETERS                   *pn_fabric_local-parameters*

              Parameter             Choices/Defaults                                                                     Comments
   pn_cliswitch                                       Target switch to run the CLI on.
   string / required
                                     Choices:
   pn_control_network                • in-band        control plane network.
   -                                 • mgmt
                                     • vmgmt
                                     Choices:
   pn_fabric_advertisement_network   • inband-mgmt
   -                                 • inband-only    network to send fabric advertisements on.
                                     • inband-vmgmt
                                     • mgmt-only
                                     Choices:
   pn_fabric_network                 • in-band        fabric administration network.
   -                                 • mgmt ←
                                     • vmgmt
   pn_vlan                                            VLAN assigned to fabric.
   string
   state                             Choices:         State the action to perform. Use update to modify the fabric-local.
   string                            • update ←

EXAMPLES                   *pn_fabric_local-examples*

 - name: Fabric local module
   pn_fabric_local:
     pn_cliswitch: "sw01"
     pn_vlan: "500"

 - name: Fabric local module
   pn_fabric_local:
     pn_cliswitch: "sw01"
     pn_fabric_advertisement_network: "mgmt-only"

RETURN VALUES                   *pn_fabric_local-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the fabric-local command.
   list
   stdout  always   set of responses from the fabric-local command.
   list
