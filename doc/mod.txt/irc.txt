SYNOPSIS                   *irc*

     • Send a message to an IRC channel. This is a very simplistic implementation.

REQUIREMENTS                   *irc-requirements*

   The below requirements are needed on the host that executes this module.

     • socket

PARAMETERS                   *irc-parameters*

   Parameter  Choices/Defaults                                                                                Comments
   channel
   - /                         Channel name. One of nick_to or channel needs to be set. When both are set, the message will be sent to both of them.
   required
                Choices:
                • none ←
                • white
                • black
                • blue
                • green
                • red
                • brown
   color        • purple       Text color for the message. ("none" is a valid option in 1.6 or later, in 1.6 and prior, the default color is black, not "none"). Added 11 more colors in version 2.0.
   -            • orange
                • yellow
                • light_green
                • teal
                • light_cyan
                • light_blue
                • pink
                • gray
                • light_gray
   key                         Channel key
   -
   msg
   - /                         The message body.
   required
   nick       Default:         Nickname to send the message from. May be shortened, depending on server's NICKLEN setting.
   -          "ansible"
   nick_to                     A list of nicknames to send the message to. One of nick_to or channel needs to be set. When both are defined, the message will be sent to both of them.
   -
   part         Choices:
   boolean      • no           Designates whether user should part from channel after sending message or not. Useful for when using a faux bot and not wanting join/parts between messages.
                • yes ←
   passwd                      Server password
   -
   port       Default:         IRC server port number
   -          6667
   server     Default:         IRC server name/address
   -          "localhost"
                Choices:
   style        • bold
   -            • underline    Text style for the message. Note italic does not work on some clients
                • reverse
                • italic
   timeout    Default:         Timeout to use while waiting for successful registration and join messages, this is to prevent an endless loop
   -          30
   topic                       Set the channel topic
   -
   use_ssl      Choices:
   boolean      • no ←         Designates whether TLS/SSL should be used when connecting to the IRC server
                • yes

EXAMPLES                   *irc-examples*

 - irc:
     server: irc.example.net
     channel: #t1
     msg: Hello world

 - local_action:
     module: irc
     port: 6669
     server: irc.example.net
     channel: #t1
     msg: 'All finished at {{ ansible_date_time.iso8601 }}'
     color: red
     nick: ansibleIRC

 - local_action:
     module: irc
     port: 6669
     server: irc.example.net
     channel: #t1
     nick_to:
       - nick1
       - nick2
     msg: 'All finished at {{ ansible_date_time.iso8601 }}'
     color: red
     nick: ansibleIRC
