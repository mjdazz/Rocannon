SYNOPSIS                   *fortios_ips_rule*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify ips feature and rule category. Examples include all parameters and values need to
       be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_ips_rule-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_ips_rule-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure IPS rules.
     fortios_ips_rule:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       ips_rule:
         action: "pass"
         application: "<your_own_value>"
         date: "5"
         group: "<your_own_value>"
         location: "<your_own_value>"
         log: "disable"
         log_packet: "disable"
         metadata:
          -
             id:  "11"
             metaid: "12"
             valueid: "13"
         name: "default_name_14"
         os: "<your_own_value>"
         rev: "16"
         rule_id: "17"
         service: "<your_own_value>"
         severity: "<your_own_value>"
         status: "disable"
