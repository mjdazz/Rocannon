SYNOPSIS                   *hetzner_failover_ip_info*

     • Retrieve information on Hetzner’s failover IPs.

PARAMETERS                   *hetzner_failover_ip_info-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   failover_ip                        The failover IP address.
   string / required
   hetzner_password                   The password for the Robot webservice user.
   string / required
   hetzner_user                       The username for the Robot webservice user.
   string / required

SEE ALSO                   *hetzner_failover_ip_info-see also*

   See also

   Failover IP documentation
   Hetzner’s documentation on failover IPs.

   hetzner_failover_ip – Manage Hetzner’s failover IPs
   Manage failover IPs.

EXAMPLES                   *hetzner_failover_ip_info-examples*

 - name: Get value of failover IP 1.2.3.4
   hetzner_failover_ip_info:
     hetzner_user: foo
     hetzner_password: bar
     failover_ip: 1.2.3.4
     value: 5.6.7.8
   register: result

 - name: Print value of failover IP 1.2.3.4 in case it is routed
   debug:
     msg: "1.2.3.4 routes to {{ result.value }}"
   when: result.state == 'routed'

RETURN VALUES                   *hetzner_failover_ip_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key        Returned                                                                               Description
   failover_ip               The failover IP.
   string           success  Sample:
                             1.2.3.4
   failover_netmask          The netmask for the failover IP.
   string           success  Sample:
                             255.255.255.255
   server_ip        success  The main IP of the server this failover IP is associated to.
   string                    This is not the server the failover IP is routed to.
   server_number    success  The number of the server this failover IP is associated to.
   integer                   This is not the server the failover IP is routed to.
   state            success  Will be routed or unrouted.
   string
   value            success  The value of the failover IP.
   string                    Will be none if the IP is unrouted.
