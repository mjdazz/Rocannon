SYNOPSIS                   *aws_netapp_cvs_snapshots*

     • Create, Update, Delete Snapshot on AWS Cloud Volumes Service.

PARAMETERS                   *aws_netapp_cvs_snapshots-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_key                            The access key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   api_url                            The url to the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   fileSystemId                       Name or Id of the filesystem.
   string                             Required for create operation
   from_name                          ID or Name of the snapshot to rename.
   string                             Required to create an snapshot called 'name' by renaming 'from_name'.
   name                               Name of the snapshot
   string / required
   region                             The region to which the snapshot belongs to.
   string / required
   secret_key                         The secret_key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   state               Choices:
   string / required   • present      Whether the specified snapshot should exist or not.
                       • absent
   validate_certs      Choices:
   boolean             • no           Should https certificates be validated?
                       • yes ←

NOTES                   *aws_netapp_cvs_snapshots-notes*

   Note

     • The modules prefixed with aws_cvs_netapp are built to Manage AWS Cloud Volume Service .

EXAMPLES                   *aws_netapp_cvs_snapshots-examples*

 - name: Create Snapshot
   aws_netapp_cvs_snapshots:
     state: present
     region: us-east-1
     name: testSnapshot
     fileSystemId: testVolume
     api_url : cds-aws-bundles.netapp.com
     api_key: myApiKey
     secret_key : mySecretKey

 - name: Update Snapshot
   aws_netapp_cvs_snapshots:
     state: present
     region: us-east-1
     name: testSnapshot - renamed
     from_name: testSnapshot
     fileSystemId: testVolume
     api_url : cds-aws-bundles.netapp.com
     api_key: myApiKey
     secret_key : mySecretKey

 - name: Delete Snapshot
   aws_netapp_cvs_snapshots:
     state: absent
     region: us-east-1
     name: testSnapshot
     api_url : cds-aws-bundles.netapp.com
     api_key: myApiKey
     secret_key : mySecretKey
