SYNOPSIS                   *cpm_serial_port_info*

     • Get Serial port parameters from WTI OOB and PDU devices

PARAMETERS                   *cpm_serial_port_info-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   cpm_password                    This is the Password of the WTI device to send the module.
   - / required
   cpm_url                         This is the URL of the WTI device to send the module.
   - / required
   cpm_username                    This is the Username of the WTI device to send the module.
   - / required
   port           Default:         This is the serial port number that is getting retrieved. It can include a single port number, multiple port numbers separated by commas, a list of port numbers,
   list /         ["""]            or an '"' character for all ports.
   required
   use_https        Choices:
   boolean          • no           Designates to use an https connection or http connection.
                    • yes ←
   use_proxy        Choices:
   boolean          • no ←         Flag to control if the lookup will observe HTTP proxy environment variables when present.
                    • yes
   validate_certs   Choices:       If false, SSL certificates will not be validated. This should only be used
   boolean          • no           on personally controlled sites using self-signed certificates.
                    • yes ←

NOTES                   *cpm_serial_port_info-notes*

   Note

     • Use groups/cpm in module_defaults to set common options used between CPM modules.)

EXAMPLES                   *cpm_serial_port_info-examples*

 - name: Get the Serial Port Parameters for port 2 of a WTI device
   cpm_serial_port_info:
     cpm_url: "nonexist.wti.com"
     cpm_username: "super"
     cpm_password: "super"
     use_https: true
     validate_certs: false
     port: 2

 - name: Get the Serial Port Parameters for ports 2 and 4 of a WTI device
   cpm_serial_port_info:
     cpm_url: "nonexist.wti.com"
     cpm_username: "super"
     cpm_password: "super"
     use_https: true
     validate_certs: false
     port: 2,4

 - name: Get the Serial Port Parameters for all ports of a WTI device
   cpm_serial_port_info:
     cpm_url: "nonexist.wti.com"
     cpm_username: "super"
     cpm_password: "super"
     use_https: true
     validate_certs: false
     port: """
