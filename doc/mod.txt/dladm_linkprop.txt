SYNOPSIS                   *dladm_linkprop*

     • Set / reset link properties on Solaris/illumos systems.

PARAMETERS                   *dladm_linkprop-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   link                          Link interface name.
   - / required                  aliases: nic, interface
   property                      Specifies the name of the property we want to manage.
   - / required                  aliases: name
                  Choices:
   state          • present ←    Set or reset the property value.
   -              • absent
                  • reset
   temporary      Choices:
   boolean        • no ←         Specifies that lin property configuration is temporary. Temporary link property configuration does not persist across reboots.
                  • yes
   value                         Specifies the value we want to set for the link property.
   -

EXAMPLES                   *dladm_linkprop-examples*

 - name: Set 'maxbw' to 100M on e1000g1
   dladm_linkprop: name=e1000g1 property=maxbw value=100M state=present

 - name: Set 'mtu' to 9000 on e1000g1
   dladm_linkprop: name=e1000g1 property=mtu value=9000

 - name: Reset 'mtu' property on e1000g1
   dladm_linkprop: name=e1000g1 property=mtu state=reset

RETURN VALUES                   *dladm_linkprop-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key    Returned                                                                                   Description
   link               link name
   string    always   Sample:
                      e100g0
   property           property name
   string    always   Sample:
                      mtu
   state              state of the target
   string    always   Sample:
                      present
   temporary          specifies if operation will persist across reboots
   boolean   always   Sample:
                      True
   value              property value
   string    always   Sample:
                      9000
