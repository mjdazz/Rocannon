SYNOPSIS                   *purefb_subnet*

     • This module manages network subnets on Pure Storage FlashBlade.

REQUIREMENTS                   *purefb_subnet-requirements*

   The below requirements are needed on the host that executes this module.

     • netaddr
     • purity_fb >= 1.1
     • python >= 2.7

PARAMETERS                   *purefb_subnet-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_token                          FlashBlade API token for admin privileged user.
   string
   fb_url                             FlashBlade management IP address or Hostname.
   string
   gateway                            IPv4 or IPv6 address of subnet gateway.
   string
   mtu               Default:         MTU size of the subnet. Range is 1280 to 9216.
   integer           1500
   name                               Subnet Name.
   string / required
   prefix                             IPv4 or IPv6 address associated with the subnet.
   string                             Supply the prefix length (CIDR) as well as the IP address.
   state               Choices:
   string              • present ←    Create, delete or modifies a subnet.
                       • absent
   vlan              Default:         VLAN ID of the subnet.
   integer           0

NOTES                   *purefb_subnet-notes*

   Note

     • Requires the netaddr Python package on the host.
     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_subnet-examples*

 - name: Create new network subnet named foo
   purefb_subnet:
     name: foo
     prefix: "10.21.200.3/24"
     gateway: 10.21.200.1
     mtu: 9000
     vlan: 2200
     state: present
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Change configuration of existing subnet foo
   purefb_network:
     name: foo
     state: present
     prefix: "10.21.100.3/24"
     gateway: 10.21.100.1
     mtu: 1500
     address: 10.21.200.123
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Delete network subnet named foo
   purefb_subnet:
     name: foo
     state: absent
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641
