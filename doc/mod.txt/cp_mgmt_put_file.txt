SYNOPSIS                   *cp_mgmt_put_file*

     • put file on Check Point over Web Services API
     • All operations are performed over Web Services API.

PARAMETERS                   *cp_mgmt_put_file-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   comments                       Comments string.
   string
   file_content                   N/A
   string
   file_name                      N/A
   string
   file_path                      N/A
   string
   targets                        On what targets to execute this command. Targets may be identified by their name, or object unique identifier.
   list
   version                        Version of checkpoint. If not given one, the latest version taken.
   string
   wait_for_task   Choices:
   boolean         • no           Wait for the task to end. Such as publish task.
                   • yes ←

EXAMPLES                   *cp_mgmt_put_file-examples*

 - name: put-file
   cp_mgmt_put_file:
     file_content: 'vs ip 192.0.2.1
 vs2 ip 192.0.2.2'
     file_name: vsx_conf
     file_path: /home/admin/
     targets:
     - corporate-gateway
