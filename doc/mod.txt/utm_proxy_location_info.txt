SYNOPSIS                   *utm_proxy_location_info*

     • Create, update or destroy a reverse_proxy location entry in SOPHOS UTM.
     • This module needs to have the REST Ability of the UTM to be activated.

PARAMETERS                   *utm_proxy_location_info-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   headers                         A dictionary of additional headers to be sent to POST and PUT requests.
   dictionary                      Is needed for some modules

   state            Choices:       The desired state of the object.
   string           • absent       present will create or update an object

   required
