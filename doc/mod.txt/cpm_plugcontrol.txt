SYNOPSIS                   *cpm_plugcontrol*

     • Get and Set Plug actions on WTI OOB and PDU devices

PARAMETERS                   *cpm_plugcontrol-parameters*

     Parameter     Choices/Defaults                                                                              Comments
   cpm_action       Choices:
   - / required     • getplugcontrol This is the Action to send the module.
                    • setplugcontrol
   cpm_password                      This is the Password of the WTI device to send the module.
   -
   cpm_url                           This is the URL of the WTI device to send the module.
   - / required
   cpm_username                      This is the Username of the WTI device to send the module.
   -
   plug_id                           This is the plug number or the plug name that is to be manipulated For the plugget command, the plug_id 'all' will return the status of all the plugs the user
   - / required                      has rights to access.
                    Choices:
   plug_state       • on
   -                • off            This is what action to take on the plug.
                    • boot
                    • default
   use_https        Choices:
   boolean          • no             Designates to use an https connection or http connection.
                    • yes ←
   use_proxy        Choices:
   boolean          • no ←           Flag to control if the lookup will observe HTTP proxy environment variables when present.
                    • yes
   validate_certs   Choices:         If false, SSL certificates will not be validated. This should only be used
   boolean          • no             on personally controlled sites using self-signed certificates.
                    • yes ←

EXAMPLES                   *cpm_plugcontrol-examples*

 # Get Plug status for all ports
 - name: Get the Plug status for ALL ports of a WTI device
   cpm_plugcontrol:
     cpm_action: "getplugcontrol"
     cpm_url: "rest.wti.com"
     cpm_username: "restpower"
     cpm_password: "restfulpowerpass12"
     use_https: true
     validate_certs: true
     plug_id: "all"

 # Get Plug status for port 2
 - name: Get the Plug status for the given port of a WTI device
   cpm_plugcontrol:
     cpm_action: "getplugcontrol"
     cpm_url: "rest.wti.com"
     cpm_username: "restpower"
     cpm_password: "restfulpowerpass12"
     use_https: true
     validate_certs: false
     plug_id: "2"

 # Reboot plug 5
 - name: Reboot Plug 5 on a given WTI device
   cpm_plugcontrol:
     cpm_action: "setplugcontrol"
     cpm_url: "rest.wti.com"
     cpm_username: "restpower"
     cpm_password: "restfulpowerpass12"
     use_https: true
     plug_id: "5"
     plug_state: "boot"
