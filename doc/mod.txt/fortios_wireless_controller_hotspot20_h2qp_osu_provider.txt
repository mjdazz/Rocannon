SYNOPSIS                   *fortios_wireless_controller_hotspot20_h2qp_osu_provider*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify wireless_controller_hotspot20 feature and h2qp_osu_provider category. Examples
       include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_wireless_controller_hotspot20_h2qp_osu_provider-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_wireless_controller_hotspot20_h2qp_osu_provider-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure online sign up (OSU) provider list.
     fortios_wireless_controller_hotspot20_h2qp_osu_provider:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       wireless_controller_hotspot20_h2qp_osu_provider:
         friendly_name:
          -
             friendly_name: "<your_own_value>"
             index: "5"
             lang: "<your_own_value>"
         icon: "<your_own_value> (source wireless-controller.hotspot20.icon.name)"
         name: "default_name_8"
         osu_method: "oma-dm"
         osu_nai: "<your_own_value>"
         server_uri: "<your_own_value>"
         service_description:
          -
             lang: "<your_own_value>"
             service_description: "<your_own_value>"
             service_id: "15"
