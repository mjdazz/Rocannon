SYNOPSIS                   *pn_log_audit_exception*

     • This module can be used to create an audit exception and delete an audit exception.

PARAMETERS                   *pn_log_audit_exception-parameters*

       Parameter     Choices/Defaults                                                                            Comments
                       Choices:
   pn_access           • any          Specify the access type to match exceptions.
   string / required   • read-only
                       • read-write
                       Choices:
   pn_audit_type       • cli          Specify the type of audit exception.
   string              • shell
                       • vtysh
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_pattern                         Specify a regular expression to match exceptions.
   string
   pn_scope            Choices:
   string              • local        scope - local or fabric.
                       • fabric
   state               Choices:
   string              • present ←    State the action to perform. Use 'present' to create audit-exception and 'absent' to delete audit-exception.
                       • absent

EXAMPLES                   *pn_log_audit_exception-examples*

 - name: create a log-audit-exception
   pn_log_audit_exception:
     pn_audit_type: "cli"
     pn_pattern: "test"
     state: "present"
     pn_access: "any"
     pn_scope: "local"

 - name: delete a log-audit-exception
   pn_log_audit_exception:
     pn_audit_type: "shell"
     pn_pattern: "test"
     state: "absent"
     pn_access: "any"

RETURN VALUES                   *pn_log_audit_exception-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the log_audit_exceptions command.
   list
   stdout  always   set of responses from the pn_log_audit_exceptions command.
   list
