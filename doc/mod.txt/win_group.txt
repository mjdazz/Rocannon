SYNOPSIS                   *win_group*

     • Add and remove local groups.
     • For non-Windows targets, please use the group module instead.

PARAMETERS                   *win_group-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   description                        Description of the group.
   string
   name                               Name of the group.
   string / required
   state               Choices:
   string              • absent       Create or remove the group.
                       • present ←

SEE ALSO                   *win_group-see also*

   See also

   group – Add or remove groups
   The official documentation on the group module.

   win_domain_group – Creates, modifies or removes domain groups
   The official documentation on the win_domain_group module.

   win_group_membership – Manage Windows local group membership
   The official documentation on the win_group_membership module.

EXAMPLES                   *win_group-examples*

 - name: Create a new group
   win_group:
     name: deploy
     description: Deploy Group
     state: present

 - name: Remove a group
   win_group:
     name: deploy
     state: absent
