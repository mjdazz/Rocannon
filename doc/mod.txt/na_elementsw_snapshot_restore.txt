SYNOPSIS                   *na_elementsw_snapshot_restore*

     • Element OS Cluster restore snapshot to volume.

REQUIREMENTS                   *na_elementsw_snapshot_restore-requirements*

   The below requirements are needed on the host that executes this module.

     • The modules were developed with SolidFire 10.1
     • solidfire-sdk-python (1.1.0.92) or greater. Install using ‘pip install solidfire-sdk-python’

PARAMETERS                   *na_elementsw_snapshot_restore-parameters*

      Parameter     Choices/Defaults                                                                             Comments
   account_id                        Account ID or Name of Parent/Source Volume.
   - / required
   dest_volume_name                  New Name of destination for restoring the snapshot
   - / required
   hostname                          The hostname or IP address of the SolidFire cluster.
   - / required
   password                          Password for the specified user.
   - / required                      aliases: pass
   src_snapshot_id                   ID or Name of an existing snapshot.
   - / required
   src_volume_id                     ID or Name of source active volume.
   - / required
   username                          Please ensure that the user has the adequate permissions. For more information, please read the official documentation
   - / required                      https://mysupport.netapp.com/documentation/docweb/index.html?productID=62636&language=en-US.
                                     aliases: user

NOTES                   *na_elementsw_snapshot_restore-notes*

   Note

     • The modules prefixed with na\_elementsw are built to support the SolidFire storage platform.
