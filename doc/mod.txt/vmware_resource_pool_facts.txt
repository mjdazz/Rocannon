DEPRECATED                   *vmware_resource_pool_facts*

                    Removed in Ansible:
                  version: 2.13
        Why:      Deprecated in favour of _info module.
    Alternative:  Use vmware_resource_pool_info instead.

SYNOPSIS                   *vmware_resource_pool_facts-synopsis*

     • This module can be used to gather facts about all resource configuration information.

REQUIREMENTS                   *vmware_resource_pool_facts-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_resource_pool_facts-notes*

   Note

     • Tested on vSphere 6.5

EXAMPLES                   *vmware_resource_pool_facts-examples*

 - name: Gather resource pool facts about all resource pools available
   vmware_resource_pool_facts:
     hostname: '{{ vcenter_hostname }}'
     username: '{{ vcenter_username }}'
     password: '{{ vcenter_password }}'
   register: rp_facts
   delegate_to: localhost
