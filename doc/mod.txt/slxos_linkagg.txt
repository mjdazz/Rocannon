SYNOPSIS                   *slxos_linkagg*

     • This module provides declarative management of link aggregation groups on Extreme Networks SLX-OS network devices.

PARAMETERS                   *slxos_linkagg-parameters*

   Parameter Choices/Defaults                                                                                Comments
   aggregate                  List of link aggregation definitions.
   -
   group                      Channel-group number for the port-channel Link aggregation group. Range 1-1024.
   -
   members                    List of members of the link aggregation group.
   -
               Choices:
   mode        • active       Mode of the link aggregation group.
   -           • on
               • passive
   purge       Choices:
   boolean     • no           Purge links not defined in the aggregate parameter.
               • yes
   state       Choices:
   -           • present ←    State of the link aggregation group.
               • absent

NOTES                   *slxos_linkagg-notes*

   Note

     • Tested against SLX-OS 17s.1.02

EXAMPLES                   *slxos_linkagg-examples*

 - name: create link aggregation group
   slxos_linkagg:
     group: 10
     state: present

 - name: delete link aggregation group
   slxos_linkagg:
     group: 10
     state: absent

 - name: set link aggregation group to members
   slxos_linkagg:
     group: 200
     mode: active
     members:
       - Ethernet 0/1
       - Ethernet 0/2

 - name: remove link aggregation group from Ethernet 0/1
   slxos_linkagg:
     group: 200
     mode: active
     members:
       - Ethernet 0/1

 - name: Create aggregate of linkagg definitions
   slxos_linkagg:
     aggregate:
       - { group: 3, mode: on, members: [Ethernet 0/1] }
       - { group: 100, mode: passive, members: [Ethernet 0/2] }
