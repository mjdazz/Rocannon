SYNOPSIS                   *kernel_blacklist*

     • Add or remove kernel modules from blacklist.

PARAMETERS                   *kernel_blacklist-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   blacklist_file                  If specified, use this blacklist file instead of /etc/modprobe.d/blacklist-ansible.conf.
   -
   name                            Name of kernel module to black- or whitelist.
   - / required
   state            Choices:
   -                • absent       Whether the module should be present in the blacklist or absent.
                    • present ←

EXAMPLES                   *kernel_blacklist-examples*

 - name: Blacklist the nouveau driver module
   kernel_blacklist:
     name: nouveau
     state: present
