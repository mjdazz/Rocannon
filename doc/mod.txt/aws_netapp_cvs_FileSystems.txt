SYNOPSIS                   *aws_netapp_cvs_FileSystems*

     • Create, Update, Delete fileSystem on AWS Cloud Volumes Service.

PARAMETERS                   *aws_netapp_cvs_FileSystems-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_key                            The access key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   api_url                            The url to the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
   creationToken                      Name of the filesystem
   string / required
   exportPolicy                       The policy rules to export the filesystem
   dictionary
     rules                            Set of rules to export the filesystem
     list                             Requires allowedClients, access and protocol
      allowedClients                  Comma separated list of ip address blocks of the clients to access the fileSystem
      string                          Each address block contains the starting IP address and size for the block
      cifs             Choices:
      boolean          • no           Enable or disable cifs filesystem
                       • yes
      nfsv3            Choices:
      boolean          • no           Enable or disable nfsv3 fileSystem
                       • yes
      nfsv4            Choices:
      boolean          • no           Enable or disable nfsv4 filesystem
                       • yes
      ruleIndex                       Index number of the rule
      integer
      unixReadOnly     Choices:
      boolean          • no           Should fileSystem have read only permission or not
                       • yes
      unixReadWrite    Choices:
      boolean          • no           Should fileSystem have read write permission or not
                       • yes
   quotaInBytes                       Size of the filesystem
   integer                            Required for create
   region                             The region to which the filesystem belongs to.
   string / required
   secret_key                         The secret_key to authenticate with the AWSCVS Web Services Proxy or Embedded Web Services API.
   string / required
                       Choices:
   serviceLevel        • standard     Service Level of a filesystem.
   string              • premium
                       • extreme
   state               Choices:
   string / required   • present      Whether the specified fileSystem should exist or not.
                       • absent
   validate_certs      Choices:
   boolean             • no           Should https certificates be validated?
                       • yes ←

NOTES                   *aws_netapp_cvs_FileSystems-notes*

   Note

     • The modules prefixed with aws_cvs_netapp are built to Manage AWS Cloud Volume Service .

EXAMPLES                   *aws_netapp_cvs_FileSystems-examples*

 - name: Create FileSystem
   aws_netapp_cvs_FileSystems:
     state: present
     region: us-east-1
     creationToken: newVolume-1
     exportPolicy:
         rules:
           - allowedClients: 172.16.0.4
             cifs: False
             nfsv3: True
             nfsv4: True
             ruleIndex: 1
             unixReadOnly: True
             unixReadWrite: False
     quotaInBytes: 100000000000
     api_url : cds-aws-bundles.netapp.com
     api_key: Q1ZRR0p0VGNuZ3VhMnJBYk5zczM1RkZ3Z0lCbUE3
     secret_key : U1FwdHdKSGRQQUhIdkIwMktMU1ZCV2x6WUowZWRD

 - name: Update FileSystem
   aws_netapp_cvs_FileSystems:
      state: present
      region: us-east-1
      creationToken: newVolume-1
      exportPolicy:
          rules:
            - allowedClients: 172.16.0.4
              cifs: False
              nfsv3: True
              nfsv4: True
              ruleIndex: 1
              unixReadOnly: True
              unixReadWrite: False
      quotaInBytes: 200000000000
      api_url : cds-aws-bundles.netapp.com
      api_key: Q1ZRR0p0VGNuZ3VhMnJBYk5zczM1RkZ3Z0lCbUE3
      secret_key : U1FwdHdKSGRQQUhIdkIwMktMU1ZCV2x6WUowZWRD

 - name: Delete FileSystem
   aws_netapp_cvs_FileSystems:
      state: present
      region: us-east-1
      creationToken: newVolume-1
      quotaInBytes: 100000000000
      api_url : cds-aws-bundles.netapp.com
      api_key: Q1ZRR0p0VGNuZ3VhMnJBYk5zczM1RkZ3Z0lCbUE3
      secret_key : U1FwdHdKSGRQQUhIdkIwMktMU1ZCV2x6WUowZWRD
