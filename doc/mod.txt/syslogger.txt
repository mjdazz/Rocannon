SYNOPSIS                   *syslogger*

     • Uses syslog to add log entries to the host.
     • Can specify facility and priority.

EXAMPLES                   *syslogger-examples*

 # Full example
 - name: Test syslog
   syslogger:
     msg: "Hello from ansible"
     priority: "err"
     facility: "daemon"
     log_pid: true

 # Basic usage
 - name: Simple Usage
   syslogger:
     msg: "I will end up as daemon.info"
