SYNOPSIS                   *fortios_authentication_setting*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify authentication feature and setting category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_authentication_setting-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

PARAMETERS                   *fortios_authentication_setting-parameters*

         Parameter        Choices/Defaults                                                                          Comments
   authentication_setting Default:         Configure authentication setting.
   dictionary             null

     captive_portal                        Captive portal host name. Source firewall.address.name.

     string

     string                 • fqdn         Captive portal type.
                            • ip

   string

                            • yes ←

NOTES                   *fortios_authentication_setting-notes*

   Note

     • Requires fortiosapi library developed by Fortinet
     • Run as a local_action in your playbook

EXAMPLES                   *fortios_authentication_setting-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure authentication setting.
     fortios_authentication_setting:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       authentication_setting:
         active_auth_scheme: "<your_own_value> (source authentication.scheme.name)"
         captive_portal: "<your_own_value> (source firewall.address.name)"
         captive_portal_ip: "<your_own_value>"
         captive_portal_ip6: "<your_own_value>"
         captive_portal_port: "7"
         captive_portal_type: "fqdn"
         captive_portal6: "<your_own_value> (source firewall.address6.name)"
         sso_auth_scheme: "<your_own_value> (source authentication.scheme.name)"
