SYNOPSIS                   *hcloud_rdns*

     • Create, update and delete reverse DNS entries on the Hetzner Cloud.

REQUIREMENTS                   *hcloud_rdns-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0
     • hcloud-python >= 1.3.0

PARAMETERS                   *hcloud_rdns-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   dns_ptr                                          The DNS address the ip_address should resolve to.
   string                                           Omit the param to reset the reverse DNS entry to the default value.
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   ip_address                                       The IP address that should point to dns_ptr.
   string / required
   server                                           The name of the Hetzner Cloud server you want to add the reverse DNS entry to.
   string / required
   state               Choices:
   string              • absent                     State of the reverse DNS entry.
                       • present ←

SEE ALSO                   *hcloud_rdns-see also*

   See also

   Documentation for Hetzner Cloud API
   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_rdns-examples*

 - name: Create a reverse DNS entry for a server
   hcloud_rdns:
     server: my-server
     ip_address: 123.123.123.123
     dns_ptr: example.com
     state: present

 - name: Ensure the reverse DNS entry is absent (remove if needed)
   hcloud_rdns:
     server: my-server
     ip_address: 123.123.123.123
     dns_ptr: example.com
     state: absent

RETURN VALUES                   *hcloud_rdns-return values*

   Common return values are documented here, the following are the fields unique to this module:

       Key      Returned                                                                                 Description
   hcloud_rdns  always   The reverse DNS entry
   complex
     dns_ptr             The DNS that resolves to the IP
     string     always   Sample:
                         example.com
     ip_address          The IP address that point to the DNS ptr
     string     always   Sample:
                         123.123.123.123
     server              Name of the server
     string     always   Sample:
                         my-server
