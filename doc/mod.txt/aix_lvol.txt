SYNOPSIS                   *aix_lvol*

     • This module creates, removes or resizes AIX logical volumes. Inspired by lvol module.

PARAMETERS                   *aix_lvol-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   copies            Default:         The number of copies of the logical volume.
   integer           1                Maximum copies are 3.
   lv                                 The name of the logical volume.
   string / required
   lv_type           Default:         The type of the logical volume.
   string            "jfs2"
   opts                               Free-form options to be passed to the mklv command.
   string
   policy              Choices:       Sets the interphysical volume allocation policy.
   string              • maximum ←    maximum allocates logical partitions across the maximum number of physical volumes.
                       • minimum      minimum allocates logical partitions across the minimum number of physical volumes.
   pvs                                A list of physical volumes e.g. hdisk1,hdisk2.
   list
   size                               The size of the logical volume with one of the [MGT] units.
   string
   state               Choices:
   string              • absent       Control if the logical volume exists. If present and the volume does not already exist then the size option is required.
                       • present ←
   vg                                 The volume group this logical volume is part of.
   string / required

EXAMPLES                   *aix_lvol-examples*

 - name: Create a logical volume of 512M
   aix_lvol:
     vg: testvg
     lv: testlv
     size: 512M

 - name: Create a logical volume of 512M with disks hdisk1 and hdisk2
   aix_lvol:
     vg: testvg
     lv: test2lv
     size: 512M
     pvs: [ hdisk1, hdisk2 ]

 - name: Create a logical volume of 512M mirrored
   aix_lvol:
     vg: testvg
     lv: test3lv
     size: 512M
     copies: 2

 - name: Create a logical volume of 1G with a minimum placement policy
   aix_lvol:
     vg: rootvg
     lv: test4lv
     size: 1G
     policy: minimum

 - name: Create a logical volume with special options like mirror pool
   aix_lvol:
     vg: testvg
     lv: testlv
     size: 512M
     opts: -p copy1=poolA -p copy2=poolB

 - name: Extend the logical volume to 1200M
   aix_lvol:
     vg: testvg
     lv: test4lv
     size: 1200M

 - name: Remove the logical volume
   aix_lvol:
     vg: testvg
     lv: testlv
     state: absent

RETURN VALUES                   *aix_lvol-return values*

   Common return values are documented here, the following are the fields unique to this module:

    Key   Returned                                                                                    Description
   msg             A friendly message describing the task result.
   string always   Sample:
                   Logical volume testlv created.
