SYNOPSIS                   *ce_bfd_view*

     • Manages BFD session view configuration on HUAWEI CloudEngine devices.

NOTES                   *ce_bfd_view-notes*

   Note

     • This module requires the netconf system service be enabled on the remote device being managed.
     • Recommended connection is netconf.
     • This module also works with local connections for legacy playbooks.

EXAMPLES                   *ce_bfd_view-examples*

 - name: bfd view module test
   hosts: cloudengine
   connection: local
   gather_facts: no
   vars:
     cli:
       host: "{{ inventory_hostname }}"
       port: "{{ ansible_ssh_port }}"
       username: "{{ username }}"
       password: "{{ password }}"
       transport: cli

   tasks:
   - name: Set the local discriminator of a BFD session to 80 and the remote discriminator to 800
     ce_bfd_view:
       session_name: atob
       local_discr: 80
       remote_discr: 800
       state: present
       provider: '{{ cli }}'

   - name: Set the minimum interval for receiving BFD packets to 500 ms
     ce_bfd_view:
       session_name: atob
       min_rx_interval: 500
       state: present
       provider: '{{ cli }}'
