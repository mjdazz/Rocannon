SYNOPSIS                   *pn_cpu_mgmt_class*

     • This module can we used to update mgmt port ingress policers.

PARAMETERS                   *pn_cpu_mgmt_class-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_burst_size                      ingress traffic burst size (bytes) or default.
   string
   pn_cliswitch                       Target switch to run the CLI on.
   string
                       Choices:
                       • arp
                       • icmp
                       • ssh
   pn_name             • snmp
   string              • fabric       mgmt port ingress traffic class.
                       • bcast
                       • nfs
                       • web
                       • web-ssl
                       • net-api
   pn_rate_limit                      ingress rate limit on mgmt port(bps) or unlimited.
   string
   state               Choices:       State the action to perform. Use update to modify cpu-mgmt-class.
   string / required   • update

EXAMPLES                   *pn_cpu_mgmt_class-examples*

 - name: cpu mgmt class modify ingress policers
   pn_cpu_mgmt_class:
     pn_cliswitch: "sw01"
     state: "update"
     pn_name: "icmp"
     pn_rate_limit: "10000"
     pn_burst_size: "14000"

 - name: cpu mgmt class modify ingress policers
   pn_cpu_mgmt_class:
     pn_cliswitch: "sw01"
     state: "update"
     pn_name: "snmp"
     pn_burst_size: "8000"
     pn_rate_limit: "100000"

 - name: cpu mgmt class modify ingress policers
   pn_cpu_mgmt_class:
     pn_cliswitch: "sw01"
     state: "update"
     pn_name: "web"
     pn_rate_limit: "10000"
     pn_burst_size: "1000"

   string
   stderr  on error set of error responses from the cpu-mgmt-class command.
   list
   stdout  always   set of responses from the cpu-mgmt-class command.
   list
