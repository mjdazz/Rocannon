SYNOPSIS                   *dpkg_selections*

     • Change dpkg package selection state via –get-selections and –set-selections.

PARAMETERS                   *dpkg_selections-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   name                          Name of the package
   - / required
                  Choices:
   selection      • install
   - / required   • hold         The selection state to set the package to.
                  • deinstall
                  • purge

NOTES                   *dpkg_selections-notes*

   Note

     • This module won’t cause any packages to be installed/removed/purged, use the apt module for that.

EXAMPLES                   *dpkg_selections-examples*

 # Prevent python from being upgraded.
 - dpkg_selections:
     name: python
     selection: hold
