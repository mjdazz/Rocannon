SYNOPSIS                   *urpmi*

     • Manages packages with urpmi (such as for Mageia or Mandriva)

PARAMETERS                   *urpmi-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   force           Choices:
   boolean         • no           Assume "yes" is the answer to any question urpmi has to ask. Corresponds to the --force option for urpmi.
                   • yes ←
   name                           A list of package names to install, upgrade or remove.
   - / required                   aliases: package, pkg
   added in 2.6
   no-recommends   Choices:       Corresponds to the --no-recommends option for urpmi.
   boolean         • no           aliases: no-recommends
                   • yes ←
   root          Default:         Specifies an alternative install root, relative to which all packages will be installed. Corresponds to the --root option for urpmi.
   -             "/"              aliases: installroot
   added in 2.4
   state           Choices:
   -               • absent       Indicates the desired package state.
                   • present ←
   update_cache    Choices:
   boolean         • no ←         Update the package database first urpmi.update -a.
                   • yes

EXAMPLES                   *urpmi-examples*

 - name: Install package foo
   urpmi:
     pkg: foo
     state: present

 - name: Remove package foo
   urpmi:
     pkg: foo
     state: absent

 - name: Remove packages foo and bar
   urpmi:
     pkg: foo,bar
     state: absent

 - name: Update the package database (urpmi.update -a -q) and install bar (bar will be the updated if a newer version exists)
 - urpmi:
     name: bar
     state: present
     update_cache: yes
