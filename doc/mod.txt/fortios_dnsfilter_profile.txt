SYNOPSIS                   *fortios_dnsfilter_profile*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify dnsfilter feature and profile category. Examples include all parameters and values
       need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_dnsfilter_profile-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_dnsfilter_profile-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure DNS domain filter profiles.
     fortios_dnsfilter_profile:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       dnsfilter_profile:
         block_action: "block"
         block_botnet: "disable"
         comment: "Comment."
         domain_filter:
             domain_filter_table: "7 (source dnsfilter.domain-filter.id)"
         external_ip_blocklist:
          -
             name: "default_name_9 (source system.external-resource.name)"
         ftgd_dns:
             filters:
              -
                 action: "block"
                 category: "13"
                 id:  "14"
                 log: "enable"
             options: "error-allow"
         log_all_domain: "enable"
         name: "default_name_18"
         redirect_portal: "<your_own_value>"
         safe_search: "disable"
         sdns_domain_log: "enable"
         sdns_ftgd_err_log: "enable"
         youtube_restrict: "strict"
