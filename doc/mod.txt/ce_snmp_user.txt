SYNOPSIS                   *ce_snmp_user*

     • Manages SNMP user configurations on CloudEngine switches.

PARAMETERS                   *ce_snmp_user-parameters*

      Parameter     Choices/Defaults                                                                             Comments
   aaa_local_user                    Unique name to identify the local user.
   -
   acl_number                        Access control list number.
   -
   auth_key                          The authentication password. Password length, 8-255 characters.
   -
                      Choices:
   auth_protocol      • noAuth       Authentication protocol.
   -                  • md5
                      • sha
   priv_key                          The encryption password. Password length 8-255 characters.
   -
                      Choices:
                      • noPriv
   priv_protocol      • des56
   -                  • 3des168      Encryption protocol.
                      • aes128
                      • aes192
                      • aes256
   remote_engine_id                  Remote engine id of the USM user.
   -
   user_group                        Name of the group where user belongs to.
   -
   usm_user_name                     Unique name to identify the USM user.
   -

NOTES                   *ce_snmp_user-notes*

   Note

     • This module requires the netconf system service be enabled on the remote device being managed.
     • Recommended connection is netconf.
     • This module also works with local connections for legacy playbooks.

EXAMPLES                   *ce_snmp_user-examples*

 - name: CloudEngine snmp user test
   hosts: cloudengine
   connection: local
   gather_facts: no
   vars:
     cli:
       host: "{{ inventory_hostname }}"
       port: "{{ ansible_ssh_port }}"
       username: "{{ username }}"
       password: "{{ password }}"
       transport: cli

   tasks:

   - name: "Config SNMP usm user"
     ce_snmp_user:
       state: present
       usm_user_name: wdz_snmp
       remote_engine_id: 800007DB03389222111200
       acl_number: 2000
       user_group: wdz_group
       provider: "{{ cli }}"

   - name: "Undo SNMP usm user"
     ce_snmp_user:
       state: absent
       usm_user_name: wdz_snmp
       remote_engine_id: 800007DB03389222111200
       acl_number: 2000
       user_group: wdz_group
       provider: "{{ cli }}"

   - name: "Config SNMP local user"
     ce_snmp_user:
       state: present
       aaa_local_user: wdz_user
       auth_protocol: md5
       auth_key: huawei123
       priv_protocol: des56
       priv_key: huawei123
       provider: "{{ cli }}"

   - name: "Config SNMP local user"
     ce_snmp_user:
       state: absent
       aaa_local_user: wdz_user
       auth_protocol: md5
       auth_key: huawei123
       priv_protocol: des56
       priv_key: huawei123
       provider: "{{ cli }}"
