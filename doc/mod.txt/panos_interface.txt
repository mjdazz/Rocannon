SYNOPSIS                   *panos_interface-synopsis*

     • Configure data-port (DP) network interface for DHCP. By default DP interfaces are static.

REQUIREMENTS                   *panos_interface-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python can be obtained from PyPI https://pypi.org/project/pan-python/

PARAMETERS                   *panos_interface-parameters*

        Parameter       Choices/Defaults                                                                           Comments
   commit                 Choices:
   boolean                • no           Commit if changed
                          • yes ←
   create_default_route   Choices:
   boolean                • no ←         Whether or not to add default route with router learned via DHCP.
                          • yes
   if_name                               Name of the interface to configure.
   - / required
   ip_address                            IP address (or hostname) of PAN-OS device.
   string / required
   password                              Password for authentication.
   string / required
   username             Default:         Username for authentication.
   string               "admin"
   zone_name                             Name of the zone for the interface. If the zone does not exist it is created but if the zone exists and it is not of the layer3 type the operation will
   - / required                          fail.

NOTES                   *panos_interface-notes*

   Note

     • Checkmode is not supported.

EXAMPLES                   *panos_interface-examples*

 - name: enable DHCP client on ethernet1/1 in zone public
   interface:
     password: "admin"
     ip_address: "192.168.1.1"
     if_name: "ethernet1/1"
     zone_name: "public"
     create_default_route: "yes"
