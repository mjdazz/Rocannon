SYNOPSIS                   *panos_admin-synopsis*

     • PanOS module that allows changes to the user account passwords by doing API calls to the Firewall using pan-api as the protocol.

REQUIREMENTS                   *panos_admin-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_admin-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   admin_password                     password for admin user
   - / required
   admin_username    Default:         username for admin user
   -                 "admin"
   commit              Choices:
   boolean             • no           commit if changed
                       • yes ←
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   role                               role for admin user
   -
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_admin-examples*

 # Set the password of user admin to "badpassword"
 # Doesn't commit the candidate config
   - name: set admin password
     panos_admin:
       ip_address: "192.168.1.1"
       password: "admin"
       admin_username: admin
       admin_password: "badpassword"
       commit: False
