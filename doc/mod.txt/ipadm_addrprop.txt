SYNOPSIS                   *ipadm_addrprop*

     • Modify IP address properties on Solaris/illumos systems.

PARAMETERS                   *ipadm_addrprop-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   addrobj                       Specifies the address object we want to manage.
   - / required                  aliases: nic, interface
   property                      Specifies the name of the address property we want to manage.
   - / required                  aliases: name
                  Choices:
   state          • present ←    Set or reset the property value.
   -              • absent
                  • reset
   temporary      Choices:
   boolean        • no ←         Specifies that the address property value is temporary. Temporary values do not persist across reboots.
                  • yes
   value                         Specifies the value we want to set for the address property.
   -

EXAMPLES                   *ipadm_addrprop-examples*

 - name: Mark address on addrobj as deprecated
   ipadm_addrprop: property=deprecated value=on addrobj=e1000g0/v6

 - name: Set network prefix length for addrobj
   ipadm_addrprop: addrobj=bge0/v4 name=prefixlen value=26

RETURN VALUES                   *ipadm_addrprop-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key           Returned                                                                                   Description
   addrobj                          address object name
   string    always                 Sample:
                                    bge0/v4
   property                         property name
   string    always                 Sample:
                                    deprecated
   state                            state of the target
   string    always                 Sample:
                                    present
   temporary                        specifies if operation will persist across reboots
   boolean   always                 Sample:
                                    True
   value                            property value
   string    when value is provided Sample:
                                    26
