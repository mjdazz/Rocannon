SYNOPSIS                   *vyos_interfaces*

     • This module manages the interface attributes on VyOS network devices.
     • This module supports managing base attributes of Ethernet, Bonding, VXLAN, Loopback and Virtual Tunnel Interfaces.

PARAMETERS                   *vyos_interfaces-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   config                              The provided interfaces configuration.
   list
    description                        Interface description.
    string
                        Choices:
    duplex              • full         Interface duplex mode.
    string              • half         Applicable for Ethernet interfaces only.
                        • auto
    enabled             Choices:       Administrative state of the interface.
    boolean             • no           Set the value to true to administratively enable the interface or false to disable it.
                        • yes ←
    mtu                                MTU for a specific interface. Refer to vendor documentation for valid values.
    integer                            Applicable for Ethernet, Bonding, VXLAN and Virtual Tunnel interfaces.
    name                               Full name of the interface, e.g. eth0, eth1, bond0, vti1, vxlan2.
    string / required
                        Choices:
                        • auto
    speed               • 10           Interface link speed.
    string              • 100          Applicable for Ethernet interfaces only.
                        • 1000
                        • 2500
                        • 10000
    vifs                               Virtual sub-interfaces related configuration.
    list                               802.1Q VLAN interfaces are represented as virtual sub-interfaces in VyOS.
        description                    Virtual sub-interface description.
        string
        enabled         Choices:       Administrative state of the virtual sub-interface.
        boolean         • no           Set the value to true to administratively enable the interface or false to disable it.
                        • yes ←
        mtu                            MTU for the virtual sub-interface.
        integer                        Refer to vendor documentation for valid values.
        vlan_id                        Identifier for the virtual sub-interface.
        integer
                        Choices:
   state                • merged ←
   string               • replaced     The state of the configuration after module completion.
                        • overridden
                        • deleted

NOTES                   *vyos_interfaces-notes*

   Note

     • Tested against VyOS 1.1.8 (helium).
     • This module works with connection network_cli. See the VyOS OS Platform Options.

   Common return values are documented here, the following are the fields unique to this module:

     Key      Returned                                                                                   Description
   after                 The configuration as structured data after module completion.
   list     when changed Sample:
                         The configuration returned will always be in the same format of the parameters above.
   before                The configuration as structured data prior to module invocation.
   list     always       Sample:
                         The configuration returned will always be in the same format of the parameters above.
   commands              The set of commands pushed to the remote device.
   list     always       Sample:
                         ['set interfaces ethernet eth1 mtu 1200', 'set interfaces ethernet eth2 vif 100 description VIF 100']
