SYNOPSIS                   *avi_useraccount*

     • This module can be used for updating the password of a user.
     • This module is useful for setting up admin password for Controller bootstrap.

REQUIREMENTS                   *avi_useraccount-requirements*

   The below requirements are needed on the host that executes this module.

     • avisdk

EXAMPLES                   *avi_useraccount-examples*

 - name: Update user password
   avi_useraccount:
     controller: ""
     username: ""
     password: new_password
     old_password: ""
     api_version: ""
     force_change: false

 - name: Update user password using avi_credentials
   avi_useraccount:
     avi_credentials: ""
     old_password: ""
     force_change: false
