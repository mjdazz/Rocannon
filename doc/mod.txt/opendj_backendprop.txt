SYNOPSIS                   *opendj_backendprop*

     • This module will update settings for OpenDJ with the command set-backend-prop.
     • It will check first via de get-backend-prop if configuration needs to be applied.

PARAMETERS                   *opendj_backendprop-parameters*

     Parameter      Choices/Defaults                                                                              Comments
   backend                              The name of the backend on which the property needs to be updated.
   - / required
   hostname                             The hostname of the OpenDJ server.
   - / required
   name                                 The configuration setting to update.
   - / required
   opendj_bindir Default:               The path to the bin directory of OpenDJ.
   -             "/opt/opendj/bin"
   password                             The password for the cn=Directory Manager user.
   -                                    Either password or passwordfile is needed.
   passwordfile                         Location to the password file which holds the password for the cn=Directory Manager user.
   -                                    Either password or passwordfile is needed.
   port                                 The Admin port on which the OpenDJ instance is available.
   - / required
   state         Default:               If configuration needs to be added/updated
   -             "present"
   username      Default:               The username to connect to.
   -             "cn=Directory Manager"
   value                                The value for the configuration item.
   - / required

EXAMPLES                   *opendj_backendprop-examples*

 - name: "Add or update OpenDJ backend properties"
   action: opendj_backendprop
           hostname=localhost
           port=4444
           username="cn=Directory Manager"
           password=password
           backend=userRoot
           name=index-entry-limit
           value=5000
