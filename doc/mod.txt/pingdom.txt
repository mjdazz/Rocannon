SYNOPSIS                   *pingdom*

     • This module will let you pause/unpause Pingdom alerts

REQUIREMENTS                   *pingdom-requirements*

   The below requirements are needed on the host that executes this module.

     • This pingdom python library: https://github.com/mbabineau/pingdom-python

PARAMETERS                   *pingdom-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   checkid                       Pingdom ID of the check.
   - / required
   key                           Pingdom API key.
   - / required
   passwd                        Pingdom user password.
   - / required
   state          Choices:
   - / required   • running      Define whether or not the check should be running or paused.
                  • paused
   uid                           Pingdom user ID.
   - / required

NOTES                   *pingdom-notes*

   Note

     • This module does not yet have support to add/remove checks.

EXAMPLES                   *pingdom-examples*

 # Pause the check with the ID of 12345.
 - pingdom:
     uid: [email protected]
     passwd: password123
     key: apipassword123
     checkid: 12345
     state: paused

 # Unpause the check with the ID of 12345.
 - pingdom:
     uid: [email protected]
     passwd: password123
     key: apipassword123
     checkid: 12345
     state: running
