SYNOPSIS                   *pn_vflow_table_profile*

     • This module can be used to modify a vFlow table profile.

PARAMETERS                   *pn_vflow_table_profile-parameters*

       Parameter     Choices/Defaults                                                                             Comments
   pn_cliswitch                        Target switch to run the CLI on.
   string
   pn_enable           Choices:
   boolean             • no            enable or disable vflow profile table.
                       • yes
                       Choices:
   pn_hw_tbl           • switch-main
   string              • switch-hash   hardware table used by vFlow.
                       • npu-main
                       • npu-hash
                       Choices:
   pn_profile          • application   type of vFlow profile.
   string              • ipv6
                       • qos
   state               Choices:        State the action to perform. Use update to modify the vflow-table-profile.
   string / required   • update

EXAMPLES                   *pn_vflow_table_profile-examples*

 - name: Modify vflow table profile
   pn_vflow_table_profile:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_profile: 'ipv6'
     pn_hw_tbl: 'switch-main'
     pn_enable: true

 - name: Modify vflow table profile
   pn_vflow_table_profile:
     state: 'update'
     pn_profile: 'qos'
     pn_hw_tbl: 'switch-main'
     pn_enable: false

RETURN VALUES                   *pn_vflow_table_profile-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the vflow-table-profile command.
   list
   stdout  always   set of responses from the vflow-table-profile command.
   list
