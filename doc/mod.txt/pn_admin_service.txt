SYNOPSIS                   *pn_admin_service*

     • This module is used to modify services on the server-switch.

PARAMETERS                   *pn_admin_service-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn__if              Choices:
   string              • mgmt         administrative service interface.
                       • data
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_icmp             Choices:
   boolean             • no           Internet Message Control Protocol (ICMP) to enable or disable.
                       • yes
   pn_net_api          Choices:
   boolean             • no           Netvisor API to enable or disable APIs.
                       • yes
   pn_nfs              Choices:
   boolean             • no           Network File System (NFS) to enable or disable.
                       • yes
   pn_snmp             Choices:
   boolean             • no           Simple Network Monitoring Protocol (SNMP) to enable or disable.
                       • yes
   pn_ssh              Choices:
   boolean             • no           Secure Shell to enable or disable.
                       • yes
   pn_web              Choices:
   boolean             • no           Web (HTTP) to enable or disable.
                       • yes
   pn_web_log          Choices:
   boolean             • no           Web logging to enable or disable.
                       • yes
   pn_web_port                        Web (HTTP) port to enable or disable.
   string
   pn_web_ssl          Choices:
   boolean             • no           Web SSL (HTTPS) to enable or disable.
                       • yes
   pn_web_ssl_port                    Web SSL (HTTPS) port to enable or disable.
   string
   state               Choices:       State the action to perform. Use update to modify the admin-service.
   string / required   • update

EXAMPLES                   *pn_admin_service-examples*

 - name: admin service functionality
   pn_admin_service:
     pn_cliswitch: "sw01"
     state: "update"
     pn__if: "mgmt"
     pn_web: False
     pn_icmp: True

 - name: admin service functionality
   pn_admin_service:
     pn_cliswitch: "sw01"
     state: "update"
     pn_web: False
     pn__if: "mgmt"
     pn_snmp: True
     pn_net_api: True
     pn_ssh: True

RETURN VALUES                   *pn_admin_service-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the admin-service command.
   list
   stdout  always   set of responses from the admin-service command.
   list
