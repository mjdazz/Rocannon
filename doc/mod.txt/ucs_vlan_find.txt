SYNOPSIS                   *ucs_vlan_find*

     • Find VLANs on Cisco UCS Manager based on different criteria.

REQUIREMENTS                   *ucs_vlan_find-requirements*

   The below requirements are needed on the host that executes this module.

     • ucsmsdk

PARAMETERS                   *ucs_vlan_find-parameters*

       Parameter     Choices/Defaults                                                                            Comments
                       Choices:       The fabric configuration of the VLAN. This can be one of the following:
   fabric              • common ←     common - The VLAN applies to both fabrics and uses the same configuration parameters in both cases.
   string              • A            A — The VLAN only applies to fabric A.
                       • B            B — The VLAN only applies to fabric B.
   hostname                           IP address or hostname of Cisco UCS Manager.
   string / required
   password                           Password for Cisco UCS Manager authentication.
   string / required
   pattern                            Regex pattern to find within the name property of the fabricVlan class.
   string                             This is required if vlanid parameter is not supplied.
   port                               Port number to be used during connection (by default uses 443 for https and 80 for http connection).
   integer
   proxy                              If use_proxy is no, specfies proxy to be used for connection. e.g. 'http://proxy.xy.z:8080'
   string
   use_proxy           Choices:
   boolean             • no           If no, will not use the proxy as defined by system environment variable.
                       • yes ←
   use_ssl             Choices:
   boolean             • no           If no, an HTTP connection will be used instead of the default HTTPS connection.
                       • yes ←
   username          Default:         Username for Cisco UCS Manager authentication.
   string            "admin"
   vlanid                             The unique string identifier assigned to the VLAN.
   string                             A VLAN ID can be between '1' and '3967', or between '4048' and '4093'.
                                      This is required if pattern parameter is not supplied.

EXAMPLES                   *ucs_vlan_find-examples*

 - name: Get all vlans in fabric A
   ucs_vlan_find:
     hostname: 172.16.143.150
     username: admin
     password: password
     fabric: 'A'
     pattern: '.'
 - name: Confirm if vlan 15 is present
   ucs_vlan_find:
     hostname: 172.16.143.150
     username: admin
     password: password
     vlanid: '15'

RETURN VALUES                   *ucs_vlan_find-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key     Returned                                                                                   Description
   vlan_list            basic details of vlans found
   list      on success Sample:
                        [{'id': '0', 'name': 'vlcloud1'}]
