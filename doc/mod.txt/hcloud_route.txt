SYNOPSIS                   *hcloud_route*

     • Create, update and delete cloud routes on the Hetzner Cloud.

REQUIREMENTS                   *hcloud_route-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0
     • hcloud-python >= 1.3.0

PARAMETERS                   *hcloud_route-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   destination                                      Destination network or host of this route.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   gateway                                          Gateway for the route.
   string / required
   network                                          The name of the Hetzner Cloud Network.
   string / required
   state               Choices:
   string              • absent                     State of the route.
                       • present ←

SEE ALSO                   *hcloud_route-see also*

   See also

   Documentation for Hetzner Cloud API
   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_route-examples*

 - name: Create a basic route
   hcloud_route:
     network: my-network
     destination: 10.100.1.0/24
     gateway: 10.0.1.1
     state: present

 - name: Ensure the route is absent
   hcloud_route:
     network: my-network
     destination: 10.100.1.0/24
     gateway: 10.0.1.1
     state: absent
