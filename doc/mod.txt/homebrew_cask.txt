SYNOPSIS                   *homebrew_cask*

     • Manages Homebrew casks.

REQUIREMENTS                   *homebrew_cask-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6

PARAMETERS                   *homebrew_cask-parameters*

        Parameter       Choices/Defaults                                                                           Comments
   accept_external_apps   Choices:
   boolean                • no ←         allow external apps
   added in 2.5.0         • yes
   greedy                 Choices:
   boolean                • no ←         upgrade casks that auto update; passes --greedy to brew cask outdated when checking if an installed cask has a newer version available
   added in 2.7.0         • yes
   install_options                       options flags to install a package
   -                                     aliases: options
   name                                  name of cask to install/remove
   - / required                          aliases: pkg, package, cask
   path                 Default:         ':' separated list of paths to search for 'brew' executable.
   -                    "/usr/local/bin"
                          Choices:
   state                  • present ←    state of the cask
   -                      • absent
                          • upgraded
   sudo_password
   -                                     The sudo password to be passed to SUDO_ASKPASS.
   added in 2.8
   update_homebrew        Choices:       update homebrew itself first. Note that brew cask update is a synonym for brew update.
   boolean                • no ←         aliases: update-brew
                          • yes
   upgrade                Choices:
   boolean                • no ←         upgrade all casks (mutually exclusive with `upgrade_all`)
   added in 2.5.0         • yes
   upgrade_all            Choices:
   boolean                • no ←         upgrade all casks (mutually exclusive with `upgrade`)
   added in 2.5.0         • yes

EXAMPLES                   *homebrew_cask-examples*

 - homebrew_cask:
     name: alfred
     state: present

 - homebrew_cask:
     name: alfred
     state: absent

 - homebrew_cask:
     name: alfred
     state: present
     install_options: 'appdir=/Applications'

 - homebrew_cask:
     name: alfred
     state: present
     install_options: 'debug,appdir=/Applications'

 - homebrew_cask:
     name: alfred
     state: present
     accept_external_apps: True

 - homebrew_cask:
     name: alfred
     state: absent
     install_options: force

 - homebrew_cask:
     upgrade_all: true

 - homebrew_cask:
     name: alfred
     state: upgraded
     install_options: force

 - homebrew_cask:
     name: 1password
     state: upgraded
     greedy: True

 - homebrew_cask:
     name: wireshark
     state: present
     sudo_password: "{{ ansible_become_pass }}"
