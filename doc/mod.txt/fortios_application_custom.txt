SYNOPSIS                   *fortios_application_custom*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify application feature and custom category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_application_custom-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_application_custom-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure custom application signatures.
     fortios_application_custom:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       application_custom:
         behavior: "<your_own_value>"
         category: "4"
         comment: "Comment."
         id:  "6"
         name: "default_name_7"
         protocol: "<your_own_value>"
         signature: "<your_own_value>"
         tag: "<your_own_value>"
         technology: "<your_own_value>"
         vendor: "<your_own_value>"
