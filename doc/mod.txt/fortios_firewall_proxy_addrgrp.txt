SYNOPSIS                   *fortios_firewall_proxy_addrgrp*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall feature and proxy_addrgrp category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_proxy_addrgrp-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_proxy_addrgrp-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Web proxy address group configuration.
     fortios_firewall_proxy_addrgrp:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_proxy_addrgrp:
         color: "3"
         comment: "Optional comments."
         member:
          -
             name: "default_name_6 (source firewall.proxy-address.name firewall.proxy-addrgrp.name)"
         name: "default_name_7"
         tagging:
          -
             category: "<your_own_value> (source system.object-tagging.category)"
             name: "default_name_10"
             tags:
              -
                 name: "default_name_12 (source system.object-tagging.tags.name)"
         type: "src"
         uuid: "<your_own_value>"
         visibility: "enable"
