SYNOPSIS                   *win_hostname*

     • Manages local Windows computer name.
     • A reboot is required for the computer name to take effect.

PARAMETERS                   *win_hostname-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   name                               The hostname to set for the computer.
   string / required

SEE ALSO                   *win_hostname-see also*

   See also

   win_dns_client – Configures DNS lookup on Windows hosts
   The official documentation on the win_dns_client module.

EXAMPLES                   *win_hostname-examples*

 - name: Change the hostname to sample-hostname
   win_hostname:
     name: sample-hostname
   register: res

 - name: Reboot
   win_reboot:
   when: res.reboot_required

RETURN VALUES                   *win_hostname-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key       Returned                                                                                Description
   old_name                 The original hostname that was set before it was changed.
   string          always   Sample:
                            old_hostname
   reboot_required          Whether a reboot is required to complete the hostname change.
   boolean         always   Sample:
                            True
