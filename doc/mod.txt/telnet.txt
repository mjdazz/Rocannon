SYNOPSIS                   *telnet*

     • Executes a low-down and dirty telnet command, not going through the module subsystem.
     • This is mostly to be used for enabling ssh on devices that only have telnet enabled by default.

PARAMETERS                   *telnet-parameters*

      Parameter    Choices/Defaults                                                                             Comments
   command                          List of commands to be executed in the telnet session.
   - / required                     aliases: commands
   host            Default:         The host/target on which to execute the command
   -               "remote_addr"
   login_prompt    Default:         Login or username prompt to expect
   -               "login: "
   password                         The password for login
   -
   password_prompt Default:         Login or username prompt to expect
   -               "Password: "
   pause           Default:         Seconds to pause between each command issued
   -               1
   port            Default:         Remote port to use
   -               23
   prompts         Default:         List of prompts expected before sending next command
   -               ["$"]
   send_newline      Choices:
   boolean           • no ←         Sends a newline character upon successful connection to start the terminal session.
   added in 2.7      • yes
   timeout         Default:         timeout for remote operations
   -               120
   user            Default:         The user for login
   -               "remote_user"

NOTES                   *telnet-notes*

   Note

     • The environment keyword does not work with this task

EXAMPLES                   *telnet-examples*

 - name: send configuration commands to IOS
   telnet:
     user: cisco
     password: cisco
     login_prompt: "Username: "
     prompts:
       - "[>#]"
     command:
       - terminal length 0
       - configure terminal
       - hostname ios01

 - name: run show commands
   telnet:
     user: cisco
     password: cisco
     login_prompt: "Username: "
     prompts:
       - "[>#]"
     command:
       - terminal length 0
       - show version
