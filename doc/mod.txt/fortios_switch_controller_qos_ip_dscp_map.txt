SYNOPSIS                   *fortios_switch_controller_qos_ip_dscp_map*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify switch_controller_qos feature and ip_dscp_map category. Examples include all
       parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_switch_controller_qos_ip_dscp_map-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_switch_controller_qos_ip_dscp_map-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure FortiSwitch QoS IP precedence/DSCP.
     fortios_switch_controller_qos_ip_dscp_map:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       switch_controller_qos_ip_dscp_map:
         description: "<your_own_value>"
         map:
          -
             cos_queue: "5"
             diffserv: "CS0"
             ip_precedence: "network-control"
             name: "default_name_8"
             value: "<your_own_value>"
         name: "default_name_10"
