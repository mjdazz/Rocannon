SYNOPSIS                   *fortios_webfilter_content_header*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify webfilter feature and content_header category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_webfilter_content_header-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_webfilter_content_header-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure content types used by Web filter.
     fortios_webfilter_content_header:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       webfilter_content_header:
         comment: "Optional comments."
         entries:
          -
             action: "block"
             category: "<your_own_value>"
             pattern: "<your_own_value>"
         id:  "8"
         name: "default_name_9"
