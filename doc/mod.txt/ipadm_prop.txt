SYNOPSIS                   *ipadm_prop*

     • Modify protocol properties on Solaris/illumos systems.

PARAMETERS                   *ipadm_prop-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   property                      Specifies the name of property we want to manage.
   - / required
   protocol                      Specifies the protocol for which we want to manage properties.
   - / required
                  Choices:
   state          • present ←    Set or reset the property value.
   -              • absent
                  • reset
   temporary      Choices:
   boolean        • no ←         Specifies that the property value is temporary. Temporary property values do not persist across reboots.
                  • yes
   value                         Specifies the value we want to set for the property.
   -

EXAMPLES                   *ipadm_prop-examples*

 # Set TCP receive buffer size
 - ipadm_prop: protocol=tcp property=recv_buf value=65536

 # Reset UDP send buffer size to the default value
 - ipadm_prop: protocol=udp property=send_buf state=reset

RETURN VALUES                   *ipadm_prop-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key    Returned                                                                                   Description
   property           name of the property
   string    always   Sample:
                      recv_maxbuf
   protocol           property's protocol
   string    always   Sample:
                      TCP
   state              state of the target
   string    always   Sample:
                      present
   temporary          property's persistence
   boolean   always   Sample:
                      True
   value              value of the property. May be int or string depending on property.
   integer   always   Sample:
                      '1024' or 'never'
