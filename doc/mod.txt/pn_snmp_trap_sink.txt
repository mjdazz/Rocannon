SYNOPSIS                   *pn_snmp_trap_sink*

     • This module can be used to create a SNMP trap sink and delete a SNMP trap sink.

PARAMETERS                   *pn_snmp_trap_sink-parameters*

       Parameter          Choices/Defaults                                                                            Comments
   pn_cliswitch                                 Target switch to run the CLI on.
   string
   pn_community                                 community type.
   string
   pn_dest_host                                 destination host.
   string
   pn_dest_port      Default:                   destination port.
   string            "162"
                       Choices:
   pn_type             • TRAP_TYPE_V1_TRAP      trap type.
   string              • TRAP_TYPE_V2C_TRAP ←
                       • TRAP_TYPE_V2_INFORM
   state               Choices:
   string / required   • present                State the action to perform. Use present to create snmp-trap-sink and absent to delete snmp-trap-sink.
                       • absent

EXAMPLES                   *pn_snmp_trap_sink-examples*

 - name: snmp trap sink functionality
   pn_snmp_trap_sink:
     pn_cliswitch: "sw01"
     state: "present"
     pn_community: "foo"
     pn_type: "TRAP_TYPE_V2_INFORM"
     pn_dest_host: "192.168.67.8"

 - name: snmp trap sink functionality
   pn_snmp_trap_sink:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_community: "foo"
     pn_type: "TRAP_TYPE_V2_INFORM"
     pn_dest_host: "192.168.67.8"

RETURN VALUES                   *pn_snmp_trap_sink-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the snmp-trap-sink command.
   list
   stdout  always   set of responses from the snmp-trap-sink command.
   list
