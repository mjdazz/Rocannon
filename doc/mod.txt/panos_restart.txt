SYNOPSIS                   *panos_restart-synopsis*

     • Restart a device

REQUIREMENTS                   *panos_restart-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_restart-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_restart-examples*

 - panos_restart:
     ip_address: "192.168.1.1"
     username: "admin"
     password: "admin"
