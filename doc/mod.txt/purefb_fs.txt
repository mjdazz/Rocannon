SYNOPSIS                   *purefb_fs*

     • This module manages filesystems on Pure Storage FlashBlade.

REQUIREMENTS                   *purefb_fs-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purity_fb >= 1.1

PARAMETERS                   *purefb_fs-parameters*

       Parameter        Choices/Defaults                                                                            Comments
   api_token                                FlashBlade API token for admin privileged user.
   string
   eradicate           Choices:
   boolean             • no ←               Define whether to eradicate the filesystem on delete or leave in trash.
                       • yes
   fastremove          Choices:
   boolean             • no ←               Define whether the fast remove directory is enabled for the filesystem.
                       • yes
   fb_url                                   FlashBlade management IP address or Hostname.
   string
   group_quota
   string                                   Default quota in M, G, T or P units for a group under this file system.
   added in 2.9
   hard_limit          Choices:             Define whether the capacity for a filesystem is a hard limit.
   boolean             • no ←               CAUTION This will cause the filesystem to go Read-Only if the capacity has already exceeded the logical size of the filesystem.
   added in 2.8        • yes
   http                Choices:
   boolean             • no ←               Define whether to HTTP/HTTPS protocol is enabled for the filesystem.
                       • yes
   name                                     Filesystem Name.
   string / required
   nfs                 Choices:             (Deprecate) Define whether to NFSv3 protocol is enabled for the filesystem.
   boolean             • no                 This option will be deprecated in 2.10, use nfsv3 instead.
                       • yes ←
   nfs_rules         Default:               Define the NFS rules in operation.

   boolean             • no                 Define whether to NFSv3 protocol is enabled for the filesystem.

   nfsv4               Choices:
   boolean             • no                 Define whether to NFSv4.1 protocol is enabled for the filesystem.
   added in 2.9        • yes ←
   size              Default:               Volume size in M, G, T or P units. See examples.
   string            "32G"
   smb                 Choices:
   boolean             • no ←               Define whether to SMB protocol is enabled for the filesystem.

   boolean             • no ←               Define whether a snapshot directory is enabled for the filesystem.
                       • yes
   state               Choices:
   string              • present ←          Create, delete or modifies a filesystem.
                       • absent
   user_quota

NOTES                   *purefb_fs-notes*

   Note

     • This module requires the purity_fb Python library
     • You must set PUREFB_URL and PUREFB_API environment variables if fb_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefb_fs-examples*

 - name: Create new filesystem named foo
   purefb_fs:
     name: foo
     size: 1T
     state: present
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Delete filesystem named foo
   purefb_fs:
     name: foo
     state: absent
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Recover filesystem named foo
   purefb_fs:
     name: foo
     state: present
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Eradicate filesystem named foo
   purefb_fs:
     name: foo
     state: absent
     eradicate: true
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641

 - name: Modify attributes of an existing filesystem named foo
   purefb_fs:
     name: foo
     size: 2T
     nfsv3 : false
     nfsv4 : true
     user_quota: 10K
     group_quota: 25M
     nfs_rules: '"(ro)'
     snapshot: true
     fastremove: true
     hard_limit: true
     smb: true
     state: present
     fb_url: 10.10.10.2
     api_token: T-55a68eb5-c785-4720-a2ca-8b03903bf641
