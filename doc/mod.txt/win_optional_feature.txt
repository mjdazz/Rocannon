SYNOPSIS                   *win_optional_feature*

     • Install or uninstall optional Windows features on non-Server Windows.
     • This module uses the Enable-WindowsOptionalFeature and Disable-WindowsOptionalFeature cmdlets.

PARAMETERS                   *win_optional_feature-parameters*

      Parameter    Choices/Defaults                                                                             Comments
   include_parent    Choices:
   boolean           • no ←         Whether to enable the parent feature and the parent's dependencies.
                     • yes
   name                             The name(s) of the feature to install.
   list / required                  This relates to FeatureName in the Powershell cmdlet.
                                    To list all available features use the PowerShell command Get-WindowsOptionalFeature.
   source                           Specify a source to install the feature from.
   string                           Can either be {driveletter}:\sources\sxs or \\{IP}\share\sources\sxs.
   state             Choices:
   string            • absent       Whether to ensure the feature is absent or present on the system.
                     • present ←

EXAMPLES                   *win_optional_feature-examples*

 - name: Install .Net 3.5
   win_optional_feature:
     name: NetFx3
     state: present

 - name: Install .Net 3.5 from source
   win_optional_feature:
     name: NetFx3
     source: \\share01\win10\sources\sxs
     state: present

 - name: Install Microsoft Subsystem for Linux
   win_optional_feature:
     name: Microsoft-Windows-Subsystem-Linux
     state: present
   register: wsl_status

 - name: Reboot if installing Linux Subsytem as feature requires it
   win_reboot:
   when: wsl_status.reboot_required

 - name: Install multiple features in one task
   win_optional_feature:
     name:
     - NetFx3
     - Microsoft-Windows-Subsystem-Linux
     state: present
