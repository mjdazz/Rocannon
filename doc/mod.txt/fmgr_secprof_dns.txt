SYNOPSIS                   *fmgr_secprof_dns*

     • Manage DNS security profiles in FortiManager

PARAMETERS                   *fmgr_secprof_dns-parameters*

               Parameter              Choices/Defaults                                                                    Comments
   adom                              Default:           The ADOM the configuration should belong to.
   -                                 "root"
   block_action                        Choices:         Action to take for blocked domains.
   string                              • block          choice | block | Return NXDOMAIN for blocked domains.
                                       • redirect       choice | redirect | Redirect blocked domains to SDNS portal.
   block_botnet                        Choices:         Enable/disable blocking botnet C&C; DNS lookups.
   string                              • disable        choice | disable | Disable blocking botnet C&C; DNS lookups.
                                       • enable         choice | enable | Enable blocking botnet C&C; DNS lookups.
   comment                                              Comment for the security profile to show in the FortiManager GUI.
   string
   domain_filter_domain_filter_table                    DNS domain filter table ID.
   string
   external_ip_blocklist                                One or more external IP block lists.
   string
   ftgd_dns_filters_action             Choices:         Action to take for DNS requests matching the category.
   string                              • monitor        choice | monitor | Allow DNS requests matching the category and log the result.
                                       • block          choice | block | Block DNS requests matching the category.
   ftgd_dns_filters_category                            Category number.
   string
   ftgd_dns_filters_log                Choices:         Enable/disable DNS filter logging for this DNS profile.
   string                              • disable        choice | disable | Disable DNS filter logging.
                                       • enable         choice | enable | Enable DNS filter logging.
                                       Choices:         FortiGuard DNS filter options.
   ftgd_dns_options                    • error-allow    FLAG Based Options. Specify multiple in list form.
   string                              • ftgd-disable   flag | error-allow | Allow all domains when FortiGuard DNS servers fail.
                                                        flag | ftgd-disable | Disable FortiGuard DNS domain rating.
   log_all_domain                      Choices:         Enable/disable logging of all domains visited (detailed DNS logging).
   string                              • disable        choice | disable | Disable logging of all domains visited.
                                       • enable         choice | enable | Enable logging of all domains visited.
                                       Choices:
   mode                                • add ←          Sets one of three modes for managing the object.
   -                                   • set            Allows use of soft-adds instead of overwriting existing values.
                                       • delete
                                       • update
   name                                                 Profile name.
   string
   redirect_portal                                      IP address of the SDNS redirect portal.
   string
   safe_search                         Choices:         Enable/disable Google, Bing, and YouTube safe search.
   string                              • disable        choice | disable | Disable Google, Bing, and YouTube safe search.
                                       • enable         choice | enable | Enable Google, Bing, and YouTube safe search.
   sdns_domain_log                     Choices:         Enable/disable domain filtering and botnet domain logging.
   string                              • disable        choice | disable | Disable domain filtering and botnet domain logging.
                                       • enable         choice | enable | Enable domain filtering and botnet domain logging.
   sdns_ftgd_err_log                   Choices:         Enable/disable FortiGuard SDNS rating error logging.
   string                              • disable        choice | disable | Disable FortiGuard SDNS rating error logging.
                                       • enable         choice | enable | Enable FortiGuard SDNS rating error logging.
   youtube_restrict                    Choices:         Set safe search for YouTube restriction level.
   string                              • strict         choice | strict | Enable strict safe seach for YouTube.
                                       • moderate       choice | moderate | Enable moderate safe search for YouTube.

EXAMPLES                   *fmgr_secprof_dns-examples*

 - name: DELETE Profile
   fmgr_secprof_dns:
     name: "Ansible_DNS_Profile"
     comment: "Created by Ansible Module TEST"
     mode: "delete"

 - name: CREATE Profile
   fmgr_secprof_dns:
     name: "Ansible_DNS_Profile"
     comment: "Created by Ansible Module TEST"
     mode: "set"
     block_action: "block"
