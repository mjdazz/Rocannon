SYNOPSIS                   *cp_mgmt_run_script*

     • Executes the script on a given list of targets.
     • All operations are performed over Web Services API.

PARAMETERS                   *cp_mgmt_run_script-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   args                           Script arguments.
   string
   comments                       Comments string.
   string
   script                         Script body.
   string
   script_name                    Script name.
   string
   targets                        On what targets to execute this command. Targets may be identified by their name, or object unique identifier.
   list
   version                        Version of checkpoint. If not given one, the latest version taken.
   string
   wait_for_task   Choices:
   boolean         • no           Wait for the task to end. Such as publish task.
                   • yes ←

EXAMPLES                   *cp_mgmt_run_script-examples*

 - name: run-script
   cp_mgmt_run_script:
     script: ls -l /
     script_name: 'Script Example: List files under / dir'
     targets:
     - corporate-gateway

RETURN VALUES                   *cp_mgmt_run_script-return values*
