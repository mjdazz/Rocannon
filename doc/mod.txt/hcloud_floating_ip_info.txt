SYNOPSIS                   *hcloud_floating_ip_info*

     • Gather facts about your Hetzner Cloud Floating IPs.
     • This module was called hcloud_floating_ip_facts before Ansible 2.9, returning ansible_facts and hcloud_floating_ip_facts. Note that the hcloud_floating_ip_info module no longer returns
       ansible_facts and the value was renamed to hcloud_floating_ip_info!

   Aliases: hcloud_floating_ip_facts

REQUIREMENTS                   *hcloud_floating_ip_info-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_floating_ip_info-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the Floating IP you want to get.
   integer
   label_selector                                   The label selector for the Floating IP you want to get.
   string

SEE ALSO                   *hcloud_floating_ip_info-see also*

   See also

   Documentation for Hetzner Cloud API
   Complete reference for the Hetzner Cloud API.

EXAMPLES                   *hcloud_floating_ip_info-examples*

 - name: Gather hcloud Floating ip infos
   hcloud_floating_ip_info:
   register: output
 - name: Print the gathered infos
   debug:
     var: output

RETURN VALUES                   *hcloud_floating_ip_info-return values*

   Common return values are documented here, the following are the fields unique to this module:

             Key           Returned                                                                            Description
   hcloud_floating_ip_info always   The Floating ip infos as list
   complex
         description                Description of the Floating IP
         string            always   Sample:
                                    Falkenstein DC 8
         home_location              Location the Floating IP was created in
         string            always   Sample:
                                    fsn1
         id                         Numeric identifier of the Floating IP
         integer           always   Sample:
                                    1937415
         ip                         IP address of the Floating IP
         string            always   Sample:
                                    131.232.99.1
         labels            always   User-defined labels (key-value pairs)
         dictionary
         server                     Name of the server where the Floating IP is assigned to.
         string            always   Sample:
                                    my-server
         type                       Type of the Floating IP
         string            always   Sample:
                                    ipv4
