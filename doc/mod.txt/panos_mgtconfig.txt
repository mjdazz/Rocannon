DEPRECATED                   *panos_mgtconfig*

                                      Removed in Ansible:
                  version: 2.12
        Why:      Consolidating code base.
    Alternative:  Use https://galaxy.ansible.com/PaloAltoNetworks/paloaltonetworks instead.

SYNOPSIS                   *panos_mgtconfig-synopsis*

     • Configure management settings of device

REQUIREMENTS                   *panos_mgtconfig-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_mgtconfig-parameters*

        Parameter       Choices/Defaults                                                                           Comments
   commit                 Choices:
   boolean                • no           commit if changed
                          • yes ←
   dns_server_primary                    address of primary DNS server
   -
   dns_server_secondary                  address of secondary DNS server
   -
   ip_address                            IP address (or hostname) of PAN-OS device.
   string / required
   panorama_primary                      address of primary Panorama server
   -
   panorama_secondary                    address of secondary Panorama server
   -
   password                              Password for authentication.
   string / required
   username             Default:         Username for authentication.
   string               "admin"

EXAMPLES                   *panos_mgtconfig-examples*

 - name: set dns and panorama
   panos_mgtconfig:
     ip_address: "192.168.1.1"
     password: "admin"
     dns_server_primary: "1.1.1.1"
     dns_server_secondary: "1.1.1.2"
     panorama_primary: "1.1.1.3"
     panorama_secondary: "1.1.1.4"
