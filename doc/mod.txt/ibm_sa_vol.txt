SYNOPSIS                   *ibm_sa_vol*

     • This module creates or deletes volumes to be used on IBM Spectrum Accelerate Family storage systems.

REQUIREMENTS                   *ibm_sa_vol-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • pyxcli

PARAMETERS                   *ibm_sa_vol-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   endpoints                          The hostname or management IP of Spectrum Accelerate storage system.
   string / required
   password                           Password for username on the spectrum accelerate storage system.
   string / required
   pool                               Volume pool.
   -
   size                               Volume size.
   -
   state               Choices:
   - / required        • present ←    Volume state.
                       • absent
   username                           Management user on the spectrum accelerate storage system.
   string / required
   vol                                Volume name.
   - / required

NOTES                   *ibm_sa_vol-notes*

   Note

     • This module requires pyxcli python library. Use ‘pip install pyxcli’ in order to get pyxcli.

EXAMPLES                   *ibm_sa_vol-examples*

 - name: Create a new volume.
   ibm_sa_vol:
     vol: volume_name
     pool: pool_name
     size: 17
     state: present
     username: admin
     password: secret
     endpoints: hostdev-system

 - name: Delete an existing volume.
   ibm_sa_vol:
     vol: volume_name
     state: absent
     username: admin
     password: secret
     endpoints: hostdev-system
