SYNOPSIS                   *emc_vnx_sg_member*

     • This module manages the members of an existing storage group.

REQUIREMENTS                   *emc_vnx_sg_member-requirements*

   The below requirements are needed on the host that executes this module.

     • An EMC VNX Storage device.
     • Ansible 2.7.
     • storops (0.5.10 or greater). Install using ‘pip install storops’.

PARAMETERS                   *emc_vnx_sg_member-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   lunid                              Lun id to be added.
   - / required
   name                               Name of the Storage group to manage.
   - / required
   sp_address                         Address of the SP of target/secondary storage.
   string / required
   sp_password       Default:         password for accessing SP.
   string            "sysadmin"
   sp_user           Default:         Username for accessing SP.
   string            "sysadmin"
   state               Choices:       Indicates the desired lunid state.
   -                   • present ←    present ensures specified lunid is present in the Storage Group.
                       • absent       absent ensures specified lunid is absent from Storage Group.

NOTES                   *emc_vnx_sg_member-notes*

   Note

     • The modules prefixed with emc_vnx are built to support the EMC VNX storage platform.

EXAMPLES                   *emc_vnx_sg_member-examples*

 - name: Add lun to storage group
   emc_vnx_sg_member:
     name: sg01
     sp_address: sp1a.fqdn
     sp_user: sysadmin
     sp_password: sysadmin
     lunid: 100
     state: present

 - name: Remove lun from storage group
   emc_vnx_sg_member:
     name: sg01
     sp_address: sp1a.fqdn
     sp_user: sysadmin
     sp_password: sysadmin
     lunid: 100
     state: absent

RETURN VALUES                   *emc_vnx_sg_member-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   hluid   success  LUNID that hosts attached to the storage group will see.
   integer
