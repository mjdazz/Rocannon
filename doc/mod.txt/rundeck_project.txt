SYNOPSIS                   *rundeck_project*

     • Create and remove Rundeck projects through HTTP API.

PARAMETERS                   *rundeck_project-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   api_version  Default:         Sets the API version used by module.
   -            14               API version must be at least 14.
   name                          Sets the project name.
   - / required
   state          Choices:
   -              • present ←    Create or remove Rundeck project.
                  • absent
   token                         Sets the token to authenticate against Rundeck API.
   - / required
   url                           Sets the rundeck instance URL.
   - / required

EXAMPLES                   *rundeck_project-examples*

 - name: Create a rundeck project
   rundeck_project:
     name: "Project_01"
     api_version: 18
     url: "https://rundeck.example.org"
     token: "mytoken"
     state: present

 - name: Remove a rundeck project
   rundeck_project:
     name: "Project_02"
     url: "https://rundeck.example.org"
     token: "mytoken"
     state: absent

RETURN VALUES                   *rundeck_project-return values*

   Common return values are documented here, the following are the fields unique to this module:

         Key        Returned                                                                               Description
   after            success  dictionary containing project information after modification
   dictionary
   before           success  dictionary containing project information before modification
   dictionary
   rundeck_response failed   Rundeck response when a failure occurs
   string
