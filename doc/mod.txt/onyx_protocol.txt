SYNOPSIS                   *onyx_protocol*

     • This module provides a mechanism for enabling and disabling protocols Mellanox on ONYX network devices.

PARAMETERS                   *onyx_protocol-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   bgp             Choices:
   -               • enabled      BGP protocol
                   • disabled
   dcb_pfc         Choices:
   -               • enabled      DCB priority flow control
                   • disabled
   igmp_snooping   Choices:
   -               • enabled      IP IGMP snooping
                   • disabled
   ip_l3           Choices:
   -               • enabled      IP L3 support
                   • disabled
   ip_routing      Choices:
   -               • enabled      IP routing support
                   • disabled
   lacp            Choices:
   -               • enabled      LACP protocol
                   • disabled
   lldp            Choices:
   -               • enabled      LLDP protocol
                   • disabled
   magp            Choices:
   -               • enabled      MAGP protocol
                   • disabled
   mlag            Choices:
   -               • enabled      MLAG protocol
                   • disabled
   nve             Choices:
   -               • enabled      nve protocol
   added in 2.9    • disabled
   ospf            Choices:
   -               • enabled      OSPF protocol
                   • disabled
   spanning_tree   Choices:
   -               • enabled      Spanning Tree support
                   • disabled

NOTES                   *onyx_protocol-notes*

   Note

     • Tested on ONYX 3.6.4000

EXAMPLES                   *onyx_protocol-examples*

 - name: enable protocols for MLAG
   onyx_protocol:
     lacp: enabled
     spanning_tree: disabled
     ip_routing: enabled
     mlag: enabled
     dcb_pfc: enabled

RETURN VALUES                   *onyx_protocol-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key    Returned                                                                                   Description
   commands          The list of configuration mode commands to send to the device.
   list     always   Sample:
                     ['no spanning-tree', 'protocol mlag']
