SYNOPSIS                   *fortios_firewall_central_snat_map*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall feature and central_snat_map category. Examples include all parameters
       and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_firewall_central_snat_map-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_firewall_central_snat_map-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure central SNAT policies.
     fortios_firewall_central_snat_map:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       firewall_central_snat_map:
         comments: "<your_own_value>"
         dst_addr:
          -
             name: "default_name_5 (source firewall.address.name firewall.addrgrp.name)"
         dstintf:
          -
             name: "default_name_7 (source system.interface.name system.zone.name)"
         nat: "disable"
         nat_ippool:
          -
             name: "default_name_10 (source firewall.ippool.name)"
         nat_port: "<your_own_value>"
         orig_addr:
          -
             name: "default_name_13 (source firewall.address.name firewall.addrgrp.name)"
         orig_port: "<your_own_value>"
         policyid: "15"
         protocol: "16"
         srcintf:
          -
             name: "default_name_18 (source system.interface.name system.zone.name)"
         status: "enable"
