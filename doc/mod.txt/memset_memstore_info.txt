SYNOPSIS                   *memset_memstore_info*

     • Retrieve Memstore product usage information.
     • This module was called memset_memstore_facts before Ansible 2.9. The usage did not change.

   Aliases: memset_memstore_facts

PARAMETERS                   *memset_memstore_info-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   api_key                       The API key obtained from the Memset control panel.
   - / required
   name                          The Memstore product name (i.e. mstestyaa1).
   - / required

NOTES                   *memset_memstore_info-notes*

   Note

     • An API key generated via the Memset customer control panel is needed with the following minimum scope - memstore.usage.

EXAMPLES                   *memset_memstore_info-examples*

 - name: get usage for mstestyaa1
   memset_memstore_info:
     name: mstestyaa1
     api_key: 5eb86c9896ab03919abcf03857163741
   delegate_to: localhost
