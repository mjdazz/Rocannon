SYNOPSIS                   *fortios_webfilter_ftgd_local_rating*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify webfilter feature and ftgd_local_rating category. Examples include all parameters
       and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_webfilter_ftgd_local_rating-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_webfilter_ftgd_local_rating-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure local FortiGuard Web Filter local ratings.
     fortios_webfilter_ftgd_local_rating:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       webfilter_ftgd_local_rating:
         rating: "<your_own_value>"
         status: "enable"
         url: "myurl.com"
