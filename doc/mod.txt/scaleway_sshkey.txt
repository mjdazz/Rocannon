SYNOPSIS                   *scaleway_sshkey*

     • This module manages SSH keys on Scaleway account https://developer.scaleway.com

PARAMETERS                   *scaleway_sshkey-parameters*

      Parameter            Choices/Defaults                                                                             Comments
   api_timeout      Default:                       HTTP timeout to Scaleway API in seconds.
   integer          30                             aliases: timeout
   api_token                                       Scaleway OAuth token.
   string                                          aliases: oauth_token
   api_url          Default:                       Scaleway API URL
   string           "https://account.scaleway.com" aliases: base_url
   query_parameters Default:                       List of parameters passed to the query string.
   dictionary       {}
   ssh_pub_key                                     The public SSH key as a string to add.
   - / required
   state              Choices:
   -                  • present ←                  Indicate desired state of the SSH key.
                      • absent
   validate_certs     Choices:
   boolean            • no                         Validate SSL certs of the Scaleway API.
                      • yes ←

NOTES                   *scaleway_sshkey-notes*

   Note

     • Also see the API documentation on https://developer.scaleway.com/
     • If api_token is not set within the module, the following environment variables can be used in decreasing order of precedence SCW_TOKEN, SCW_API_KEY, SCW_OAUTH_TOKEN or SCW_API_TOKEN.
     • If one wants to use a different api_url one can also set the SCW_API_URL environment variable.

EXAMPLES                   *scaleway_sshkey-examples*

 - name: "Add SSH key"
   scaleway_sshkey:
     ssh_pub_key: "ssh-rsa AAAA..."
     state: "present"

 - name: "Delete SSH key"
   scaleway_sshkey:
     ssh_pub_key: "ssh-rsa AAAA..."
     state: "absent"

 - name: "Add SSH key with explicit token"
   scaleway_sshkey:
     ssh_pub_key: "ssh-rsa AAAA..."
     state: "present"
     oauth_token: "6ecd2c9b-6f4f-44d4-a187-61a92078d08c"

RETURN VALUES                   *scaleway_sshkey-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key          Returned                                                                                  Description
   data                          This is only present when state=present
   dictionary when state=present Sample:
                                 {'ssh_public_keys': [{'key': 'ssh-rsa AAAA....'}]}
