SYNOPSIS                   *pn_stp_port*

     • This module can be used modify Spanning Tree Protocol (STP) parameters on ports.

PARAMETERS                   *pn_stp_port-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_block            Choices:
   boolean             • no           Specify if a STP port blocks BPDUs.
                       • yes
   pn_bpdu_guard       Choices:
   boolean             • no           STP port BPDU guard.
                       • yes
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_cost           Default:         STP port cost from 1 to 200000000.
   string            "2000"
   pn_edge             Choices:
   boolean             • no           STP port is an edge port.
                       • yes
   pn_filter           Choices:
   boolean             • no           STP port filters BPDUs.
                       • yes
   pn_port                            STP port.
   string
   pn_priority       Default:         STP port priority from 0 to 240.
   string            "128"
   pn_root_guard       Choices:
   boolean             • no           STP port Root guard.
                       • yes
   state               Choices:       State the action to perform. Use update to update stp-port.
   string / required   • update

EXAMPLES                   *pn_stp_port-examples*

 - name: Modify stp port
   pn_stp_port:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "1"
     pn_filter: True
     pn_priority: '144'

 - name: Modify stp port
   pn_stp_port:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "1"
     pn_cost: "200"

 - name: Modify stp port
   pn_stp_port:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "1"
     pn_edge: True
     pn_cost: "200"

RETURN VALUES                   *pn_stp_port-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the stp-port command.
   list
   stdout  always   set of responses from the stp-port command.
   list
