SYNOPSIS                   *gconftool2*

     • This module allows for the manipulation of GNOME 2 Configuration via gconftool-2. Please see the gconftool-2(1) man pages for more details.

PARAMETERS                   *gconftool2-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   config_source                  Specify a configuration source to use rather than the default path. See man gconftool-2(1)
   -
   direct          Choices:
   boolean         • no ←         Access the config database directly, bypassing server. If direct is specified then the config_source must be specified as well. See man gconftool-2(1)
                   • yes
   key                            A GConf preference key is an element in the GConf repository that corresponds to an application preference. See man gconftool-2(1)
   - / required
                   Choices:
   state           • absent       The action to take upon the key/value.
   - / required    • get
                   • present
   value                          Preference keys typically have simple values such as strings, integers, or lists of strings and integers. This is ignored if the state is "get". See man
   -                              gconftool-2(1)
                   Choices:
   value_type      • bool
   -               • float        The type of value being set. This is ignored if the state is "get".
                   • int
                   • string

EXAMPLES                   *gconftool2-examples*

 - name: Change the widget font to "Serif 12"
   gconftool2:
     key: "/desktop/gnome/interface/font_name"
     value_type: "string"
     value: "Serif 12"

RETURN VALUES                   *gconftool2-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key     Returned                                                                                  Description
   key                 The key specified in the module parameters
   string     success  Sample:
                       /desktop/gnome/interface/font_name
   value               The value of the preference key after executing the module
   string     success  Sample:
                       Serif 12
   value_type          The type of the value that was changed
   string     success  Sample:
                       string
