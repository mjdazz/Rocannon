SYNOPSIS                   *cpanm*

     • Manage Perl library dependencies.

PARAMETERS                   *cpanm-parameters*

    Parameter  Choices/Defaults                                                                               Comments
   executable                   Override the path to the cpanm executable
   path
   from_path                    The local directory from where to install
   -
   installdeps   Choices:
   boolean       • no ←         Only install dependencies
                 • yes
   locallib                     Specify the install base to install modules
   path
   mirror                       Specifies the base URL for the CPAN mirror to use
   string
   mirror_only   Choices:
   boolean       • no ←         Use the mirror's index file instead of the CPAN Meta DB
                 • yes
   name                         The name of the Perl library to install. You may use the "full distribution path", e.g. MIYAGAWA/Plack-0.99_05.tar.gz
   -                            aliases: pkg
   notest        Choices:
   boolean       • no ←         Do not run unit tests
                 • yes
   system_lib    Choices:       Use this if you want to install modules to the system perl include path. You must be root or have "passwordless" sudo for this to work.
   boolean       • no ←         This uses the cpanm commandline option '--sudo', which has nothing to do with ansible privilege escalation.
                 • yes          aliases: use_sudo
   version                      minimum version of perl module to consider acceptable
   string

NOTES                   *cpanm-notes*

   Note

     • Please note that http://search.cpan.org/dist/App-cpanminus/bin/cpanm, cpanm must be installed on the remote host.

EXAMPLES                   *cpanm-examples*

 # install Dancer perl package
 - cpanm:
     name: Dancer

 # install version 0.99_05 of the Plack perl package
 - cpanm:
     name: MIYAGAWA/Plack-0.99_05.tar.gz

 # install Dancer into the specified locallib
 - cpanm:
     name: Dancer
     locallib: /srv/webapps/my_app/extlib

 # install perl dependencies from local directory
 - cpanm:
     from_path: /srv/webapps/my_app/src/

 # install Dancer perl package without running the unit tests in indicated locallib
 - cpanm:
     name: Dancer
     notest: True
     locallib: /srv/webapps/my_app/extlib

 # install Dancer perl package from a specific mirror
 - cpanm:
     name: Dancer
     mirror: 'http://cpan.cpantesters.org/'

 # install Dancer perl package into the system root path
 - cpanm:
     name: Dancer
     system_lib: yes

 # install Dancer if it's not already installed
 # OR the installed version is older than version 1.0
 - cpanm:
     name: Dancer
     version: '1.0'
