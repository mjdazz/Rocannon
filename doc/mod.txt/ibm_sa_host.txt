SYNOPSIS                   *ibm_sa_host*

     • This module adds hosts to or removes them from IBM Spectrum Accelerate Family storage systems.

REQUIREMENTS                   *ibm_sa_host-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • pyxcli

PARAMETERS                   *ibm_sa_host-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   cluster                            The name of the cluster to include the host.
   -
   domain                             The domains the cluster will be attached to. To include more than one domain, separate domain names with commas. To include all existing domains, use an
   -                                  asterisk (""").
   endpoints                          The hostname or management IP of Spectrum Accelerate storage system.
   string / required
   host                               Host name.
   - / required
   iscsi_chap_name                    The host's CHAP name identifier
   -
   iscsi_chap_secret                  The password of the initiator used to authenticate to the system when CHAP is enable
   -
   password                           Password for username on the spectrum accelerate storage system.
   string / required
   state               Choices:
   - / required        • present ←    Host state.
                       • absent
   username                           Management user on the spectrum accelerate storage system.
   string / required

NOTES                   *ibm_sa_host-notes*

   Note

     • This module requires pyxcli python library. Use ‘pip install pyxcli’ in order to get pyxcli.

EXAMPLES                   *ibm_sa_host-examples*

 - name: Define new host.
   ibm_sa_host:
     host: host_name
     state: present
     username: admin
     password: secret
     endpoints: hostdev-system

 - name: Delete host.
   ibm_sa_host:
     host: host_name
     state: absent
     username: admin
     password: secret
     endpoints: hostdev-system
