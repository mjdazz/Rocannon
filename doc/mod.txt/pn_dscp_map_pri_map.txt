SYNOPSIS                   *pn_dscp_map_pri_map*

     • This module can be used to update priority mappings in tables.

PARAMETERS                   *pn_dscp_map_pri_map-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_dsmap                           DSCP value(s).
   string
   pn_name                            Name for the DSCP map.
   string
   pn_pri                             CoS priority.
   string
   state               Choices:       State the action to perform. Use update to modify the dscp-map-pri-map.
   string / required   • update

EXAMPLES                   *pn_dscp_map_pri_map-examples*

 - name: dscp map pri map modify
   pn_dscp_map_pri_map:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_name: 'foo'
     pn_pri: '0'
     pn_dsmap: '40'

 - name: dscp map pri map modify
   pn_dscp_map_pri_map:
     pn_cliswitch: 'sw01'
     state: 'update'
     pn_name: 'foo'
     pn_pri: '1'
     pn_dsmap: '8,10,12,14'

RETURN VALUES                   *pn_dscp_map_pri_map-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the dscp-map-pri-map command.
   list
   stdout  always   set of responses from the dscp-map-pri-map command.
   list
