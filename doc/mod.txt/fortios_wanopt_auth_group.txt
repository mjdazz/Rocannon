SYNOPSIS                   *fortios_wanopt_auth_group*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify wanopt feature and auth_group category. Examples include all parameters and values
       need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_wanopt_auth_group-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_wanopt_auth_group-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure WAN optimization authentication groups.
     fortios_wanopt_auth_group:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       wanopt_auth_group:
         auth_method: "cert"
         cert: "<your_own_value> (source vpn.certificate.local.name)"
         name: "default_name_5"
         peer: "<your_own_value> (source wanopt.peer.peer-host-id)"
         peer_accept: "any"
         psk: "<your_own_value>"
