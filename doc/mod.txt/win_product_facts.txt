SYNOPSIS                   *win_product_facts*

     • Provides Windows product and license information.

EXAMPLES                   *win_product_facts-examples*

 - name: Get product id and product key
   win_product_facts:

RETURNED FACTS                   *win_product_facts-returned facts*

   Facts returned by this module are added/updated in the hostvars host facts and can be referenced by name just like any other host fact. They do not need to be registered in order to use them.

              Fact            Returned                                                                          Description
   ansible_os_license_channel          The Windows license channel.
   string                     always   Sample:
   added in 2.8                        Volume:MAK
   ansible_os_license_edition          The Windows license edition.
   string                     always   Sample:
   added in 2.8                        Windows(R) ServerStandard edition
   ansible_os_license_status           The Windows license status.
   string                     always   Sample:
   added in 2.8                        Licensed
   ansible_os_product_id               The Windows product ID.
   string                     always   Sample:
                                       00326-10000-00000-AA698
   ansible_os_product_key              The Windows product key.
   string                     always   Sample:
                                       T49TD-6VFBW-VV7HY-B2PXY-MY47H
