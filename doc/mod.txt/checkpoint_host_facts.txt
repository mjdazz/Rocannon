SYNOPSIS                   *checkpoint_host_facts*

     • Get host objects facts on Check Point devices. All operations are performed over Web Services API.

PARAMETERS                   *checkpoint_host_facts-parameters*

   Parameter Choices/Defaults                                                                                Comments
   name                       Name of the host object. If name is not provided, UID is required.
   string
   uid                        UID of the host object. If UID is not provided, name is required.
   string

EXAMPLES                   *checkpoint_host_facts-examples*

 - name: Get host object facts
   checkpoint_host_facts:
     name: attacker

RETURN VALUES                   *checkpoint_host_facts-return values*

   Common return values are documented here, the following are the fields unique to this module:

        Key      Returned                                                                                 Description
   ansible_hosts always.  The checkpoint host object facts.
   list
