SYNOPSIS                   *pn_vrouter_pim_config*

     • This module can be used to modify pim parameters.

PARAMETERS                   *pn_vrouter_pim_config-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   pn_cliswitch                        Target switch to run the CLI on.
   string
   pn_hello_interval                   hello interval in seconds.
   string
   pn_querier_timeout                  igmp querier timeout in seconds.
   string
   pn_query_interval                   igmp query interval in seconds.
   string
   pn_vrouter_name                     name of service config.
   string
   state                Choices:       State the action to perform. Use update to modify the vrouter-pim-config.
   string / required    • update

EXAMPLES                   *pn_vrouter_pim_config-examples*

 - name: pim config modify
   pn_vrouter_pim_config:
     pn_cliswitch: '192.168.1.1'
     pn_query_interval: '10'
     pn_querier_timeout: '30'
     state: 'update'
     pn_vrouter_name: 'ansible-spine1-vrouter'

RETURN VALUES                   *pn_vrouter_pim_config-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the vrouter-pim-config command.
   list
   stdout  always   set of responses from the vrouter-pim-config command.
   list
