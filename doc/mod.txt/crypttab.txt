SYNOPSIS                   *crypttab*

     • Control Linux encrypted block devices that are set up during system boot in /etc/crypttab.

PARAMETERS                   *crypttab-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   backing_device                  Path to the underlying block device or file, or the UUID of a block-device prefixed with UUID=.
   string
   name                            Name of the encrypted block device as it appears in the /etc/crypttab file, or optionally prefixed with /dev/mapper/, as it appears in the filesystem.
   string /                        /dev/mapper/ will be stripped from name.
   required
   opts                            A comma-delimited list of options. See crypttab(5 ) for details.
   string
   password                        Encryption password, the path to a file containing the password, or - or unset if the password should be entered at boot.
   path
   path           Default:         Path to file to use instead of /etc/crypttab.
   path           "/etc/crypttab"  This might be useful in a chroot environment.
                    Choices:       Use present to add a line to /etc/crypttab or update its definition if already present.
   state            • absent       Use absent to remove a line with matching name.
   string /         • opts_absent  Use opts_present to add options to those already present; options with different values will be updated.
   required         • opts_present Use opts_absent to remove options from the existing set.
                    • present

EXAMPLES                   *crypttab-examples*

 - name: Set the options explicitly a device which must already exist
   crypttab:
     name: luks-home
     state: present
     opts: discard,cipher=aes-cbc-essiv:sha256

 - name: Add the 'discard' option to any existing options for all devices
   crypttab:
     name: '{{ item.device }}'
     state: opts_present
     opts: discard
   loop: '{{ ansible_mounts }}'
   when: "'/dev/mapper/luks-' in {{ item.device }}"
