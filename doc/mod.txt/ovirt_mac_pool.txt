SYNOPSIS                   *ovirt_mac_pool*

     • This module manage MAC pools in oVirt/RHV.

REQUIREMENTS                   *ovirt_mac_pool-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • ovirt-engine-sdk-python >= 4.3.0

EXAMPLES                   *ovirt_mac_pool-examples*

 # Examples don't contain auth parameter for simplicity,
 # look at ovirt_auth module to see how to reuse authentication:

 # Create MAC pool:
 - ovirt_mac_pool:
     name: mymacpool
     allow_duplicates: false
     ranges:
       - 00:1a:4a:16:01:51,00:1a:4a:16:01:61
       - 00:1a:4a:16:02:51,00:1a:4a:16:02:61

 # Remove MAC pool:
 - ovirt_mac_pool:
     state: absent
     name: mymacpool

 # Change MAC pool Name
 - ovirt_nic:
     id: 00000000-0000-0000-0000-000000000000
     name: "new_mac_pool_name"
