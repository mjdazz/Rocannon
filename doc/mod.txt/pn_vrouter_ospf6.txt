SYNOPSIS                   *pn_vrouter_ospf6*

     • This module can be used to add interface ip to OSPF6 protocol or remove interface ip from OSPF6 protocol on vRouter.

PARAMETERS                   *pn_vrouter_ospf6-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_nic                             OSPF6 control for this interface.
   string
   pn_ospf6_area                      area id for this interface in IPv4 address format.
   string
   pn_vrouter_name                    name of service config.
   string
   state               Choices:
   string / required   • present      State the action to perform. Use present to add vrouter-ospf6 and absent to remove interface from vrouter-ospf6.
                       • absent

EXAMPLES                   *pn_vrouter_ospf6-examples*

 - name: Add vrouter interface nic to ospf6
   pn_vrouter_ospf6:
     pn_cliswitch: "sw01"
     state: "present"
     pn_vrouter_name: "foo-vrouter"
     pn_nic: "eth0.4092"
     pn_ospf6_area: "0.0.0.0"

 - name: Remove vrouter interface nic to ospf6
   pn_vrouter_ospf6:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_vrouter_name: "foo-vrouter"
     pn_nic: "eth0.4092"

RETURN VALUES                   *pn_vrouter_ospf6-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the vrouter-ospf6 command.
   list
   stdout  always   set of responses from the vrouter-ospf6 command.
   list
