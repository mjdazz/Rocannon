SYNOPSIS                   *nxos_system*

     • This module provides declarative management of node system attributes on Cisco NXOS devices. It provides an option to configure host system parameters or remove those parameters from the
       device active configuration.

EXAMPLES                   *nxos_system-examples*

 - name: configure hostname and domain-name
   nxos_system:
     hostname: nxos01
     domain_name: test.example.com

 - name: remove configuration
   nxos_system:
     state: absent

 - name: configure name servers
   nxos_system:
     name_servers:
       - 8.8.8.8
       - 8.8.4.4

 - name: configure name servers with VRF support
   nxos_system:
     name_servers:
       - { server: 8.8.8.8, vrf: mgmt }
       - { server: 8.8.4.4, vrf: mgmt }
