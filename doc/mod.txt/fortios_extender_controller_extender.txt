SYNOPSIS                   *fortios_extender_controller_extender*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify extender_controller feature and extender category. Examples include all parameters
       and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_extender_controller_extender-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8
