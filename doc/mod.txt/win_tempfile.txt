SYNOPSIS                   *win_tempfile*

     • Creates temporary files and directories.
     • For non-Windows targets, please use the tempfile module instead.

PARAMETERS                   *win_tempfile-parameters*

   Parameter Choices/Defaults                                                                                Comments
   path      Default:         Location where temporary file or directory should be created.
   path      "%TEMP%"         If path is not specified default system temporary directory (%TEMP%) will be used.
                              aliases: dest
   prefix    Default:         Prefix of file/directory name created by module.
   string    "ansible."
   state       Choices:
   string      • directory    Whether to create file or directory.
               • file ←
   suffix    Default:         Suffix of file/directory name created by module.
   string    ""

SEE ALSO                   *win_tempfile-see also*

   See also

   tempfile – Creates temporary files and directories
   The official documentation on the tempfile module.

EXAMPLES                   *win_tempfile-examples*

 - name: Create temporary build directory
   win_tempfile:
     state: directory
     suffix: build

 - name: Create temporary file
   win_tempfile:
     state: file
     suffix: temp

RETURN VALUES                   *win_tempfile-return values*

   Common return values are documented here, the following are the fields unique to this module:

    Key   Returned                                                                                    Description
   path            The absolute path to the created file or directory.
   string success  Sample:
                   C:\Users\Administrator\AppData\Local\Temp\ansible.bMlvdk
