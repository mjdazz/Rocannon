SYNOPSIS                   *fortios_antivirus_profile*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify antivirus feature and profile category. Examples include all parameters and values
       need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_antivirus_profile-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_antivirus_profile-examples*

   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
