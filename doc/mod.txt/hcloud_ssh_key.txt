SYNOPSIS                   *hcloud_ssh_key*

     • Create, update and manage ssh keys on the Hetzner Cloud.

REQUIREMENTS                   *hcloud_ssh_key-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0

PARAMETERS                   *hcloud_ssh_key-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   fingerprint                                      The Fingerprint of the Hetzner Cloud ssh_key to manage.
   string                                           Only required if no ssh_key id or name is given.
   id                                               The ID of the Hetzner Cloud ssh_key to manage.
   integer                                          Only required if no ssh_key name is given
   labels                                           User-defined labels (key-value pairs)
   dictionary
   name                                             The Name of the Hetzner Cloud ssh_key to manage.
   string                                           Only required if no ssh_key id is given or a ssh_key does not exists.
   public_key                                       The Public Key to add.
   string                                           Required if ssh_key does not exists.
   state               Choices:
   string              • absent                     State of the ssh_key.
                       • present ←

EXAMPLES                   *hcloud_ssh_key-examples*

 - name: Create a basic ssh_key
   hcloud_ssh_key:
     name: my-ssh_key
     public_key: "ssh-rsa AAAjjk76kgf...Xt"
     state: present

 - name: Create a ssh_key with labels
   hcloud_ssh_key:
     name: my-ssh_key
     public_key: "ssh-rsa AAAjjk76kgf...Xt"
     labels:
         key: value
         mylabel: 123
     state: present

 - name: Ensure the ssh_key is absent (remove if needed)
   hcloud_ssh_key:
     name: my-ssh_key
     state: absent
