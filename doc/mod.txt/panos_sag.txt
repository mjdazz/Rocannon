SYNOPSIS                   *panos_sag-synopsis*

     • Create a static address group object in the firewall used for policy rules.

REQUIREMENTS                   *panos_sag-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python can be obtained from PyPI https://pypi.org/project/pan-python/
     • pandevice can be obtained from PyPI https://pypi.org/project/pandevice/
     • xmltodict can be obtained from PyPI https://pypi.org/project/xmltodict/

PARAMETERS                   *panos_sag-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   api_key                            API key that can be used instead of username/password credentials.
   -
   commit              Choices:
   boolean             • no           commit if changed
                       • yes ←
   description                        The purpose / objective of the static Address Group
   -
   devicegroup                        - The name of the Panorama device group. The group must exist on Panorama. If device group is not defined it is assumed that we are contacting a firewall.
   -
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
                       Choices:
   operation           • add          The operation to perform Supported values are add/list/delete.
   - / required        • list
                       • delete
   password                           Password for authentication.
   string / required
   sag_match_filter                   Static filter user by the address group
   list
   sag_name                           name of the dynamic address group
   - / required
   tags                               Tags to be associated with the address group
   -
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_sag-examples*

 - name: sag
   panos_sag:
     ip_address: "192.168.1.1"
     password: "admin"
     sag_name: "sag-1"
     static_value: ['test-addresses', ]
     description: "A description for the static address group"
     tags: ["tags to be associated with the group", ]
