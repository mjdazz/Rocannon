SYNOPSIS                   *na_elementsw_cluster_config*

     • Configure Element Software cluster.

REQUIREMENTS                   *na_elementsw_cluster_config-requirements*

   The below requirements are needed on the host that executes this module.

     • The modules were developed with SolidFire 10.1
     • solidfire-sdk-python (1.1.0.92) or greater. Install using ‘pip install solidfire-sdk-python’

PARAMETERS                   *na_elementsw_cluster_config-parameters*

                Parameter              Choices/Defaults                                                                   Comments
   enable_virtual_volumes                Choices:
   boolean                               • no           Enable the NetApp SolidFire VVols cluster feature
                                         • yes ←
   encryption_at_rest                    Choices:
   -                                     • present      enable or disable the Advanced Encryption Standard (AES) 256-bit encryption at rest on the cluster
                                         • absent
   hostname                                             The hostname or IP address of the SolidFire cluster.
   - / required
                                                        The capacity level at which the cluster generates an event
   modify_cluster_full_threshold                        Requires a stage3_block_threshold_percent or
   -                                                    max_metadata_over_provision_factor or
                                                        stage2_aware_threshold
    max_metadata_over_provision_factor                  The number of times metadata space can be overprovisioned relative to the amount of space available
    -
    stage2_aware_threshold                              The number of nodes of capacity remaining in the cluster before the system triggers a notification
    -
    stage3_block_threshold_percent                      The percentage below the "Error" threshold that triggers a cluster "Warning" alert
    -
   password                                             Password for the specified user.
   - / required                                         aliases: pass
   set_ntp_info                                         configure NTP on cluster node

NOTES                   *na_elementsw_cluster_config-notes*

   Note

     • The modules prefixed with na\_elementsw are built to support the SolidFire storage platform.

EXAMPLES                   *na_elementsw_cluster_config-examples*

 - name: Configure cluster
   tags:
   - elementsw_cluster_config
   na_elementsw_cluster_config:
     hostname: "{{ elementsw_hostname }}"
     username: "{{ elementsw_username }}"
     password: "{{ elementsw_password }}"
     modify_cluster_full_threshold:
       stage2_aware_threshold: 2
       stage3_block_threshold_percent: 10
       max_metadata_over_provision_factor: 2
     encryption_at_rest: absent
     set_ntp_info:
       broadcastclient: False
       ntp_servers:
       - 1.1.1.1
       - 2.2.2.2
     enable_virtual_volumes: True
