SYNOPSIS                   *mso_tenant*

     • Manage tenants on Cisco ACI Multi-Site.

REQUIREMENTS                   *mso_tenant-requirements*

   The below requirements are needed on the host that executes this module.

     • Multi Site Orchestrator v2.1 or newer

PARAMETERS                   *mso_tenant-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   description                        The description for this tenant.
   string
   display_name                       The name of the tenant to be displayed in the web UI.
   string / required
   host                               IP Address or hostname of the ACI Multi Site Orchestrator host.
   string / required                  aliases: hostname
                       Choices:       Influence the output of this ACI module.
   output_level        • debug        normal means the standard output, incl. current dict

   port                               Port number to be used for the REST connection.
   integer                            The default value depends on parameter `use_ssl`.

   state               • absent       Use present or absent for adding or removing.

NOTES                   *mso_tenant-notes*

   Note

     • Please read the Cisco ACI Guide for more detailed information on how to manage your ACI infrastructure using Ansible.
     • This module was written to support ACI Multi Site Orchestrator v2.1 or newer. Some or all functionality may not work on earlier versions.

EXAMPLES                   *mso_tenant-examples*

 - name: Add a new tenant
   mso_tenant:
     host: mso_host
     username: admin
     password: SomeSecretPassword
     tenant: north_europe
     display_name: North European Datacenter
     description: This tenant manages the NEDC environment.
     state: present
   delegate_to: localhost

 - name: Remove a tenant
   mso_tenant:
     host: mso_host
     username: admin
     password: SomeSecretPassword
     tenant: north_europe
     state: absent
   delegate_to: localhost

 - name: Query a tenant
   mso_tenant:
     host: mso_host
     username: admin
     password: SomeSecretPassword
     tenant: north_europe
     state: query
   delegate_to: localhost
   register: query_result

 - name: Query all tenants
   mso_tenant:
     host: mso_host
     username: admin
     password: SomeSecretPassword
     state: query
   delegate_to: localhost
   register: query_result
