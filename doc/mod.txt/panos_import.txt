DEPRECATED                   *panos_import*

                                      Removed in Ansible:
                  version: 2.12
        Why:      Consolidating code base.
    Alternative:  Use https://galaxy.ansible.com/PaloAltoNetworks/paloaltonetworks instead.

SYNOPSIS                   *panos_import-synopsis*

     • Import file on PAN-OS device

REQUIREMENTS                   *panos_import-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python
     • requests
     • requests_toolbelt

PARAMETERS                   *panos_import-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   category          Default:         Category of file uploaded. The default is software.
   -                 "software"       See API > Import section of the API reference for category options.
   file                               Location of the file to import into device.
   -
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   url                                URL of the file that will be imported to device.
   -
   username          Default:         Username for authentication.
   string            "admin"
   validate_certs      Choices:
   boolean             • no           If no, SSL certificates will not be validated. Disabling certificate validation is not recommended.
   added in 2.6        • yes ←

NOTES                   *panos_import-notes*

   Note

     • API reference documentation can be read from the /api/ directory of your appliance
     • Certificate validation is enabled by default as of Ansible 2.6. This may break existing playbooks but should be disabled with caution.

EXAMPLES                   *panos_import-examples*

 # import software image PanOS_vm-6.1.1 on 192.168.1.1
 - name: import software image into PAN-OS
   panos_import:
     ip_address: 192.168.1.1
     username: admin
     password: admin
     file: /tmp/PanOS_vm-6.1.1
     category: software
