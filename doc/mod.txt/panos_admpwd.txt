SYNOPSIS                   *panos_admpwd-synopsis*

     • Change the admin password of PAN-OS via SSH using a SSH key for authentication.
     • Useful for AWS instances where the first login should be done via SSH.

REQUIREMENTS                   *panos_admpwd-requirements*

   The below requirements are needed on the host that executes this module.

     • paramiko

PARAMETERS                   *panos_admpwd-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   ip_address                    IP address (or hostname) of PAN-OS device
   - / required
   key_filename                  filename of the SSH Key to use for authentication
   - / required
   newpassword                   password to configure for admin on the PAN-OS device
   - / required
   username     Default:         username for initial authentication
   -            "admin"

EXAMPLES                   *panos_admpwd-examples*

 # Tries for 10 times to set the admin password of 192.168.1.1 to "badpassword"
 # via SSH, authenticating using key /tmp/ssh.key
 - name: set admin password
   panos_admpwd:
     ip_address: "192.168.1.1"
     username: "admin"
     key_filename: "/tmp/ssh.key"
     newpassword: "badpassword"
   register: result
   until: result is not failed
   retries: 10
   delay: 30
