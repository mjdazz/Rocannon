SYNOPSIS                   *purefa_alert*

     • Configure alert email configuration for Pure Storage FlashArrays.
     • Add or delete an individual syslog server to the existing list of serves.

REQUIREMENTS                   *purefa_alert-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • purestorage

PARAMETERS                   *purefa_alert-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   address                            Email address (valid format required)
   string / required
   api_token                          FlashArray API token for admin privileged user.
   string / required
   enabled             Choices:
   boolean             • no           Set specified email address to be enabled or disabled
                       • yes ←
   fa_url                             FlashArray management IPv4 address or Hostname.
   string / required
   state               Choices:
   string              • absent       Create or delete alert email
                       • present ←

NOTES                   *purefa_alert-notes*

   Note

     • This module requires the purestorage Python library
     • You must set PUREFA_URL and PUREFA_API environment variables if fa_url and api_token arguments are not passed to the module directly

EXAMPLES                   *purefa_alert-examples*

 - name: Add new email recipient and enable, or enable existing email
   purefa_alert:
     address: "[email protected]"
     enabled: true
     state: present
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592

 - name: Delete existing email recipient
   purefa_alert:
     state: absent
     address: "[email protected]"
     fa_url: 10.10.10.2
     api_token: e31060a7-21fc-e277-6240-25983c6c4592
