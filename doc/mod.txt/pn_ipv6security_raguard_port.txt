SYNOPSIS                   *pn_ipv6security_raguard_port*

     • This module can be used to add ports to RA Guard Policy and remove ports to RA Guard Policy.

PARAMETERS                   *pn_ipv6security_raguard_port-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            RA Guard Policy Name.
   string / required
   pn_ports                           Ports attached to RA Guard Policy.
   string / required
   state               Choices:
   string              • present ←    ipv6security-raguard-port configuration command.
                       • absent

EXAMPLES                   *pn_ipv6security_raguard_port-examples*

 - name: ipv6 security raguard port add
   pn_ipv6security_raguard_port:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_ports: "1"

 - name: ipv6 security raguard port remove
   pn_ipv6security_raguard_port:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "absent"
     pn_ports: "1"

RETURN VALUES                   *pn_ipv6security_raguard_port-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the ipv6security-raguard-port command.
   list
   stdout  always   set of responses from the ipv6security-raguard-port command.
   list
