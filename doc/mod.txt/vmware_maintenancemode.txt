SYNOPSIS                   *vmware_maintenancemode*

     • This module can be used for placing a ESXi host into maintenance mode.
     • Support for VSAN compliant maintenance mode when selected.

REQUIREMENTS                   *vmware_maintenancemode-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_maintenancemode-notes*

   Note

     • Tested on vSphere 5.5, 6.0 and 6.5

EXAMPLES                   *vmware_maintenancemode-examples*

 - name: Enter VSAN-Compliant Maintenance Mode
   vmware_maintenancemode:
     hostname: "{{ vcenter_hostname }}"
     username: "{{ vcenter_username }}"
     password: "{{ vcenter_password }}"
     esxi_hostname: "{{ esxi_hostname }}"
     vsan: ensureObjectAccessibility
     evacuate: yes
     timeout: 3600
     state: present
   delegate_to: localhost
