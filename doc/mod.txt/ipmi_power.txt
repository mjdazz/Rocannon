SYNOPSIS                   *ipmi_power*

     • Use this module for power management

REQUIREMENTS                   *ipmi_power-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • pyghmi

PARAMETERS                   *ipmi_power-parameters*

    Parameter                              Choices/Defaults                                                                               Comments
   name                                                                                 Hostname or ip address of the BMC.
   - / required
   password                                                                             Password to connect to the BMC.
   - / required
   port         Default:                                                                Remote RMCP port.
   -            623
                  Choices:
                  • on -- Request system turn on
   state          • off -- Request system turn off without waiting for OS to shutdown   Whether to ensure that the machine in desired state.
   - / required   • shutdown -- Have system request OS proper shutdown
                  • reset -- Request system reset without waiting for OS
                  • boot -- If system is off, then 'on', else 'reset'
   timeout      Default:                                                                Maximum number of seconds before interrupt request.
   -            300
   user                                                                                 Username to use to connect to the BMC.
   - / required

EXAMPLES                   *ipmi_power-examples*

 # Ensure machine is powered on.
 - ipmi_power:
     name: test.testdomain.com
     user: admin
     password: password
     state: on

RETURN VALUES                   *ipmi_power-return values*

   Common return values are documented here, the following are the fields unique to this module:

      Key     Returned                                                                                  Description
   powerstate          The current power state of the machine.
   string     success  Sample:
                       True
