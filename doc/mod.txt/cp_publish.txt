SYNOPSIS                   *cp_publish*

     • All the changes done by this user will be seen by all users only after publish is called. All operations are performed over Web Services API.

PARAMETERS                   *cp_publish-parameters*

     Parameter   Choices/Defaults                                                                              Comments
   uid                            Session unique identifier. Specify it to publish a different session than the one you currently use.
   string
   version                        Version of checkpoint. If not given one, the latest version taken.
   string
   wait_for_task   Choices:
   boolean         • no           Wait for the task to end. Such as publish task.
                   • yes ←
