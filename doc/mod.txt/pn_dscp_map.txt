SYNOPSIS                   *pn_dscp_map*

     • This module can be used to create a DSCP priority mapping table.

PARAMETERS                   *pn_dscp_map-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            Name for the DSCP map.
   string
   pn_scope            Choices:
   -                   • local        Scope for dscp map.
                       • fabric
   state               Choices:
   string / required   • present      State the action to perform. Use present to create dscp-map and absent to delete.
                       • absent

EXAMPLES                   *pn_dscp_map-examples*

 - name: dscp map create
   pn_dscp_map:
     pn_cliswitch: "sw01"
     state: "present"
     pn_name: "foo"
     pn_scope: "local"

 - name: dscp map delete
   pn_dscp_map:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_name: "foo"

RETURN VALUES                   *pn_dscp_map-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the dscp-map command.
   list
   stdout  always   set of responses from the dscp-map command.
   list
