SYNOPSIS                   *iosxr_netconf*

     • This module provides an abstraction that enables and configures the netconf system service running on Cisco IOS-XR Software. This module can be used to easily enable the Netconf API. Netconf
       provides a programmatic interface for working with configuration and state resources as defined in RFC 6242.

EXAMPLES                   *iosxr_netconf-examples*

 - name: enable netconf service on port 830
   iosxr_netconf:
     listens_on: 830
     state: present

 - name: disable netconf service
   iosxr_netconf:
     state: absent
