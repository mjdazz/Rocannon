SYNOPSIS                   *stacki_host*

     • Use this module to add or remove hosts to a stacki front-end via API.
     • https://github.com/StackIQ/stacki

PARAMETERS                   *stacki_host-parameters*

      Parameter    Choices/Defaults                                                                             Comments
   force_install     Choices:
   boolean           • no           Set value to True to force node into install state if it already exists in stacki.
                     • yes
   name                             Name of the host to be added to Stacki.
   - / required
   prim_intf                        Name of the primary network interface.
   -
   prim_intf_ip                     IP Address for the primary network interface.
   -
   prim_intf_mac                    MAC Address for the primary PXE boot network interface.
   -
   stacki_endpoint                  URL for the Stacki API Endpoint.
   - / required
   stacki_password                  Password for authenticating with Stacki API, but if not specified, the environment variable stacki_password is used instead.
   - / required
   stacki_user                      Username for authenticating with Stacki API, but if not specified, the environment variable stacki_user is used instead.
   - / required

EXAMPLES                   *stacki_host-examples*

 - name: Add a host named test-1
   stacki_host:
     name: test-1
     stacki_user: usr
     stacki_password: pwd
     stacki_endpoint: url
     prim_intf_mac: mac_addr
     prim_intf_ip: x.x.x.x
     prim_intf: eth0

 - name: Remove a host named test-1
   stacki_host:
     name: test-1
     stacki_user: usr
     stacki_password: pwd
     stacki_endpoint: url
     state: absent
