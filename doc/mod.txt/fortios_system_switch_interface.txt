SYNOPSIS                   *fortios_system_switch_interface*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and switch_interface category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_switch_interface-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

PARAMETERS                   *fortios_system_switch_interface-parameters*

          Parameter        Choices/Defaults                                                                         Comments
   host                                     FortiOS or FortiGate IP address.

   https                     Choices:
   boolean                   • no           Indicates if the requests towards FortiGate must use HTTPS protocol.
                             • yes ←
   password                Default:         FortiOS or FortiGate password.
   string                  ""
   ssl_verify                Choices:
   boolean                   • no           Ensures FortiGate certificate must be verified by a proper CA.
                             • yes ←
   state                     Choices:
   string / required         • present      Indicates whether to create or remove the object.
                             • absent
   system_switch_interface Default:         Configure software switch interfaces by grouping physical and WiFi interfaces.
   dictionary              null
     intra_switch_policy     Choices:
     string                  • implicit     Allow any traffic between switch interfaces or require firewall policies to allow traffic between switch interfaces.
                             • explicit
     member                                 Names of the interfaces that belong to the virtual switch.
     list
         interface_name                     Physical interface name. Source system.interface.name.
         string
     name                                   Interface name (name cannot be in use by any other interfaces, VLANs, or inter-VDOM links).
     string / required
     span                    Choices:
     string                  • disable      Enable/disable port spanning. Port spanning echoes traffic received by the software switch to the span destination port.
                             • enable
     span_dest_port                         SPAN destination port name. All traffic on the SPAN source ports is echoed to the SPAN destination port. Source system.interface.name.
     string
                             Choices:
     span_direction          • rx           The direction in which the SPAN port operates, either: rx, tx, or both.
     string                  • tx
                             • both

     list
         interface_name                     Physical interface name. Source system.interface.name.
         string
     type                    Choices:
     string                  • switch       Type of switch based on functionality: switch for normal functionality, or hub to duplicate packets to all port members.
                             • hub
     vdom                                   VDOM that the software switch belongs to. Source system.vdom.name.
     string
   username                                 FortiOS or FortiGate username.
   string

   string                  "root"

EXAMPLES                   *fortios_system_switch_interface-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure software switch interfaces by grouping physical and WiFi interfaces.
     fortios_system_switch_interface:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       system_switch_interface:
         intra_switch_policy: "implicit"
         member:
          -
             interface_name: "<your_own_value> (source system.interface.name)"
         name: "default_name_6"
         span: "disable"
         span_dest_port: "<your_own_value> (source system.interface.name)"
         span_direction: "rx"
         span_source_port:
          -
             interface_name: "<your_own_value> (source system.interface.name)"
         type: "switch"
         vdom: "<your_own_value> (source system.vdom.name)"
