SYNOPSIS                   *cp_mgmt_threat_profile_facts*

     • Get threat-profile objects facts on Check Point devices.
     • All operations are performed over Web Services API.
     • This module handles both operations, get a specific object and get several objects, For getting a specific object use the parameter ‘name’.

PARAMETERS                   *cp_mgmt_threat_profile_facts-parameters*

     Parameter   Choices/Defaults                                                                              Comments
                   Choices:
   details_level   • uid          The level of detail for some of the fields in the response can vary from showing only the UID value of the object to a fully detailed representation of the object.
   string          • standard
                   • full
   limit                          No more than that many results will be returned. This parameter is relevant only for getting few objects.
   integer
   name                           Object name. This parameter is relevant only for getting a specific object.
   string
   offset                         Skip that many results before beginning to return them. This parameter is relevant only for getting few objects.
   integer
   order                          Sorts results by the given field. By default the results are sorted in the ascending order by name. This parameter is relevant only for getting few objects.
   list
       ASC         Choices:       Sorts results by the given field in ascending order.
       string      • name
       DESC        Choices:       Sorts results by the given field in descending order.
       string      • name
   version                        Version of checkpoint. If not given one, the latest version taken.
   string

EXAMPLES                   *cp_mgmt_threat_profile_facts-examples*

 - name: show-threat-profile
   cp_mgmt_threat_profile_facts:
     name: Recommended_Profile

 - name: show-threat-profiles
   cp_mgmt_threat_profile_facts:
     details_level: standard
     limit: 50
     offset: 0
