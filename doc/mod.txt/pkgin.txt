SYNOPSIS                   *pkgin*

     • The standard package manager for SmartOS, but also usable on NetBSD or any OS that uses pkgsrc. (Home: http://pkgin.net/)

PARAMETERS                   *pkgin-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   clean          Choices:
   boolean        • no ←         Clean packages cache
                  • yes
   force          Choices:
   boolean        • no ←         Force package reinstall
                  • yes
   full_upgrade   Choices:
   boolean        • no ←         Upgrade all packages to their newer versions
                  • yes
   name                          Name of package to install/remove;
   -                             multiple names may be given, separated by commas
   state          Choices:
   -              • present ←    Intended state of the package
                  • absent
   update_cache   Choices:
   boolean        • no ←         Update repository database. Can be run with other steps or on it's own.
                  • yes
   upgrade        Choices:
   boolean        • no ←         Upgrade main packages to their newer versions
                  • yes

NOTES                   *pkgin-notes*

   Note

     • Known bug with pkgin < 0.8.0: if a package is removed and another package depends on it, the other package will be silently removed as well. New to Ansible 1.9: check-mode support.

EXAMPLES                   *pkgin-examples*

 # install package foo
 - pkgin:
     name: foo
     state: present

 # Update database and install "foo" package
 - pkgin:
     name: foo
     update_cache: yes

 # remove package foo
 - pkgin:
     name: foo
     state: absent

 # remove packages foo and bar
 - pkgin:
     name: foo,bar
     state: absent

 # Update repositories as a separate step
 - pkgin:
     update_cache: yes

 # Upgrade main packages (equivalent to C(pkgin upgrade))
 - pkgin:
     upgrade: yes

 # Upgrade all packages (equivalent to C(pkgin full-upgrade))
 - pkgin:
     full_upgrade: yes

 # Force-upgrade all packages (equivalent to C(pkgin -F full-upgrade))
 - pkgin:
     full_upgrade: yes
     force: yes

 # clean packages cache (equivalent to C(pkgin clean))
 - pkgin:
     clean: yes
