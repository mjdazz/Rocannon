SYNOPSIS                   *profitbricks_volume*

     • Allows you to create or remove a volume from a ProfitBricks datacenter. This module has a dependency on profitbricks >= 1.0.0

REQUIREMENTS                   *profitbricks_volume-requirements*

   The below requirements are needed on the host that executes this module.

     • profitbricks

EXAMPLES                   *profitbricks_volume-examples*

 # Create Multiple Volumes

 - profitbricks_volume:
     datacenter: Tardis One
     name: vol%02d
     count: 5
     auto_increment: yes
     wait_timeout: 500
     state: present

 # Remove Volumes

 - profitbricks_volume:
     datacenter: Tardis One
     instance_ids:
       - 'vol01'
       - 'vol02'
     wait_timeout: 500
     state: absent
