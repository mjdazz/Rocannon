SYNOPSIS                   *win_dns_record*

     • Manage DNS records within an existing Windows Server DNS zone.

REQUIREMENTS                   *win_dns_record-requirements*

   The below requirements are needed on the host that executes this module.

     • This module requires Windows 8, Server 2012, or newer.

PARAMETERS                   *win_dns_record-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   computer_name                      Specifies a DNS server.
   string                             You can specify an IP address or any value that resolves to an IP address, such as a fully qualified domain name (FQDN), host name, or NETBIOS name.
   name                               The name of the record.
   string / required
   state               Choices:
   string              • absent       Whether the record should exist or not.
                       • present ←
                                      The "time to live" of the record, in seconds.
   ttl               Default:         Ignored when state=absent.
   integer           3600             Valid range is 1 - 31557600.
                                      Note that an Active Directory forest can specify a minimum TTL, and will dynamically "round up" other values to that minimum.
                       Choices:
   type                • A
   string / required   • AAAA         The type of DNS record to manage.
                       • CNAME
                       • PTR
   value                              The value(s) to specify. Required when state=present.
   list                               When c(type=PTR) only the partial part of the IP should be given.
                                      aliases: values
   zone                               The name of the zone to manage (eg example.com).
   string / required                  The zone must already exist.

EXAMPLES                   *win_dns_record-examples*

 - name: Create database server alias
   win_dns_record:
     name: "db1"
     type: "CNAME"
     value: "cgyl1404p.amer.example.com"
     zone: "amer.example.com"

 - name: PTR example
   win_dns_record:
     name: "1.1.1"
     type: "PTR"
     value: "db1"
     zone: "10.in-addr.arpa"

 - name: Remove static record
   win_dns_record:
     name: "db1"
     type: "A"
     state: absent
     zone: "amer.example.com"
