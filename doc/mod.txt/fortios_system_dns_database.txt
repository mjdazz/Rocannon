SYNOPSIS                   *fortios_system_dns_database*

     • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and dns_database category. Examples include all parameters and
       values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

REQUIREMENTS                   *fortios_system_dns_database-requirements*

   The below requirements are needed on the host that executes this module.

     • fortiosapi>=0.9.8

EXAMPLES                   *fortios_system_dns_database-examples*

 - hosts: localhost
   vars:
    host: "192.168.122.40"
    username: "admin"
    password: ""
    vdom: "root"
    ssl_verify: "False"
   tasks:
   - name: Configure DNS databases.
     fortios_system_dns_database:
       host:  "{{ host }}"
       username: "{{ username }}"
       password: "{{ password }}"
       vdom:  "{{ vdom }}"
       https: "False"
       state: "present"
       system_dns_database:
         allow_transfer: "<your_own_value>"
         authoritative: "enable"
         contact: "<your_own_value>"
         dns_entry:
          -
             canonical_name: "<your_own_value>"
             hostname: "myhostname"
             id:  "9"
             ip: "<your_own_value>"
             ipv6: "<your_own_value>"
             preference: "12"
             status: "enable"
             ttl: "14"
             type: "A"
         domain: "<your_own_value>"
         forwarder: "<your_own_value>"
         ip_master: "<your_own_value>"
         name: "default_name_19"
         primary_name: "<your_own_value>"
         source_ip: "84.230.14.43"
         status: "enable"
         ttl: "23"
         type: "master"
         view: "shadow"
