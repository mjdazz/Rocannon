SYNOPSIS                   *ios_lldp_interfaces*

     • This module manages link layer discovery protocol (LLDP) attributes of interfaces on Cisco IOS devices.

PARAMETERS                   *ios_lldp_interfaces-parameters*

         Parameter        Choices/Defaults                                                                          Comments
   config
   list /                                  A dictionary of LLDP options
   elements=dictionary
                                           Selection of LLDP MED TLVs to send
    med_tlv_select                         NOTE, if med-tlv-select is configured idempotency won't be maintained as Cisco device doesn't record configured med-tlv-select options. As such, Ansible
    dictionary                             cannot verify if the respective med-tlv-select options is already configured or not from the device side. If you try to apply med-tlv-select option in
                                           every play run, Ansible will show changed as True.
     inventory_management   Choices:
     boolean                • no           LLDP MED Inventory Management TLV
                            • yes
    name                                   Full name of the interface excluding any logical unit number, i.e. GigabitEthernet0/1.
    string / required
    receive                 Choices:
    boolean                 • no           Enable LLDP reception on interface.
                            • yes
                                           Selection of LLDP type-length-value i.e. TLVs to send
    tlv_select                             NOTE, if tlv-select is configured idempotency won't be maintained as Cisco device doesn't record configured tlv-select options. As such, Ansible cannot
    dictionary                             verify if the respective tlv-select options is already configured or not from the device side. If you try to apply tlv-select option in every play run,
                                           Ansible will show changed as True.
     power_management       Choices:
     boolean                • no           IEEE 802.3 DTE Power via MDI TLV
                            • yes
    transmit                Choices:
    boolean                 • no           Enable LLDP transmission on interface.
                            • yes
                            Choices:
   state                    • merged ←
   string                   • replaced     The state of the configuration after module completion
                            • overridden
                            • deleted

NOTES                   *ios_lldp_interfaces-notes*

   Note

     • Tested against Cisco IOSv Version 15.2 on VIRL
     • This module works with connection network_cli, See IOS Platform Options.

     Key      Returned                                                                                   Description
