SYNOPSIS                   *ucs_san_connectivity*

     • Configures SAN Connectivity Policies on Cisco UCS Manager.
     • Examples can be used with the UCS Platform Emulator https://communities.cisco.com/ucspe.

REQUIREMENTS                   *ucs_san_connectivity-requirements*

   The below requirements are needed on the host that executes this module.

     • ucsmsdk

EXAMPLES                   *ucs_san_connectivity-examples*

 - name: Configure SAN Connectivity Policy
   ucs_san_connectivity:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: Cntr-FC-Boot
     wwnn_pool: WWNN-Pool
     vhba_list:
     - name: Fabric-A
       vhba_template: vHBA-Template-A
       adapter_policy: Linux
     - name: Fabric-B
       vhba_template: vHBA-Template-B
       adapter_policy: Linux

 - name: Remove SAN Connectivity Policy
   ucs_san_connectivity:
     hostname: 172.16.143.150
     username: admin
     password: password
     name: Cntr-FC-Boot
     state: absent
