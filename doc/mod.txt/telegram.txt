SYNOPSIS                   *telegram*

     • Send notifications via telegram bot, to a verified group or user

PARAMETERS                   *telegram-parameters*

   Parameter  Choices/Defaults                                                                                Comments
   chat_id
   - /                         Telegram group or user chat_id
   required
   msg
   - /                         What message you wish to send.
   required
   msg_format   Choices:
   -            • plain ←      Message format. Formatting options `markdown` and `html` described in Telegram API docs (https://core.telegram.org/bots/api#formatting-options). If option `plain`
   added in     • markdown     set, message will not be formatted.
   2.4          • html
   token
   - /                         Token identifying your telegram bot.
   required

NOTES                   *telegram-notes*

   Note

     • You will require a telegram account and create telegram bot to use this module.

EXAMPLES                   *telegram-examples*

 - name: send a message to chat in playbook
   telegram:
     token: '9999999:XXXXXXXXXXXXXXXXXXXXXXX'
     chat_id: 000000
     msg: Ansible task finished

RETURN VALUES                   *telegram-return values*

   Common return values are documented here, the following are the fields unique to this module:

        Key       Returned                                                                                Description
   msg                     The message you attempted to send
   string         success  Sample:
                           Ansible task finished
   telegram_error          Error message gotten from Telegram API
   string         failure  Sample:
                           Bad Request: message text is empty
