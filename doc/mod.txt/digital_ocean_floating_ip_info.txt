SYNOPSIS                   *digital_ocean_floating_ip_info*

     • This module can be used to fetch DigitalOcean Floating IPs information.
     • This module was called digital_ocean_floating_ip_facts before Ansible 2.9. The usage did not change.

   Aliases: digital_ocean_floating_ip_facts

REQUIREMENTS                   *digital_ocean_floating_ip_info-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6

PARAMETERS                   *digital_ocean_floating_ip_info-parameters*

     Parameter    Choices/Defaults                                                                              Comments
                                   DigitalOcean OAuth token.
   oauth_token                     There are several other environment variables which can be used to provide this value.
   string                          i.e., - 'DO_API_TOKEN', 'DO_API_KEY', 'DO_OAUTH_TOKEN' and 'OAUTH_TOKEN'
                                   aliases: api_token
   timeout        Default:         The timeout in seconds used for polling DigitalOcean's API.
   integer        30
   validate_certs   Choices:       If set to no, the SSL certificates will not be validated.
   boolean          • no           This should only set to no used on personally controlled sites using self-signed certificates.
                    • yes ←

NOTES                   *digital_ocean_floating_ip_info-notes*

   Note

     • Version 2 of DigitalOcean API is used.

EXAMPLES                   *digital_ocean_floating_ip_info-examples*

 - name: "Gather information about all Floating IPs"
   digital_ocean_floating_ip_info:
   register: result

 - name: "List of current floating ips"
   debug: var=result.floating_ips
