SYNOPSIS                   *matrix*

     • This module sends html formatted notifications to matrix rooms.

REQUIREMENTS                   *matrix-requirements*

   The below requirements are needed on the host that executes this module.

     • matrix-client (Python library)

PARAMETERS                   *matrix-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   hs_url                        URL of the homeserver, where the CS-API is reachable
   - / required
   msg_html                      HTML form of the message to send to matrix
   - / required
   msg_plain                     Plain text form of the message to send to matrix, usually markdown
   - / required
   password                      The password to log in with
   -
   room_id                       ID of the room to send the notification to
   - / required
   token                         Authentication token for the API call. If provided, user_id and password are not required
   -
   user_id                       The user id of the user
   -

EXAMPLES                   *matrix-examples*

 - name: Send matrix notification with token
   matrix:
     msg_plain: """hello world"""
     msg_html: "<b>hello world</b>"
     room_id: "!12345678:server.tld"
     hs_url: "https://matrix.org"
     token: "{{ matrix_auth_token }}"

 - name: Send matrix notification with user_id and password
   matrix:
     msg_plain: """hello world"""
     msg_html: "<b>hello world</b>"
     room_id: "!12345678:server.tld"
     hs_url: "https://matrix.org"
     user_id: "ansible_notification_bot"
     password: "{{ matrix_auth_password }}"
