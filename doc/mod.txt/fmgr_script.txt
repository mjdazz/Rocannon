SYNOPSIS                   *fmgr_script*

     • Create/edit/delete scripts and execute the scripts on the FortiManager using jsonrpc API

PARAMETERS                   *fmgr_script-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   adom                                The administrative domain (admon) the configuration belongs to
   - / required
                        Choices:
   mode                 • add ←
   -                    • delete       The desired mode of the specified object. Execute will run the script.
   added in 2.8         • execute
                        • set
   script_content                      The script content that will be executed.
   -
   script_description                  The description of the script.
   -
   script_name                         The name of the script.
   - / required
   script_package                      (datasource) Policy package object to run the script against
   -
   script_scope                        (datasource) The devices that the script will run on, can have both device member and device group member.
   -
   script_target                       The target of the script to be run.
   -
   script_type                         The type of script (CLI or TCL).
   -
   vdom                                The virtual domain (vdom) the configuration belongs to
   -

NOTES                   *fmgr_script-notes*

   Note

     • Full Documentation at https://ftnt-ansible-docs.readthedocs.io/en/latest/.

EXAMPLES                   *fmgr_script-examples*

 - name: CREATE SCRIPT
   fmgr_script:
     adom: "root"
     script_name: "TestScript"
     script_type: "cli"
     script_target: "remote_device"
     script_description: "Create by Ansible"
     script_content: "get system status"

 - name: EXECUTE SCRIPT
   fmgr_script:
     adom: "root"
     script_name: "TestScript"
     mode: "execute"
     script_scope: "FGT1,FGT2"

 - name: DELETE SCRIPT
   fmgr_script:
     adom: "root"
     script_name: "TestScript"
     mode: "delete"
