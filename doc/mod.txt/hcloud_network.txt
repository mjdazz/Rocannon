SYNOPSIS                   *hcloud_network*

     • Create, update and manage cloud Networks on the Hetzner Cloud.
     • You need at least hcloud-python 1.3.0.

REQUIREMENTS                   *hcloud_network-requirements*

   The below requirements are needed on the host that executes this module.

     • hcloud-python >= 1.0.0
     • hcloud-python >= 1.3.0

PARAMETERS                   *hcloud_network-parameters*

       Parameter            Choices/Defaults                                                                            Comments
   api_token                                        This is the API Token for the Hetzner Cloud.
   string / required
   endpoint          Default:                       This is the API Endpoint for the Hetzner Cloud.
   string            "https://api.hetzner.cloud/v1"
   id                                               The ID of the Hetzner Cloud Networks to manage.
   integer                                          Only required if no Network name is given.
   ip_range                                         IP range of the Network.
   string                                           Required if Network does not exists.
   labels                                           User-defined labels (key-value pairs).
   dictionary
   name                                             The Name of the Hetzner Cloud Network to manage.
   string                                           Only required if no Network id is given or a Network does not exists.
   state               Choices:
   string              • absent                     State of the Network.
                       • present ←

EXAMPLES                   *hcloud_network-examples*

 - name: Create a basic network
   hcloud_network:
     name: my-network
     ip_range: 10.0.0.0/8
     state: present

 - name: Ensure the Network is absent (remove if needed)
   hcloud_network:
     name: my-network
     state: absent
