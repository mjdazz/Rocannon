SYNOPSIS                   *pn_dhcp_filter*

     • This module can be used to create, delete and modify a DHCP filter config.

PARAMETERS                   *pn_dhcp_filter-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            name of the DHCP filter.
   string
   pn_trusted_ports                   trusted ports of dhcp config.
   string
                       Choices:
   state               • present      State the action to perform. Use present to create dhcp-filter and absent to delete dhcp-filter update to modify the dhcp-filter.
   string / required   • absent
                       • update

EXAMPLES                   *pn_dhcp_filter-examples*

 - name: dhcp filter create
   pn_dhcp_filter:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "present"
     pn_trusted_ports: "1"

 - name: dhcp filter delete
   pn_dhcp_filter:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "absent"
     pn_trusted_ports: "1"

 - name: dhcp filter modify
   pn_dhcp_filter:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     state: "update"
     pn_trusted_ports: "1,2"

RETURN VALUES                   *pn_dhcp_filter-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the dhcp-filter command.
   list
   stdout  always   set of responses from the dhcp-filter command.
   list
