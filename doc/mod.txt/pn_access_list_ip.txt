SYNOPSIS                   *pn_access_list_ip*

     • This modules can be used to add and remove IPs associated with access list.

PARAMETERS                   *pn_access_list_ip-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   pn_cliswitch                  Target switch to run the CLI on.
   string
   pn_ip        Default:         IP associated with the access list.
   string       "::"
   pn_name                       Access List Name.
   string
   state          Choices:
   - / required   • present      State the action to perform. Use 'present' to add access-list-ip and 'absent' to remove access-list-ip.
                  • absent

EXAMPLES                   *pn_access_list_ip-examples*

 - name: access list ip functionality
   pn_access_list_ip:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_ip: "172.16.3.1"
     state: "present"

 - name: access list ip functionality
   pn_access_list_ip:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_ip: "172.16.3.1"
     state: "absent"

RETURN VALUES                   *pn_access_list_ip-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the access-list-ip command.
   list
   stdout  always   set of responses from the access-list-ip command.
   list
