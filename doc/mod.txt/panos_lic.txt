                  version: 2.12
        Why:      Consolidating code base.
    Alternative:  Use https://galaxy.ansible.com/PaloAltoNetworks/paloaltonetworks instead.

SYNOPSIS                   *panos_lic-synopsis*

     • Apply an authcode to a device.
     • The authcode should have been previously registered on the Palo Alto Networks support portal.
     • The device should have Internet access.

REQUIREMENTS                   *panos_lic-requirements*

   The below requirements are needed on the host that executes this module.

     • pan-python

PARAMETERS                   *panos_lic-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   auth_code                          authcode to be applied
   - / required
   force               Choices:
   boolean             • no ←         whether to apply authcode even if device is already licensed
                       • yes
   ip_address                         IP address (or hostname) of PAN-OS device.
   string / required
   password                           Password for authentication.
   string / required
   username          Default:         Username for authentication.
   string            "admin"

EXAMPLES                   *panos_lic-examples*

 - hosts: localhost
   connection: local
   tasks:
     - name: fetch license
       panos_lic:
         ip_address: "192.168.1.1"
         password: "paloalto"
         auth_code: "IBADCODE"
       register: result
 - name: Display serialnumber (if already registered)
   debug:
     var: "{{result.serialnumber}}"

RETURN VALUES                   *panos_lic-return values*

   Common return values are documented here, the following are the fields unique to this module:

       Key      Returned                                                                                 Description
   serialnumber          serialnumber of the device in case that it has been already registered
   string       success  Sample:
                         973080716
