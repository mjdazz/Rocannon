SYNOPSIS                   *pn_port_cos_rate_setting*

     • This modules can be used to update the port cos rate limit.

PARAMETERS                   *pn_port_cos_rate_setting-parameters*

       Parameter      Choices/Defaults                                                                            Comments
   pn_cliswitch                         Target switch to run the CLI on.
   string
   pn_cos0_rate                         cos0 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos1_rate                         cos1 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos2_rate                         cos2 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos3_rate                         cos3 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos4_rate                         cos4 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos5_rate                         cos5 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos6_rate                         cos6 rate limit (pps) unlimited or 0 to 10000000.
   string
   pn_cos7_rate                         cos7 rate limit (pps) unlimited or 0 to 10000000.
   string
                       Choices:
   pn_port             • control-port   port.
   string              • data-port
                       • span-ports
   state               Choices:         State the action to perform. Use update to modify the port-cos-rate-setting.
   string / required   • update

EXAMPLES                   *pn_port_cos_rate_setting-examples*

 - name: port cos rate modify
   pn_port_cos_rate_setting:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "control-port"
     pn_cos1_rate: "1000"
     pn_cos5_rate: "1000"
     pn_cos2_rate: "1000"
     pn_cos0_rate: "1000"

 - name: port cos rate modify
   pn_port_cos_rate_setting:
     pn_cliswitch: "sw01"
     state: "update"
     pn_port: "data-port"
     pn_cos1_rate: "2000"
     pn_cos5_rate: "2000"
     pn_cos2_rate: "2000"
     pn_cos0_rate: "2000"

RETURN VALUES                   *pn_port_cos_rate_setting-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the port-cos-rate-setting command.
   list
   stdout  always   set of responses from the port-cos-rate-setting command.
   list
