SYNOPSIS                   *scaleway_lb*

     • This module manages load-balancers on Scaleway.

PARAMETERS                   *scaleway_lb-parameters*

      Parameter          Choices/Defaults                                                                             Comments
   api_timeout      Default:                   HTTP timeout to Scaleway API in seconds.
   integer          30                         aliases: timeout
   api_token                                   Scaleway OAuth token.
   string                                      aliases: oauth_token
   api_url          Default:                   Scaleway API URL.
   string           "https://api.scaleway.com" aliases: base_url
   description                                 Description of the load-balancer
   - / required
   name                                        Name of the load-balancer
   - / required
   organization_id                             Organization identifier
   - / required
   query_parameters Default:                   List of parameters passed to the query string.
   dictionary       {}
   region             Choices:
   - / required       • nl-ams                 Scaleway zone
                      • fr-par
   state              Choices:
   -                  • present ←              Indicate desired state of the instance.
                      • absent
   tags                                        List of tags to apply to the load-balancer
   -
   validate_certs     Choices:
   boolean            • no                     Validate SSL certs of the Scaleway API.
                      • yes ←
   wait               Choices:
   boolean            • no ←                   Wait for the load-balancer to reach its desired state before returning.
                      • yes
   wait_sleep_time  Default:                   Time to wait before every attempt to check the state of the load-balancer
   -                3
   wait_timeout     Default:                   Time to wait for the load-balancer to reach the expected state
   -                300

NOTES                   *scaleway_lb-notes*

   Note

     • Also see the API documentation on https://developer.scaleway.com/
     • If api_token is not set within the module, the following environment variables can be used in decreasing order of precedence SCW_TOKEN, SCW_API_KEY, SCW_OAUTH_TOKEN or SCW_API_TOKEN.
     • If one wants to use a different api_url one can also set the SCW_API_URL environment variable.

EXAMPLES                   *scaleway_lb-examples*

 - name: Create a load-balancer
   scaleway_lb:
     name: foobar
     state: present
     organization_id: 951df375-e094-4d26-97c1-ba548eeb9c42
     region: fr-par
     tags:
       - hello

 - name: Delete a load-balancer
   scaleway_lb:
     name: foobar
     state: absent
     organization_id: 951df375-e094-4d26-97c1-ba548eeb9c42
     region: fr-par
