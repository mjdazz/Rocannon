SYNOPSIS                   *icinga2_feature*

     • This module can be used to enable or disable an Icinga2 feature.

PARAMETERS                   *icinga2_feature-parameters*

    Parameter   Choices/Defaults                                                                               Comments
   name                          This is the feature name to enable or disable.
   - / required
                  Choices:       If set to present and feature is disabled, then feature is enabled.
   state          • present ←    If set to present and feature is already enabled, then nothing is changed.
   -              • absent       If set to absent and feature is enabled, then feature is disabled.
                                 If set to absent and feature is already disabled, then nothing is changed.

EXAMPLES                   *icinga2_feature-examples*

 - name: Enable ido-pgsql feature
   icinga2_feature:
     name: ido-pgsql
     state: present

 - name: Disable api feature
   icinga2_feature:
     name: api
     state: absent
