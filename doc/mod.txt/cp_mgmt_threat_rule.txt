SYNOPSIS                   *cp_mgmt_threat_rule*

     • Manages threat-rule objects on Check Point devices including creating, updating and removing objects.
     • All operations are performed over Web Services API.

EXAMPLES                   *cp_mgmt_threat_rule-examples*

 - name: add-threat-rule
   cp_mgmt_threat_rule:
     comments: ''
     install_on: Policy Targets
     layer: New Layer 1
     name: First threat rule
     position: 1
     protected_scope: All_Internet
     state: present
     track: None

 - name: set-threat-rule
   cp_mgmt_threat_rule:
     action: New Profile 1
     comments: commnet for the first rule
     install_on: Policy Targets
     layer: New Layer 1
     name: Rule Name
     position: 1
     protected_scope: All_Internet
     state: present

 - name: delete-threat-rule
   cp_mgmt_threat_rule:
     layer: New Layer 1
     name: Rule Name
     state: absent
