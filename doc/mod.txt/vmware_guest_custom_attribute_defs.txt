SYNOPSIS                   *vmware_guest_custom_attribute_defs*

     • This module can be used to add and remove custom attributes definitions for the given virtual machine from VMware.

REQUIREMENTS                   *vmware_guest_custom_attribute_defs-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6
     • PyVmomi

NOTES                   *vmware_guest_custom_attribute_defs-notes*

   Note

     • Tested on vSphere 6.5

EXAMPLES                   *vmware_guest_custom_attribute_defs-examples*

 - name: Add VMware Attribute Definition
   vmware_guest_custom_attribute_defs:
     hostname: "{{ vcenter_hostname }}"
     username: "{{ vcenter_username }}"
     password: "{{ vcenter_password }}"
     state: present
     attribute_key: custom_attr_def_1
   delegate_to: localhost
   register: defs

 - name: Remove VMware Attribute Definition
   vmware_guest_custom_attribute_defs:
     hostname: "{{ vcenter_hostname }}"
     username: "{{ vcenter_username }}"
     password: "{{ vcenter_password }}"
     state: absent
     attribute_key: custom_attr_def_1
   delegate_to: localhost
   register: defs
