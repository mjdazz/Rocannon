SYNOPSIS                   *syspatch*

     • Manage OpenBSD system patches using syspatch

PARAMETERS                   *syspatch-parameters*

   Parameter Choices/Defaults                                                                                Comments
   apply     Default:         Apply all available system patches
   -         "no"
   revert      Choices:
   string      • all          Revert system patches
               • one

EXAMPLES                   *syspatch-examples*

 - name: Apply all available system patches
   syspatch:
     apply: true

 - name: Revert last patch
   syspatch:
     revert: one

 - name: Revert all patches
   syspatch:
     revert: all

RETURN VALUES                   *syspatch-return values*

   Common return values are documented here, the following are the fields unique to this module:

        Key      Returned                                                                                 Description
   rc            always   The command return code (0 means success)
   integer
   reboot_needed          Whether or not a reboot is required after an update
   boolean       always   Sample:
                          True
   stderr                 syspatch standard error
   string        always   Sample:
                          syspatch: need root privileges
   stdout                 syspatch standard output
   string        always   Sample:
                          001_rip6cksum
