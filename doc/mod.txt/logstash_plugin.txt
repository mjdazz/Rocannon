SYNOPSIS                   *logstash_plugin*

     • Manages Logstash plugins.

PARAMETERS                   *logstash_plugin-parameters*

    Parameter               Choices/Defaults                                                                               Comments
   name                                                   Install plugin with that name.
   - / required
   plugin_bin   Default:                                  Specify logstash-plugin to use for plugin management.
   -            "/usr/share/logstash/bin/logstash-plugin"
   proxy_host                                             Proxy host to use during plugin installation.
   -
   proxy_port                                             Proxy port to use during plugin installation.
   -
   state          Choices:
   -              • present ←                             Apply plugin state.
                  • absent
   version                                                Specify plugin Version of the plugin to install. If plugin exists with previous version, it will NOT be updated.
   -

EXAMPLES                   *logstash_plugin-examples*

 - name: Install Logstash beats input plugin
   logstash_plugin:
     state: present
     name: logstash-input-beats

 - name: Install specific version of a plugin
   logstash_plugin:
     state: present
     name: logstash-input-syslog
     version: '3.2.0'

 - name: Uninstall Logstash plugin
   logstash_plugin:
     state: absent
     name: logstash-filter-multiline

 - name: install Logstash plugin with alternate heap size
   logstash_plugin:
     state: present
     name: logstash-input-beats
   environment:
     LS_JAVA_OPTS: "-Xms256m -Xmx256m"
