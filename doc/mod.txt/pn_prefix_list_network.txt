SYNOPSIS                   *pn_prefix_list_network*

     • This module is used to add network associated with prefix list and remove networks associated with prefix list.

PARAMETERS                   *pn_prefix_list_network-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   pn_cliswitch                       Target switch to run the CLI on.
   string
   pn_name                            Prefix List Name.
   string
   pn_netmask                         netmask of the network associated the prefix list.
   string
   pn_network                         network associated with the prefix list.
   string
   state               Choices:
   string / required   • present      State the action to perform. Use present to create prefix-list-network and absent to delete prefix-list-network.
                       • absent

EXAMPLES                   *pn_prefix_list_network-examples*

 - name: prefix list network add
   pn_prefix_list_network:
     pn_cliswitch: "sw01"
     pn_name: "foo"
     pn_network: "172.16.3.1"
     pn_netmask: "24"
     state: "present"

 - name: prefix list network remove
   pn_prefix_list_network:
     pn_cliswitch: "sw01"
     state: "absent"
     pn_name: "foo"
     pn_network: "172.16.3.1"
     pn_netmask: "24"

RETURN VALUES                   *pn_prefix_list_network-return values*

   Common return values are documented here, the following are the fields unique to this module:

     Key   Returned                                                                                    Description
   changed always   indicates whether the CLI caused changes on the target.
   boolean
   command always   the CLI command run on the target node.
   string
   stderr  on error set of error responses from the prefix-list-network command.
   list
   stdout  always   set of responses from the prefix-list-network command.
   list
