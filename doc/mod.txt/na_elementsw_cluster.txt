SYNOPSIS                   *na_elementsw_cluster*

     • Initialize Element Software node ownership to form a cluster.

REQUIREMENTS                   *na_elementsw_cluster-requirements*

   The below requirements are needed on the host that executes this module.

     • The modules were developed with SolidFire 10.1
     • solidfire-sdk-python (1.1.0.92) or greater. Install using ‘pip install solidfire-sdk-python’

PARAMETERS                   *na_elementsw_cluster-parameters*

         Parameter        Choices/Defaults                                                                          Comments
   accept_eula              Choices:       Required to indicate your acceptance of the End User License Agreement when creating this cluster.
   boolean                  • no           To accept the EULA, set this parameter to true.
                            • yes
   attributes                              List of name-value pairs in JSON object format.
   -
   cluster_admin_password                  Initial password for the cluster admin account.
   -                                       If not provided, default to login password.
   cluster_admin_username                  Username for the cluster admin.
   -                                       If not provided, default to login username.
   hostname                                The hostname or IP address of the SolidFire cluster.
   - / required
   management_virtual_ip                   Floating (virtual) IP address for the cluster on the management network.
   - / required
   nodes                                   Storage IP (SIP) addresses of the initial set of nodes making up the cluster.
   - / required                            nodes IP must be in the list.
   password                                Password for the specified user.
   - / required                            aliases: pass
   replica_count          Default:         Number of replicas of each piece of data to store in the cluster.
   -                      "2"
   storage_virtual_ip                      Floating (virtual) IP address for the cluster on the storage (iSCSI) network.
   - / required
   username                                Please ensure that the user has the adequate permissions. For more information, please read the official documentation
   - / required                            https://mysupport.netapp.com/documentation/docweb/index.html?productID=62636&language=en-US.

NOTES                   *na_elementsw_cluster-notes*

   Note

     • The modules prefixed with na\_elementsw are built to support the SolidFire storage platform.

EXAMPLES                   *na_elementsw_cluster-examples*

 - name: Initialize new cluster
   tags:
   - elementsw_cluster
   na_elementsw_cluster:
     hostname: "{{ elementsw_hostname }}"
     username: "{{ elementsw_username }}"
     password: "{{ elementsw_password }}"
     management_virtual_ip: 10.226.108.32
     storage_virtual_ip: 10.226.109.68
     replica_count: 2
     accept_eula: true
     nodes:
     - 10.226.109.72
     - 10.226.109.74
