SYNOPSIS                   *vultr_user*

     • Create, update and remove users.

   Aliases: vr_user

REQUIREMENTS                   *vultr_user-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.6

EXAMPLES                   *vultr_user-examples*

 - name: Ensure a user exists
   local_action:
     module: vultr_user
     name: john
     email: [email protected]
     password: s3cr3t
     acls:
       - upgrade
       - dns
       - manage_users
       - subscriptions
       - upgrade

 - name: Remove a user
   local_action:
     module: vultr_user
     name: john
     state: absent
