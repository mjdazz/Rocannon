SYNOPSIS                   *netapp_e_amg_role*

     • Update a storage array to become the primary or secondary instance in an asynchronous mirror group

PARAMETERS                   *netapp_e_amg_role-parameters*

     Parameter    Choices/Defaults                                                                              Comments
   api_password                    The password to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_url                         The url to the SANtricity WebServices Proxy or embedded REST API.
   - / required
   api_username                    The username to authenticate with the SANtricity WebServices Proxy or embedded REST API.
   - / required
   force            Choices:
   boolean          • no ←         Whether to force the role reversal regardless of the online-state of the primary
                    • yes
   noSync           Choices:
   boolean          • no ←         Whether to avoid synchronization prior to role reversal
                    • yes
   role             Choices:
   - / required     • primary      Whether the array should be the primary or secondary array for the AMG
                    • secondary
   ssid                            The ID of the primary storage array for the async mirror action
   - / required
   validate_certs   Choices:
   boolean          • no           Should https certificates be validated?
                    • yes ←

EXAMPLES                   *netapp_e_amg_role-examples*

 - name: Update the role of a storage array
   netapp_e_amg_role:
     name: updating amg role
     role: primary
     ssid: "{{ ssid }}"
     api_url: "{{ netapp_api_url }}"
     api_username: "{{ netapp_api_username }}"
     api_password: "{{ netapp_api_password }}"
     validate_certs: "{{ netapp_api_validate_certs }}"

RETURN VALUES                   *netapp_e_amg_role-return values*

   Common return values are documented here, the following are the fields unique to this module:

    Key   Returned                                                                                    Description
   msg             Failure message
   string failure  Sample:
                   No Async Mirror Group with the name.
