SYNOPSIS                   *ce_snmp_target_host*

     • Manages SNMP target host configurations on HUAWEI CloudEngine switches.

PARAMETERS                   *ce_snmp_target_host-parameters*

      Parameter       Choices/Defaults                                                                             Comments
   address                               Network Address.
   -
   connect_port                          Udp port used by SNMP agent to connect the Network management.
   -
   host_name                             Unique name to identify target host entry.
   -
   interface_name                        Name of the interface to send the trap message.
   -
                      Choices:
   is_public_net      • no_use ←         To enable or disable Public Net-manager for target Host.
   -                  • true
                      • false
   notify_type        Choices:
   -                  • trap             To configure notify type as trap or inform.
                      • inform
   recv_port                             UDP Port number used by network management to receive alarm messages.
   -
                      Choices:
   security_level     • noAuthNoPriv     Security level indicating whether to use authentication and encryption.
   -                  • authentication
                      • privacy
                      Choices:
   security_model     • v1               Security Model.
   -                  • v2c
                      • v3
   security_name                         Security Name.
   -
   security_name_v3                      Security Name V3.
   -
                      Choices:
                      • none
                      • v1
   version            • v2c
   -                  • v3               Version(s) Supported by SNMP Engine.
                      • v1v2c
                      • v1v3
                      • v2cv3
                      • all
   vpn_name                              VPN instance Name.
   -

NOTES                   *ce_snmp_target_host-notes*

   Note

     • This module requires the netconf system service be enabled on the remote device being managed.
     • Recommended connection is netconf.
     • This module also works with local connections for legacy playbooks.

EXAMPLES                   *ce_snmp_target_host-examples*

 - name: CloudEngine snmp target host test
   hosts: cloudengine
   connection: local
   gather_facts: no
   vars:
     cli:
       host: "{{ inventory_hostname }}"
       port: "{{ ansible_ssh_port }}"
       username: "{{ username }}"
       password: "{{ password }}"
       transport: cli

   tasks:

   - name: "Config SNMP version"
     ce_snmp_target_host:
       state: present
       version: v2cv3
       provider: "{{ cli }}"

   - name: "Config SNMP target host"
     ce_snmp_target_host:
       state: present
       host_name: test1
       address: 1.1.1.1
       notify_type: trap
       vpn_name: js
       security_model: v2c
       security_name: wdz
       provider: "{{ cli }}"
