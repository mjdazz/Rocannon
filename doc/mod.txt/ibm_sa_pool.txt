SYNOPSIS                   *ibm_sa_pool*

     • This module creates or deletes pools to be used on IBM Spectrum Accelerate Family storage systems

REQUIREMENTS                   *ibm_sa_pool-requirements*

   The below requirements are needed on the host that executes this module.

     • python >= 2.7
     • pyxcli

PARAMETERS                   *ibm_sa_pool-parameters*

       Parameter     Choices/Defaults                                                                            Comments
   domain                             Adds the pool to the specified domain.
   -
   endpoints                          The hostname or management IP of Spectrum Accelerate storage system.
   string / required
   password                           Password for username on the spectrum accelerate storage system.
   string / required
   perf_class                         Assigns a perf_class to the pool.
   -
   pool                               Pool name.
   - / required
   size                               Pool size in GB
   -
   snapshot_size                      Pool snapshot size in GB
   -
   state               Choices:
   - / required        • present ←    Pool state.
                       • absent
   username                           Management user on the spectrum accelerate storage system.
   string / required

NOTES                   *ibm_sa_pool-notes*

   Note

     • This module requires pyxcli python library. Use ‘pip install pyxcli’ in order to get pyxcli.

EXAMPLES                   *ibm_sa_pool-examples*

 - name: Create new pool.
   ibm_sa_pool:
     name: pool_name
     size: 300
     state: present
     username: admin
     password: secret
     endpoints: hostdev-system

 - name: Delete pool.
   ibm_sa_pool:
     name: pool_name
     state: absent
     username: admin
     password: secret
     endpoints: hostdev-system
